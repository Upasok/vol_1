অ_৭ 
( বিনয় )। 
অভিরুলহ ( বিণ)  আরুঢ়, আরোহিত, আরোহণ করিয়াছে এমন, চড়িয়াছে বা উঠিয়াছে এমন। ⇒ অভিরুহতি। 
অভিরুহতি ( ক্রি)  [ অভি + √রুহ + অ ] আরোহণ করা,উপরে উঠা, চড়া, পর্বতারোহণ করা, বৃক্ষরোহণ করা,  অশ্বারোহণ করা, উর্ধ্বে উত্তোলন করা, অশ্বপৃষ্ঠে স্থাপন করা, উভয়পার্শ্বে পা দিয়া উপবেশন করা, যথাযথভাবে সজ্জিত করা, বাধানো কিছুর উপর স্থাপন করা, উর্ধ্বে গমন করা,  অত্যন্ত হওয়া, ওঠা, উচুতে ওঠা,  চেষ্টা দ্বারা উচুতে ওঠা,  উপরে যাওয়া। ( জাতক)  নাবং অভিরুহতি।( অতীত) অভিরুহি, অভিরুচ্ছি।  ( অ বিণ)  অভিরুলহ। ( পষ্ণমী)  অভিরুহ। ( ক্রি - বি)  অভিরুযহ, অভিরুহিত্বা; ( পেতবত্থু) অভিরুযহিত্ব। ( পেতবত্থু অর্থকথা) নাবং অভিরুযহিত্ব; ( জাতক)  পব্বতং অভিরুহিত্বা।
অভিরুহন ( ক্লী)  আরোহণ, চড়ন, উপরে ওঠা, অধিরোহণ, উর্ধ্বে গমন।
অভিরোচেতি ( ক্রি) [ অভি + √রুচ + এ ] ১ পছন্দ করা,অনুমোদন করা,  মনোনীত করা, উপভোগ করা, ( কিছুতে)  সন্তুষ্ট হওয়া বা আনন্দিত হওয়া,  তৃপ্ত হওয়া,  ইচ্ছা করা, আকাঙ্কা করা, পাইতে চাওয়া,  তৃষ্ণা উৎপন্ন করা, আসক্ত হওয়া।  ( বিমানবত্থু)  বতং অভিরোচযি = অভিরোচেসি রুচ্চিত্বা পূরেসী'তি অত্থো; অভিরাধযী'তি পি পাঠো ; সাধেসি নিপ্ফাদেসী'তি অত্থো ( বিমানবত্থু অর্থকথা) ।  ২ সন্তুষ্ট করা, পরিতৃপ্ত করা, প্রত্যয় জন্মানো,  নিঃসন্দেহ করা, সনদেহ ভঞ্জন করা, পরিশোধ করা,  পূর্ণ করা, মেটানো, পোষণ করা, আতিথ্য প্রদর্শন করা, আমোদিত করা,  হৃদয়ে পোষণ করা,হৃষ্ট করা, প্রফুল্ল করা, উল্লাসিত করা।  ৩ " অতিরোচেতি" শব্দের পরিবর্তে বিমানবত্থুতে " অভিরোচেতি" ব্যবহার দৃষ্ট হয়।  ( বিমানবত্থু অর্থকথা)  অভিভবিত্বা বিজ্জোততি ( সমার্থক) ।  ( অ— বিণ) অভিরুচিত।
অভিরোপন (ক্লী)  অভিরোপণ, মনের একাগ্রতা সাধন,  মন সমাবেশকরণ, মনকে কেন্দ্রীভূত করণ, মনকে সমাহরণকরণ,  মনকে ঘনীভূত করণ, মনোযোগ, মনোসংযোগ, মনোনিবেশ।  ( পরিসম্ভিদা মার্গ)  অভিনিরোপন; বুদ্ধি অভিরোপন; অভিরোপণ বিরাগ; অভিরোপন বিমুত্তি ; অভিরোপন অভিসময। ⇒ অভিনিরোপন। 
অভিরোপেতি ( ক্রি)  [ অভি + রোপেতি; সংস্কৃত— অভিরোপয়তি)  মনের একাগ্রতা সাধন করা,  মনোনিবেশ করা, মনোযোগ দেওয়া, মনোসংযোগ করা, মনকে কেন্দ্রীভূত করা,  মনকে সমাহরণ করা, মনকে একত্রীভূত করা বা ঘনীভূত করা, অভিরোপণ করা, মন সমাবেশ করা, ভক্তি দেখানো, শ্রদ্ধা প্রদর্শন করা, মর্যাদা দেওয়া,  সম্মান প্রদর্শন করা,  সম্মান করা, পূজা করা, পদোন্নতি করা, মহিমান্বিত করা, (( হুন্ডি)) প্রভৃতি)  গ্রাহ্য করিয়া
সময়মত টাকা দেওয়া। ( অতীত)  অভিরোপযি, অভিরোপেসি = পূজং কারেসি, পূজেসি। 

অভিলকখিত ( বিণ)  চিহ্নিত, নির্দিষ্ট, লক্ষ্য স্থাপিত, অঙ্কিত, প্রতিষ্টিত, অভিষিক্ত, শুভলক্ষণ বলিয়া চিহ্নিত। ⇒ অভিলকখেতি। 
অভিলকখিতত্ত (ক্লী)  চিহ্নিত বা দাগ দেওয়া হইয়াছে এরুপ বিষয় বা বস্তু, বিশেষ গুণ দ্বারা বর্ণিত ব্যক্তি বা বিষয়,  চিহ্নিত বস্তু, প্রভেদ করা হইয়াছে এমন ব্যক্তি বা বস্তু, বিশেষ গুণসমূহ, লক্ষণসমূহ। 
অভিলকখেতি ( ক্রি)  [ অভি + √লকখ + এ ] অভিষেক করা,  প্রতিষ্ঠা করা,  আরম্ভ করা,  নক্সা করা, অঙ্কিত করা, দাগ কাটা, প্যাটার্ন তোলা,ফুল-পাতা কাটা বা বোনা, অভিসন্ধি করা, চিহ্নিত করা, দাগ দেওয়া, দেখা, লক্ষ্য করা, মনোযোগ দেওয়া, দৃঢ় করা, সংলগ্ন করা, স্থাপন করা, প্রতিষ্ঠিত করা, আবদ্ধ করা, দাম দেওয়া, তারিখ স্থির বা নির্দিষ্ট করা।  ( অতীত)  অভিলকখেসি।  ( অ - বিণ) অভিলকখিত। 
ভিলঙ্ঘতি ( ক্রি) [ অভি + লঙ্ঘতি ] আরোহণ করা, উপরে উঠা, উঠা, চড়া, ফাঁপিয়া উঠা, জন্মানো, উত্থিত হওয়া, লম্ফ দেওয়া, দৃষ্টিগোচর হওয়া, আসিয়া পড়া, উত্তেজিত হওয়া, বিদ্রোহী হওয়া, বৃদ্ধি পাওয়া,  পর্যটন করা, ভ্রমণ করা,  গমন করা, ভ্রমণ করিয়া অতিক্রম করা, লঙ্ঘন করা, উপেক্ষা করা, পার হইয়া যাওয়া,  ছাড়িয়া যাওয়া। 
অভিলম্বতি ( ক্রি)  [ অভি + লম্বতি ] ( কিছুর উপর)  ঝুলিয়া পড়া। = অভিলম্বতি অজঝোলম্বতি  ( নেত্তি) । ( জাতক) পপাতং অভিলম্বতি ; বেতরনিং অভিলম্বতি। ( অ — বিণ)  অভিলম্বিত।
অভিলম্বিত ( বিণ)  নীচের দিকে ঝুলিয়া পড়িয়াছে এমন। ( জাতক)  নীলদুম অভিলম্বিত।  ⇒ অভিলম্বতি। 
অভিলাপ ( পুং)  কথা, বাক্য, বিশেষ ভাষা, কথাবার্তা,  আলাপ, ভাষণ,  কথা কথনের ধারা, বর্ণনা ।  ( সূত্রনিপাত)  বাচাভিলাপ —— আপত্তিকর কথা, বাজে কথা; তিরচ্ছানকথা ( চুলনির্দেশ)  সূত্রনিপাতে " অভিলাপাযং" পাঠ অনিশ্চিত; অভিপাযং, অভিপাপাযং, অভিসাপাযং, অভিসপাযং, অতিসপ্পাযং পাঠও দৃষ্ট হয়। সংযুক্ত নিকায়ে এই সম্পর্কিত পাঠ আছে " অভিসপযং" যাহার অর্থ " অভিশপ; সংযুক্তনিকায় অর্থকথায় বর্ণনা দেওয়া হইয়াছে " অবিলাপো'তি অক্কোসো।  " অধিবচন" এর শব্দান্তরে অর্থ প্রকাশক শব্দস্বরুপ ধর্মসঙ্গণী ও চুলনির্দেশে উল্লিখিত সমার্থবাচক যৌগিক শব্দ " অভিলাপ ব্যঞ্জন" এর অর্থ Mrs. Rh. Davids এর মতে " কথাবার্তা বা আক্রোশজনিত বাক্যে পার্থক্যমূলক কোনো চিহ্ন অথবা যে বিশিষ্ট চিহ্ন নিয়া আলাপ করা হয়।
অভিলাপাযং ( অনিশ্চিত পাঠ)  অপবাদ। ইহা

আক্রোশজনিত শব্দবিশেষ। ⇒ অভিলাপ। 
অভিলাস ( সংস্কৃত — অভিলাষ)  অভিলাষ, ইচ্ছা, লালসা, অনুরাগ, লোভ, তৃষ্ণা, বাসনা। 
অভিলিখতি ( ক্রি)  [ অভি + লিখতি)  লেখা, খোদাই করা।  ( অ — বিণ) অভিলিখতি।  ( করণে)  অভিলেখতি। 
অভিলিখিত ( বিণ)  লিখিত, খোদিত, খোদাইকৃত। ⇒ অভিলিখতি,  অভিলেখতি। 
অভিলেখ — অভিলেখ, আলেখ্য, খোদিত লিপি, লিখিত লিপি।
অভিলেখতি ( ক্রি)  লেখা, অঙ্কিত করা, খোদিত করা, অঙ্কন করা, খোদানো।  ( দাঠাবংস)  চারিত্র — লেখং অভিলেখযি। 
অভিলেপন ( ক্লী)  [ অভি + লেপন ] চর্বি ইত্যাদি লেপন, দাগ, চিহ্ন, কলঙ্ক, নিন্দা, লজ্জা, অপবিত্রকরণ, অবিশুদ্ধতা, অপরিছন্নতা।  = লগ্গন, বন্ধন, উপক্কিলেস। 
অভিবগ্গ [ অভি + বগ্গ ] মহাজনতা, অধিকতর সৈন্য। (বি - ভা)  অভিবগ্গেন ওমদ্দতি — অধিকতর সৈন্য দ্বারা বা ক্ষমতাবলে মর্দিত করা বা দমন করা ( মধ্যম নিকায়)। 
অভিঞ্চন ( ক্লী)  [ অভি + বঞ্চ ] প্রবঞ্চনা, চাতুরী, কপটতা, ছল, ছলনা, প্রবঞ্চক। 
অভিবট্ট = অভিবট্ঠ।  ⇒ অভিবট্ঠ। 
অভিবট্ঠ ( বিণ)  অভিবর্ষিত, বৃষ্টিপাত হইয়াছে এমন,  ঢালিয়াছে এরুপ, ( কিছুর উপর)  ঢালিয়াছে বা বর্ষিত হইয়াছে এরুপ।  ( ধর্মপদ অর্থকথার টাকার)  অভিবুট্ঠ।⇒ অভিবস্সতি। 
অভিবড্ঢ ( বিণ)  অভিবর্ষিত, বর্ষিত, ছাড়াইয়া উঠিয়াছে এমন।  ( ধর্মপদ)  অভিবড্ঢ বীরণতিণ — ধারণতৃণ বা বীণাঘাস বৃদ্ধিপ্রাপ্ত হইয়াছে এরুপ। ⇒ অভিবড্ঢতি। 
অভিবড্ঢতি ( ক্রি)  [ অভি + √বড্ঢ + অ; বৈদিক -- অভিবর্ধতি ]  বৃদ্ধি পাওয়া বা হওয়া,  বাড়া, বর্ধিত হওয়া, (অন্যান্য বৃক্ষাদিকে ছাড়াইয়া বা আবৃত করিয়া তৃণাদি) অতিরিক্ত বাড়িয়া উঠা, ছাড়াইয়া উঠা,  অতিশয় বর্ধিত হওয়া, অতিরিক্ত বৃদ্ধি পাওয়া, বৃদ্ধি পাইয়া অতিক্রম করা।  (বিপ)  হায়তি। (অঙ্গুত্তর নিকায়)  ভোগা অভিবড্ঢতি।  ( জাতক)  বনস্পতিং অভিবড্ঢতি। ( অতীত)  অভিবড্ঢি। ( অ— বিণ)  অভিবড্ঢ, অভিবুড্ঢ।
অভিবড্ঢন ( ক্লী)  বৃদ্ধি, ক্রমবৃদ্ধি, ক্রমিক বৃদ্ধি, উন্নতি, পুষ্টিসাধন, জন্মানো, লাভ, বংশ, উৎপন্ন দ্রব্য, সন্তান সন্ততি, ক্রমবর্ধমান, বর্ধন।  ( বিণ)  বর্ধিত, উন্নত, পুষ্ট, লব্দ।
অভিবড্ঢি ( স্ত্রী)  বৃদ্ধি, জন্মানো। = অভিবড্ঢন।

অভিবণ্ণযি ( অতীত ক্রিয়া)  প্রশংসা করিলেন, গুণকীর্তন করিলেন, ব্যাখ্যা করিলেন।  ⇒ অভিবণ্ণেতি। 
অভিবণ্ণিত (বিণ)  প্রশংসিত, গুণকীর্তিত, ব্যাখ্যাত, গুণবর্ণিত।  ⇒ অভিবণ্ণেতি। 
অভিবণ্ণেতি ( ক্রি)  [ অভি + √বণ্ণ + এ ] প্রশংসা করা, গুণকীর্তন করা, গুণবর্ণনা করা, ব্যাখ্যা করা।  ( অতীত)  অভিবণ্ণেসি, অভিবণ্ণযি।  ( অ — বিণ)  অভিবণ্ণিত। 
অভিবত্ত (বিণ)  = অভিবট্ঠ। ⇒ অভিবট্ঠ।  অভিবর্ষিত। 
অভিবত্থ ( বিণ)  = অভিবট্ঠ। ⇒ অভিবট্ঠ।  অভিবর্ষিত। 
অভিবদতি ( ক্রি) [ অভি + বদতি ]  ১ কাহার ও উদ্দেশ্যে বলা বা লেখা, বলিয়া প্রকাশ করা, বলা, কহা, জ্ঞাপন করা, প্রচার করা, প্রকাশ করা, বক্তব্য প্রকাশ করা, অঙ্গীকার করা, স্বীকার করা, আশা দেওয়া,  দিতে প্রতিশ্রুত হওয়া।  ২ অনুগ্রহ করিয়া বলা, সাদরে অভ্যর্থনা করা, সাদর সম্ভাষণ করা, 'আসুন আসুন' বলা, 'আসিতে আজ্ঞা হউক' বলা, 'স্বাগত স্বাগত' বলা,  অভিবাদন করা, অভ্যর্থনা করা, চুম্বন করা, সম্ভাষণ করা,।  ( প্রায়শ যৌগিকে)  অভিবদতি অভিনন্দতি ( মধ্যম নিকায়) ।  (অতীত) অভিবদি।  ( করণে)  অভিবাদেতি। 
অভিবদ্ধ ( বিণ)  = \ অভিবড্ঢ।  ⇒ অভিবড্ঢ, অভিবদ্ধতি। 
অভিবদ্ধতি ( ক্রি)  [ অভি + বদ্ধতি ] = অভিবড্ঢতি। ⇒ অভিবড্ঢতি।  ( অ — বিণ)  অভিবদ্ধ। 
অভিবদ্ধন ( ক্লী)  = অভিবড্ঢন। ⇒ অভিবড্ঢন। 
অভিবদ্ধি ( স্ত্রী)  = অভিবড্ঢি।  ⇒ অভিবড্ঢি। 
অভিবন্দতি ( ক্রি)  [ অভি + √বন্দ + অ ] বন্দনা করা, নমস্কার করা, প্রণাম করা, প্রণিপাত করা, সম্যক আরাধানা করা, অভিবন্দনা করা, সম্মুখে বসিয়া পূজা করা, অভিবাদন করা, সম্মান  প্রদর্শন করা।  ( অতীত)  অভিবন্দি। ( অ— বিণ)  অভিবন্দিত।  ( ব — বিণ)  অভিবন্দনীয়।  ( ক্রি — বি) অভিবন্দিয, অভিবাদিয।  ( করণে)  অভিবাদেতি,  অভিবাদাপেতি। 
অভিবন্দনা ( স্ত্রী)  = অভিবাদন।  ⇒ অভিবাদন। 
অভিবন্দিত ( বিণ)  বন্দনা করা হয় এমন,  অভিবাদিত, অভ্যর্থিত, বন্দিত,  পূজিত, পূজ্য,  আরাধিত,  সম্মানিত,  সম্মান প্রদর্শিত।  ⇒ অভিবন্দতি। 
অভিবন্দিয ( ক্রি বি)  বন্দনা করিয়া,  পূজা করিয়া,  অভিবাদন করিয়া, আরাধনা করিয়া, সম্মান প্রদর্শন করিয়া,  অভ্যর্থনা করিয়া।  = অভিবাদিত। ⇒ অভিবন্দতি। 
অভিবসতি ( ক্রি)  [ অভি + বসতি ] বাস করা,  থাকা, অবস্থান করা। 
অভিবস্সক ( বিণ)  বৃষ্টিপাত করে বা হয় এমন,  বর্ষণকারী, খসিয়া পড়ে বা ঝড়িয়া পড়ে এমন ; ( চরিত্রে)  রক্তপাত

করে বা হয় এমন, অশ্রুবর্ষণকারী, রক্তপাতকারী, পরিপূর্ণ করে বা ঢালে এমন, নমনীয়, একগুয়ে নয় এমন,বাধ্য,বশ্যতা স্বীকারকারী।  ( বিমানবত্থু অর্থকথা) পুপ্ফাভিবস্সক। 
অভিবস্সতি (ক্রি)  [ অভি + √বস্স + অ ] অভিবর্ষণ করা, অতিবর্ষণ করা, বর্ষণ করা, অতিবৃষ্টি হওয়া, প্রচুর বৃষ্টিপাত হওয়া, অধিক পরিমাণে জল ঢালা, পূর্ণ করিয়া দেওয়া; ( চরিত্রে)  রক্তপাত করা, অশ্রুবিসর্জন করা, বৃষ্টি পড়া, পূর্ণ করা, ঢালা। ( দীর্ঘনিকায় )  আভিবস্সং। ( জাতক)  পুপ্ফাভিবস্সতি — স্রোতের ন্যায় পুষ্প বর্ষিত হওয়া৷ ( তুলনীয়)  অতিবস্সতি। ( অতীত)  অভিবস্সি৷ ( অ— বিণ)  অভিবট্ট, অভিবট্ঠ, অভিবুট্ঠ, অভিবুত্থ, অভিবত্ত, অভিবত্থ৷ ( করণে)  অভিবস্সাপেতি। 
অভিবস্সাপেতি (ক্রি) ( আকাশে)  বৃষ্টিপাতের কারণ করা। ⇒ অভিবস্সতি। 
অভিবস্সিন (বিণ)  = অভিস্সক।  ⇒ অভিবস্সক৷ সব্বত্থ অভিবস্সিন। 
অভিবাদন ( ক্লী)  অভিবাদন, " নমো " উচ্চারণপূর্বক নমস্কার, পদদ্বয় স্পর্শ করিয়া প্রণতি, বন্দনা,  শ্রদ্ধাভিবাদন, সাদর অভ্যর্থনাকরণ, স্বাগত জানানো, সম্মান প্রদর্শনকরণ৷ ( ধর্মপদ)  অভিবাদনসীলিন। ( সদ্ধম্মোপাযন)  অভিবাদনসীল। 
অভিবাদনসীল ( বিণ)  অভিবাদনশীল,  ধর্মনিষ্ঠ, ন্যায়নিষ্ঠ, শ্রদ্ধাশীল, শ্রদ্ধাবিশিষ্ট,  ভক্ত, শ্রদ্ধাপ্রবণ। ⇒ অভিবাদন। 
অভিবাদনসীলিন ( বিণ)  অভিবাদনশীল, ধর্মনিষ্ঠ স্বভাবসম্পন্ন, ন্যায়নিষ্ঠ স্বভাবশীল, শ্রদ্ধাযুক্ত আচারবিশিষ্ট। = অভিবাদনশীল। 
অভিবাদয ( পঞ্চমী)  অভিবাদন করা, নমস্কার করা। = বন্দ। ⇒ বন্দতি, অভিবাদেতি। 
অভিবাদাপেতি ( ক্রি)  কাহারও দ্বারা নমস্কার করানো অথবা সম্ভাষণ করানো।  ⇒ অভিবাদেতি। 
অভিবাদিত ( বিণ)  অভিবাদিত, পূজিত,  নমস্কারকৃত, অভ্যর্থিত, সম্ভাষিত, সম্মান প্রদর্শিত,  সাদরে অভ্যর্থিত। ⇒ অভিবাদেতি। 
অভিবাদিয = অভিবন্দিয। ⇒ অভিবন্দতি। 
অভিবাদেতি (ক্রি)  অভিবাদন করা, প্রণাম করা, নমস্কার করা, ভূলুন্ঠিত নমস্কার করা, হাত জোর করিয়া নমস্কার করা, করজোড়ে নমস্কার করা, কৃতাঞ্জলি নমস্কার করা, পূজা করা, সম্মান করা, সাদর অভ্যর্থনা করা, সম্ভাষণ করা। ⇒ অভিবন্দতি।  ( বিদায়কালীন সম্ভাষণে)  অভিবাদেতি পদকখিণং করোতি ( দীর্ঘনিকায়) । ( অতীত)  অভিবাদেসি,  অভিবাদযিং = অভিবাদনং কারেসিং বন্দিং ( বিমানবত্থু অর্থকথা) ।  ( অ — বিণ)  অভিবাদিত। ( ভ— বিণ)  অভিবাদেতব্ব।  (পঞ্চমী) অভিবাদয।  ( করণে)  অভিবাদাপেতি।

অভিবাদেতব্ব  ( বিণ)  পূজার যোগ্য,  অভিবাদনের যোগ্য,  সম্মানের যোগ্য,  নমস্কার করা উচিত,  অভ্যর্থনা করা উচিত। ⇒ অভিবাদেতি। 
অভিবাযতি (ক্রি)  [ অভি + বাযতি; সংস্কৃত — অভিবাতি ] জোরে প্রবাহিত হওয়া,  প্রবল বায়ু বহা,  তুফান হওয়া,  ঝড় বহা, ব্যাপ্ত করা,  অনুপ্রবেশ করা। = অতিবাযতি। 
অভিবারেতি ( ক্রি)  [ অভি + বারেতি ] ফিরাইয়া ধরা, বারণ করা, দিতে বা লইতে অমত করা,  ' না ' বলা,  অস্বীকার করা, অসত্য বা মিথ্যা বলিয়া প্রতিবাদ করা, ( কিছু হইতে)  বঞ্চিত করা। = নিবারেতি। 
অভিবাহ ( বিণ)  বাহিত, পরিচালিত।। 
অভিবাহেতি ( ক্রি)  [ অভি + বাহেতি ] নড়ানো,  সরাইয়া ফেলা, অপসারিত করা, তুলিয়া লওয়া,  কর্মচ্যুত করা,  পদচ্যুত করা, প্রস্থান করা, ঘর বদল করা, সরাইয়া রাখা, লুক্কাইয়া রাখা, গোপন করা। 
অভিবিজযতি ( ক্রি)  [ অভি + বিজযতি ] বলে বশীভূত করা,  জয় করা, পরাভূত করা, পরাজিত করা, বশীভূত করা, অতিক্রম করা, অধ্যবসায় দ্বারা লাভ করা, জয়ী হওয়া ।  ( অতীত)  অভিবিজযি।  ( অ — বিণ)  অভিবিজিত। ( ক্রি — বি)  অভিবিজিয,  অভিবিজিত্বা। 
অভিবিজিত ( বিণ)  বিজিত, জয়কৃত, ।  ⇒ অভিবিজযতি, অভিবিজিনাতি। 
অভিবিজিনাতি ( ক্রি)  [ অভি + বি + √জি + না ] অভিবিজযতি ।  ⇒ অভিবিজযতি।  ( বর্তমান,  প্রথম পুরুষ  বহুবচন)  অভিবিজিনন্তি।  ( অতীত)  অভিবিজিনি।  ( অ — বিণ)  অভিবিজিত।  ( ক্রি — বি)  অভিবিজিনিত্বা। 
অভিবিঞ্ঞাপেতি ( ক্রি)  [ অভি + বিঞ্ঞাপেতি ] কাহারও মন আকর্ষণ করা অথবা পরিবর্তন করিয়া দেওয়া, ( কাহাকেও)  প্রবৃত্ত করা, প্রবর্তিত করা, বিজ্ঞাপিত করা, ইশারা করা, ইঙ্গিত করা। ( অতীত)  অভিবিঞ্ঞাপেসি।  ( বিনয়)  পুরাণ - দুতিযিকায মেথুনং ধম্মং অভিবিঞ্ঞাপেসি। 
অভিবিতরতি  ( ক্রি)  [ অভি + বিতরতি ] ( কিছুর)  নীচে যাওয়া, তলিয়া যাওয়া,  ভিতরে ঢুকাইয়া দেওয়া, মনোযোগ দেওয়া,  লক্ষ্য করা, যত্ন করা, অবলোকন করা, অবধান করা, পালন করা, মন্তব্য প্রকাশ করা। বিনয়ের বর্ণনায় " সঞ্চিচ্চ " শব্দ পাঠ দৃষ্ট হয়। 
অভিবিনয ( পুং)  [ অভি + বিনয় ] উচ্চশিক্ষা, উচ্চ ধরনের নিয়মানুবর্তিতা,  উচ্চ ধরনের অনুশাসন বা বিনয় অথবা শৃঙ্খলা,  অতিশয় আজ্ঞাধীনত্ব,  উচ্চ ধরনের আইন বা আদেশের অধীনতা,  উচ্চ ধরনের শাসন বা শাস্তি অথবা দন্ড, শিক্ষার বিশুদ্ধিতা, বিনয়ের বিশোধন।  ( যৌগিকে) অভিধম্ম অভিবিনয ( দীর্ঘ নিকায়) ; ( বিনয়)  বিনয অভিবিনয। 
অভিবন্দতি ( ক্রি)  [ অভি + বিন্দতি ] খুঁজিয়া পাওয়া, দেখিতে পাওয়া,  অনুভব করা,  অভিজ্ঞতা বা পরীক্ষার

অভিবিসিট্ঠ ( বিণ)  [ অভি + বিসিট্ঠ ] অতিবিশিষ্ট, অতি উত্তম,  অতি চমৎকার, অতি প্রসিদ্ধ, বিশেষরুপে চিহ্নিত।
অভিবিস্সজ্জতি ( ক্রি)  [ অভি + বিস্সজ্জতি ] বাহিরে প্রেরণ করা,  বাহিরে পাঠানো, প্রক্ষেপ করা, ছোড়া, কোনো সংবাদ বা সংবাদবাহক প্রেরণ করা, শীঘ্র পাঠানো, দেওয়া। 
অভিবিস্সত্থ ( বিণ)  অতি বিশ্বস্ত, বিশ্বাসে আনীত। ⇒ অভিবিস্সসতি।  ( মধ্যম নিকায়)  অভিবিসট্ঠ। 
অভিবিস্সসতি (ক্রি)  [ অভি + বিস্সসতি ] অতি বিশ্বাস করা, অতি বিশ্বাসের মধ্যে আনা। ( অতীত)  অভিবিস্সসি। 
অভিবিহচ্চ ( ক্রি — বি)  নষ্ট করিয়া, বিনাশ করিয়া,  সরাইয়া বা বহিষ্কৃত করিয়া,  তাড়াইয়া, বিদূরিত করিয়া, নির্বাসিত করিয়া, অপসারিত করিয়া, সূর্যের আলোতে অন্ধকার বিদূরিত করিয়া। 
অভিবিহনতি (ক্রি)  [ অভি + বিহনতি ] নষ্ট করা, বিনাশ করা, দূরে সরাইয়া দেওয়া, বহিষ্কার করা, তাড়াইয়া দেওয়া, বিদূরিত করা, নির্বাসিত করা, অপসারিত করা, সূর্যের অন্ধকার বিনষ্ট করা। 
অভিবুট্ঠ ( বিণ)  = অভিবট্ট।  ⇒ অভিবট্ট। 
অভিবুড্ঢ ( বিণ)  বর্ধিত,  বাড়ন্ত, বৃদ্ধিপ্রাপ্ত, ধনাঢ্য, সুশোভিত,  উর্বরা ।  ⇒ অবিবড্ঢতি। 
অভিবুড্ঢি ( স্ত্রী)  অভিবৃদ্ধি,  শ্রীবৃদ্ধি, বৃদ্ধি, উন্নতি, জন্মানো, উৎপাদন, সাফল্য, সৌভাগ্য।  ⇒ অভিবড্ঢি। 
অভিবুত্থ ( বিণ)  ঢালা হইয়াছে এরুপ, প্রবাহিত করা হইয়াছে এরুপ,  বেগে ধাবিত, প্রবাহিত, ঝড়িত, পূরিত, নিক্ষিপ্ত। = অভিবুট্ঠ। ⇒ অভিবস্সতি। 
অভিবুদ্ধ ( বিণ)  বুদ্ধিপ্রাপ্ত। = অভিবুড্ঢ।  ⇒ অভিবুড্ঢ, অভিবদ্ধতি।
অভিবুদ্ধি ( স্ত্রী)  = অভিবড্ঢি, অভিবুড্ঢি।  ⇒ অভিবড্ঢি, অভিবুড্ঢি। 
অভিবেঠেতি ( ক্রি)  [ অভি + বেঠেতি ] ( এই শব্দটি " অতিবেঠেতি " শব্দের পরি জাতকে ব্যবহার দৃষ্ট হয়, কিন্তু জাতক অর্থকথা বর্ণনার সহিত ইহার সামঞ্জস্য নাই) ।  চারিদিকে প্যাঁচাইয়া দেয় অথবা পাক দিয়া রাখে, গুটাইয়া ফেলে,  জড়াইয়া ফেলে।  ⇒ অতিবেঠেতি। 
অভিবেদেতি ( ক্রি)  [ অভি + বেদেতি ] ১ জানানো, সংবাদাদি প্রেরণ করা, পত্র ব্যবহার করা, সংবাদাদি আদান প্রদান করা,  সংলগ্ন থাকা। ২ জানা,  জ্ঞাত হওয়া।  = জানাতি।

অভিসংবিসতি ( ক্রি)  [ অভি + সংবিসতি ] প্রবেশ করা। ⇒ সংসবিসতি। 
অভিসংবিস্সেয্যগত্তং ( " অভিসং বিস্সেয্য — ভন্তং" অথবা অভিসংবিস্সেয্যসন্তং " ?)  Rh Davids বলেন এই শব্দটি এবং ইহার ব্যুৎপত্তি নির্ণয় ও অর্থ সন্দেহজনক, কিন্তু ধর্মপাল থেরীগাথায় ইহার অর্থ করিয়াছেন — " গলিত মাংসপূর্ণ চামড়ার থলি"।

অভিসঙ্খত ( বিণ)  প্রস্তুত, স্থিরীকৃত, স্থির, সুদৃঢ়,   শ্রেণীবদ্ধভাবে সজ্জিত, সজ্জিত, বন্দোবস্তকৃত, নিষ্পত্তিকৃত, কৃত, তৈরি, প্রত্যাপিত,পুনঃসংস্কারকৃত বা নতুন গড়া হইয়াছে এমন, পুনঃস্থাপিত, করা হইয়াছে এরুপ, পুনরায় ভান্ডার পূর্ণ করা হইয়াছে এরুপ, পূর্বের অবস্থায় ফিরাইয়া আনা হইয়াছে এরুপ।  ( মহানির্দেশ) কপ্পিত অভিসঙ্খত। ⇒ অভিসঙ্খরোতি।

অভিসঙ্গ ( পুং)  ১ অভিসঙ্গ,  সংলগ্নতা, আসক্তি,  অনুরক্তি, দৃঢ় অনুরাগ।  ( ধর্মসঙ্গণী অর্থকথা)  অভিসঙ্গ হেতুকং দুকখৎ; অভিসঙ্গরস। ( তুলনীয়)  অভিসজ্জতি; ( সংস্কৃত)  অভিসঙ্গ।  ২ বদমেজাজ, ক্রোধ,  গালাগালি, অভিশাপ। 
অভিসঙ্গিন ( বিণ)  অভিসঙ্গিন,  সংলগ্ন, আসক্ত, অনুরক্ত। 
অভিসজে ( ক্রিয়া সপ্তমী)  ক্রোধের সংলগ্ন হইলে,  ক্রোধের বশীভূত হইলে,  ক্রোধান্বিত হইলে।  = কুজঝাপন বসেন লগ্গপেয্য ( ধর্মপদ অর্থকথা) ।  ⇒ অভিসজ্জতি। 
অভিসজ্জতি ( ক্রি)  [ অভি + √সদ + অ ] বদমেজাজি হওয়া, ক্রুব্দ হওয়া, অভিশাপ দেওয়া,  গালাগালি করা, আসক্ত হওয়া, সংলগ্ন হওয়া,  অনুরক্ত হওয়া, অমঙ্গল প্রার্থনা করা।  ⇒ অভিসঙ্গ। = কোধবসেন লগ্গতি ( দীর্ঘ নিকায় অর্থকথা) ।  ( জাতক)  অভিসজ্জতি কুপ্পতি।  = কোপেতি ( সংযুক্ত নিকায় অর্থকথা) । ( অতীত)  অভিসজ্জি ; ( জাতক)  অভিসজ্জি কুপ্পি ব্যাপাজ্জি; ( তুলনীয়)  বৌদ্ধ সংস্কৃত অবদান শতক — অভিষজ্যতে বুপ্যতি ব্যাপাদ্যতে। ( সপ্তমী) অভিসজে; কুজঝাপনবসেন লগ্গপেয্য ( ধর্মপদ অর্থকথা) ।  ( করণে)  অভিসজ্জেতি। ⇒ অভিসজ্জন,  অভিসজ্জনা।
অভিসজ্জন ( ক্লী)  ক্রোধ, বদমেজাজ। ⇒ অভিসঙ্গ ( ২)। ( বিণ)  ক্রোধী, ক্রুদ্ধ, বদমেজাজি।
অভিসজ্জনং ( ক্রি — বিণ)  ক্রুদ্ধভাবে, ক্রোধান্বিত হইয়া। 
অভিসজ্জনা ( স্ত্রী)  গালাগালি,  মন্দ কথা, অভিশাপ,  দূর্বাক্য, বদমেজাজ।  ( তুলনীয়)  অভিসজ্জতি। ( যৌগিকে)  অভিসজ্জনা বাচাভিলাপ; কিন্তু চুলনির্দেশের বর্ণনায় এবং বুদ্ধঘোষ কর্তৃক বর্ণনার ইহার অর্থ করা হইয়াছে — " সংলগ্নতা, আসক্তি, আনুরক্তি,  দৃঢ় অনুরাগ,  তীব্র আকাঙ্খা, তৃষ্ণা, ইচ্ছ ; = তণ্হা। ( সূত্রনিপাত অর্থকথা)  সিনেহবসেন অভিসজ্জনা।  অভিসজ্জতি শব্দে বুদ্ধঘোষ কর্তৃক ব্যাখ্যাত দীর্ঘ নিকায় অর্থকথার উদাহারণ " কোধবসেন লগ্গতি " তুলনীয়। 
অভিসজ্জনী ( স্ত্রী)  ( "বাচা " সম্বন্ধে বর্ণনায়)  গালাগালি, মন্দ কথা, দুর্বাক্য,  অভিশাপ, বদমেজাজ।  ⇒ অভিসজ্জনা। 
অভিসজ্জেতি ( ক্রি)  = অভিসজ্জতি।  ⇒ অভিসজ্জতি। 
অভিসঞ্চযতি ( ক্রি)  [ অভি + সঞ্চযতি ] = অভিসঞ্চিনাতি। ⇒ অভিসঞ্চিনাতি। 
অভিসঞ্চিনাতি ( ক্রি)  [ অভি + সঞ্চিনাতি ] রাশীকৃত করা বা হওয়া,  স্তূপীকৃত করা বা হওয়া, সঞ্চয় করা,  জমানো,  সংখ্যায় বা পরিমাণে বাড়ানো বা বেশি হওয়া,  সংগ্রহ করা, একত্র করা বা হওয়া,  অনুমান করা, টাকা আদায় করা, পুণ্য বৃদ্ধি করা।  ( অতীত)  অভিসঞ্চিনি। ( সপ্তমী)  অভিসঞ্চেয্যং অভিসঞ্চিনেয্যং ( বিমানবত্থু অর্থকথা) । 
অভিসঞ্চেতযিত ( বিণ)  আত্নজ্ঞানে উৎপন্ন, চেতনায় অবধারিত, প্রত্যক্ষজ্ঞানে জ্ঞাত, চিন্তিত, ইচ্ছাকৃত, বাগদত্ত, সন্কল্পিত, অভিপ্রেত, পরিকল্পিত।  ⇒অভিসঞ্চেতেতি।  ( ক্লী)   অভিপ্রায়, সন্কল্প, পরিকল্পনা, সিদ্ধান্ত, চিন্তা করিয়া অবধারণ। 
অভিসঞ্চেতেতি ( ক্রি)  [ অভি + সং + √চিত + এ ] আত্নজ্ঞানের মধ্যে আনা, চিন্তা করিয়া বাহির করা, অভিসন্ধি করা, ফন্দি করা, কৌশল করা, কল্পনা করা, আবিস্কার করা, উইল করিয়া দান করা, নক্সা করা, মনে মনে অঙ্কন করা।  ( অতীত)  অভিসঞ্চেতেসি।  ( অ- বিণ)  অভিসঞ্চেতযিত।
অভিসণ্ছন্ন ( বিণ)  = অভিচ্ছন্ন।  ⇒ অভিচ্ছন্ন। 
অভিসঞ্ঞা ( স্ত্রী)  অতি সংজ্ঞা, পূর্ণচেতনা,  পূর্ণসংজ্ঞা বা জ্ঞানশক্তি। ⇒ সঞ্ঞা ।  ( যৌগিকে)  অভিসঞ্ঞা নিরোধ — পূর্ণমাত্রায় সংজ্ঞালোপ ( দীর্ঘ নিকায়)  ; ( ইহা কিন্তু বৌদ্ধ মত নহে,  কোনো কোনো বিভ্রান্ত সাধকেরই  অভিমত) ।  ( বৌদ্ধমতে)  সঞ্ঞা - বেদযিত - নিরোধ ( যথাস্থানে অর্থ দ্রষ্টব্য) । 
অভিসঞ্ঞূহতি ( ক্রি)  [ অভি + সঞ্ঞূহতি অর্থাৎ সং + নি + উহতি ] রাশীকৃত করা, স্তূপীকৃত করা, একগ্রতা সাধন করা, ধ্যানে মগ্ন হওয়া।  ( তুলনীয়)  অভিসঙ্খিপতি। 
অভিসট ( বিণ)  ১ স্রোতবেগে প্রবাহিত, একত্রে আগত বা একস্থানে মিলিত।  = সন্নিপতিত।  ২ সমীপবর্তী হইয়াছে এমন,  নিকটে আগত, সাক্ষাৎকৃত, পরিদর্শনকৃত, দেখাশুনা হইয়াছে এরুপ, দেখিতে গিয়াছে বা আসিয়াছে এমন।  ⇒ অভিসরতি। 
অভিসন্ঠান ( বিণ)  = পচ্চগ্ঘ।  ⇒ পচ্চগ্ঘ। 
অভিসত্ত ( বিণ)  অভিশপ্ত, ঘৃণার্হ, তিরস্কৃত, গালিবর্ষিত,  অপযশকৃত,  নিন্দিত, ভৎসিত।  = অক্কুট্ঠ।  ⇒ অভিসপতি। 
অভিসত্থ ( বিণ)  অভিশপ্ত, ঘৃণার্হ।  = অভিসত্ত।  ⇒ অভিসংসতি।
অভিসদ্দহতি ( ক্রি)  [ ক্রি + সং + √দহ + অ ; সংস্কৃত — অভিশ্রদ্দধাতি ]  বিশ্বাস রাখা বা করা,  ( কিছুতে)  বিশ্বাস করা, সত্য বলিয়া ধারণা করা, আস্থা রাখা, অনুমান করা। ( অতীত)  অভিসদ্দহি।  ( সপ্তমী)  অভিসদ্দহেয্য ( পেতবত্থু) ; = পটিঞ্ঞেয্য ( পেতবত্থু অর্থকথা) । 
অভিসন্তপতি ( ক্রি)  [ অভি + সন্তপতি অথবা সন্তপ্পতি ] অতি সন্তপ্ত হওয়া, অত্যন্ত উত্তপ্ত হওয়া, অতিশয় শোকান্বিত হওয়া,  ঝলসাইয়া যাওয়া, শুষ্ক হওয়া, নষ্ট করা, ধ্বংস করা, ঈষৎ দগ্ধ করা।  ( অতীত)  অভিসন্তাপি। ( করণে)  অভিসন্তাপেতি। ⇒ সন্তপ্পতি। 
অভিসন্তাপেতি ( ক্রি)  [ অভি + সন্তাপেতি ] ঝলসাইয়া দেওয়া, শুকাইয়া দেওয়া, নষ্ট করানো,  ধ্বংস করানো, ঈষৎ দগ্ধ করানো, জ্বালাইয়া দেওয়া,  ভাজা,  ঝলসানো। ⇒ অভিসন্তপতি। ( অতীত)  অভিসন্তাপেসি। 
অভিসন্দ ( পুং)  ( তুলনীয় বৌদ্ধসংস্কৃত মহাবস্তু — অভিস্যন্দ)  ( পূর্ণ হইয়া)  বাহিয়া পড়া, উথলিয়া পড়া, জলপ্লাবন, আধিক্য,  উৎপাদ, উৎপন্ন দ্রব্য, নিঃসরণ, বহিপ্রবাহ, প্রবাহ,  সন্তানসন্তুতি, লাভ, প্রচার, সংস্করণ, বিচার্য বিষয়, ( কার্যের)  ফল, সমস্যা, ফলাফল। ( বি— ভা)  চত্তারো পুঞ্ঞাভিসন্দা কুসলাভিসন্দা -- চতুর্বিধ পুণ্যপ্রবাহ সদগুণোৎপাদ ( বা সৎগুণের ফল)  ( সংযুক্ত নিকায়) । ( মিলিন্দ প্রশ্ন)  কষ্মাভিসন্দ — কর্মফল বা কর্মের ফল। ( তুলনীয়)  অভিসন্দন। 
অভিসন্দতি ( ক্রি)  [ অভি + √সন্দ + অ ] প্রবাহিত হওয়া, বহা, দ্রব হওয়া, চলাচল হওয়া, (( লম্বমান))  হওয়া, ঝুলিয়া পড়া, নির্গত হওয়া, জোয়ার হওয়া, প্লাবিত হওয়া, ধীরে ধীরে প্রবাহিত হওয়া, নিঃসরণ হওয়া, চুয়াইয়া পড়া, ক্ষরিত হওয়া, চুয়ানো, ক্ষরণ করানো।  ( অতীত)  অভিসন্দি। ( অ — বিণ)  অভিসন্ন। 
অভিসন্দন ( ক্লী)  প্রবাহ,প্লাবন, আধিক্য, ফলাফল, ( কার্যের)  ফল, পরিণাম, প্রভাব, গুরুত্ব। ( পটিসম্ভিদামার্গ)  সুখস্স অভিসন্দন। = অভিসন্দ। 
অভিসন্দহতি ( ক্রি)  [ অভি + সং + √দহ + অ ] একত্রিত করা, মিল করা, একত্রে রাখা, প্রস্তুত করিয়া রাখা, সংযুক্ত করা, ঠিক মিল করা, সন্ধিতে সন্ধিতে ছেদন করা, সম্পূর্ণ মিল হওয়া।  ( অতীত)  অভিসন্দহি। ( ক্রি - বি)  অভিসন্ধায। 

অভিসন্দায ( ক্রি - বি)  একত্রে মিল করিয়া বা রাখিয়া,  সংযুক্ত করিয়া,  প্রস্তুত রাখিয়া, ঠিক মিল করিয়া, সম্পূর্ণ মিল হইয়া; ( পদান্বয়ী অব্যয়ভাব যুক্ত হইলে)  কারণবশত, কারণে,  দরুণ,  যেহেতু।  = পটিচ্চ।  ⇒ অভিসন্দহতি। 
অভিসন্দেতি ( ক্রি)  [ অভি + সন্দেতি ] প্লাবিত করা, অভিভূত করা, উথলিয়া পড়া, ছাপাইয়া উঠা, অধিক হওয়া, পূর্ণ করা বা হওয়া, ভরা বা ভরণ করা, ব্যাপ্ত করা, অনুপ্রবেশ করা। 
অভিসন্ধায় = পটিচ্চ। ⇒ পটিচ্চ, অভিসন্দায। 
অভিসন্ন ( বিণ)  প্রচুর, প্লাবিত, উথলিত, ক্ষরিত, প্রবাহিত,পূরিত, পূর্ণকৃত, পরিপূর্ণ, নিঃসারিত। ( বিনয়)  অভিসন্ন কায। ( জাতক)  অভিসন্ন পীতিযা।  ( মিলিন্দ প্রশ্ন)  দুগ্গন্ধাভিসন্ন। ⇒ অভিসন্দতি। 
অভিসন্নকায — রসপূর্ণ দেহ। ⇒ অভিসন্ন। 
অভিসন্ন পীতিযা — পরিপূর্ণ বা নিঃসারিত প্রীতিসমূহ। ⇒ অভিসন্ন। 
অভিসপতি ( ক্রি)  [ অভি + √সপ + অ ] অভিশাপ দেওয়া,  অত্যন্ত ঘৃণা করা, গালি দেওয়া, ভৎসনা করা, অভিশাপ হইয়া দাঁড়ানো, শপথ গ্রহণ করা।  ( অ — বিন)  অভিসত্ত। 
অভিসপন ( ক্লী)  শপথ, হলফ, অভিশাপ, মঙ্গলেচ্ছা, অমঙ্গলজনক বস্তু। ইহা " অভিসম্পন্ন " শব্দের পরিবর্তে পাঠ। 
অভিসপযং ( আক্রোশজনিত ব্যবহার)  অভিশাপ দিব। ⇒ অভিলাপ। 
অভিসপযাং = অভিসপযং। = অভিসপযং। 
অভিসপ্পাযং ( আক্রোশজনিত ব্যবহার)  অভিশাপের যোগ্য,  অভিশাপ দেওয়া উচিত।  ⇒ অভিলাপ। 
অভিসময ( পুং)  [ অভি + সময; তুলনীয় বৌদ্ধসংস্কৃত দিব্যাবদান — অভিসময ] পূর্ণতাপ্রাপ্ত হইয়া আসা,  অন্তর্দৃষ্টি বা পরিজ্ঞান লাভ,  বোধশক্তি, অন্তর্ভুক্তির ক্ষমতা বা অন্তর্ভুক্তি, বাস্তবে পরিণতি, উপলব্দি, আদায়করণ, পরিস্কারভাবে বুঝানো বা নিঃসন্দেহজ্ঞান, হস্তমুষ্টি, গ্রহণ,  আয়ত্ত, ধীশক্তি, প্রবেশ, তীক্ষ্ণবুদ্ধি, বিচক্ষণতা।  ( তুলনীয়)  অভিসমেতি, সমেতি। ( সংযুক্ত নিকায়)  অত্থভিসময -- বিচক্ষণতা বা ব্যুৎপত্তি আয়ত্তকরণ। ( অঙ্গুত্তর নিকায়)  অরিযসচ্চানং অভিসময — চারি আর্য্যসত্য বিষয়ে পূর্ণজ্ঞান লাভ।  ( সূত্রনিপাত)  সচ্চাভিসময = সচ্চাববোধ ( সূত্রনিপাত অর্থকথা) ।  ( মিলিন্দ প্রশ্ন)  চতুসচ্চাভিসময। ( সদ্ধম্মোপাযন)  সচ্চানং অভিসময।  ( সংযুক্ত নিকায়)  ধম্মাভিসময -- ধর্মবিষয়ে জ্ঞানের পূর্ণতা লাভ।  ( সংযুক্ত নিকায়)  সম্মা- মানাভিসময — মিথ্যামান বা অহঙ্কার সম্বন্ধে পূর্ণজ্ঞান।  ( সংযুক্ত নিকায়)  পঞ্ঞায - অভিসময; অভিসময সংযুক্ত। ( সূত্রনিপাত)  ফস্সাভিসময; ( সূত্রনিপাত অর্থকথা বর্ণনায়)  " ফস্স- নিরোধ" দ্বারা বাহ্যিক ইন্দ্রিয় দমন হয়, কিন্তু

আভ্যন্তরিক ক্রিয়া দমিত হয় না।  = সচ্চপটিবেধ ( বিমানবত্থু অর্থকথা);  ( বিমানবত্থু অর্থকথা)  ভাবনা - ভিসময; সচ্ছিকিরিযাভিসময। 
অভিসমাগচ্ছতি ( ক্রি)  [ অভি + সম + আগচ্ছতি ] সম্পূর্ণরুপে জ্ঞানপর্যায়ে আসা, সম্পূর্ণরুপে বুঝিতে পারা, সম্পূর্ণরুপে আয়ত্ব করা,  প্রভূত্ব করা। ( তুলনীয়)  অধিগচ্ছতি।  ( অ - বিণ)  অভিসমেচ্চ। 
অভিসমাচারিক ( বিণ)  [ অভি + সমাচারিক ] যাহা ক্ষুদ্রতর বা নিকৃষ্টতর নীতিশাস্ত্র, নীতিতত্ত্ব অথবা নীতিবিজ্ঞান - সম্বন্ধীয় অভ্যাসের অধিকারী, পুনঃপুন অভ্যাস বা ব্যবহার করা হইতেছে এমন, সংসৃষ্ট অথবা যে সৎস্বভাবের ((অত্যষ্প)) গ্রহণ করিবার আশা করা হইয়াছে,  সঙ্গত, স্বকীয়, ব্যক্তিগত, উপযুক্ত, উচিত।  ( বিনয়)  অভিসমাচারিক সিক্খা — উপযুক্ত শিক্ষা।  ( মধ্যম নিকায়)  অভিসমাচারিক ধম্ম — স্বকীয় বা ব্যক্তিগত ধর্ম।
অভিসমিক্খতি ( ক্রি)  [ অভি + সমিক্খতি ] দেখা, অবলোকন করা,  দৃষ্টিপাত করা, দর্শন করা,  দেখা করা, দৃষ্টি দেওয়া, বিশেষ করিয়া দেখা, বিবেচনা করা, সম্মান করা, ব্যবহার করা,  লক্ষ্য করা, মনোযোগ দেওয়া,  বর্ণনা করা,  ভদ্র ব্যবহার করা, শিষ্টতা প্রদর্শন করা।  ( অতীত)  অভিসমিক্খি; ( মধ্যম পুরুষ একবচনে)  অভিসমেক্খসে = ওলোকেসি।  ( ক্রি — বি)  অভিসমিক্খ, অভিসমেক্খ ( বৌদ্ধসংস্কৃত জাতকমালায় -- অভিসমীক্ষা),  সঞ্চিক্খ = পস্সিত্বা ( জাতক অর্থকথা) ; = দিম্বা ( জাতক অর্থকথা) ।  ( করণে)  অভিসমেক্খতি। 
অভিসমিচ্চ = অভিসমেচ্চ। ⇒ অভিসমেচ্চ। 
অভিসমিত = অভিসমেত।  ⇒ অভিসমেত। 
অভিসমেক্খতি ( ক্রি)  অভিসমিক্খতি। ⇒ অভিসমিক্খতি। 
অভিসমেচ্চ ( বিণ)  সম্পূর্ণরুপে জ্ঞাত, অনর্গল জ্ঞাত।  ⇒ অভিসমেতি,  অভিসমাগচ্ছতি।  ( ক্রি — বিণ)  = অবেচ্চ। ⇒ অবেচ্চ। ( ক্রি — বি)  জ্ঞাত হইয়া। ⇒ অভিসমেতি। 
অভিসমেত ( বিণ)  সম্পূর্ণরুপে আয়ত্ব বা হৃদয়ঙ্গম করা হইয়াছে এরুপ, সম্পূর্ণরুপে জ্ঞাত, অধিকৃত। ⇒ অভিসমেতি।  ( সংযুক্ত নিকায়)  ধম্মাডিসমেত। 
অভিসমেতাবিন ( বিণ)  সস্পূর্ণরুপে জ্ঞান বা বিচক্ষণতার অধিকারী, সত্য সম্বন্ধে সম্পূর্ণরুপে অন্তদৃষ্টির অধিকারী,  সম্পূর্ণরুপে অন্তদৃষ্টি বা পরিজ্ঞান যাহার অধিগত বা আয়ত্তাধীন হইয়াছে। 
অভিসমেতাবী ( বিণ)  অভিসমেতাবিন।  ⇒ অভিসমেতাবিন।
অভিসমেতি ( ক্রি)  [ অভি + সমেতি ] পৌছা,  নাগাল পাওয়া, প্রাপ্ত হওয়া, আয়ত্ত করা, অধিগত করা, জ্ঞাত হওয়া, হৃদয়ঙ্গম করা, বোঝা, সাধন করা, কার্যে পরিণত করা,  আদায় করা, লাভ করা,  স্থাবর সম্পত্তিতে পরিণত

করা, পাওয়া, আকঁড়াইয়া ধরা, গ্রহণ করা, বুঝিতে পারা, জানা, ধরিয়া লওয়া।  ( তুলনীয়)  অধিগচ্ছতি।  ( তুলনীয় বৌদ্ধ — সংস্কৃত দিব্যাবদান)  অভিসমযতি ; চতুরার্য সত্যানি অভিসমযতি।  ( মিলিন্দ প্রশ্ন)  চতুসচ্চাভিসমযং অভিসমেতি) ।  ( অভ্যাসগত অর্থে যৌগিকে)  অভিসস্বুজ্ঝতি অভিসমেতি ; অভিসস্বুজ্ঝিত্বা অভিসমেত্বা ( সংযুক্ত নিকায়) ।  ( অতীত)  অভিসমেসি; ( প্রথম পুরুষ বহুবচনে)  অভিসমিংসু,  অভিসমেসং।  ( অ- বিণ)  অভিসমেত ( সমেত; সংস্কৃত - সমিত) । ( ভবিষৎ)  অভিসমেস্সতি।  ( ক্রি - বি)  অভিসমেত্বা, অভিসমাগন্ত্বা ( খুদ্দকপাঠ অর্থকথা)  ; " অভিসমিচ্চ" শব্দের পরিবর্তে " অভিসমেত্বা " শব্দের প্রভাবে ক্রিয়াবাচক বিশেষ্য " অভিসমেচ্চ" শব্দের ব্যবহারও দৃষ্ট হয়; ( অঙ্গুত্তর নিকায়)  সম্ম'ত্থ অভিসমেচ্চ — অনর্গলভাবে অথবা সস্পূর্ণরুপে প্রকৃত অর্থ জ্ঞাত হইয়া।
অভিসম্পন্ন ( এই শব্দটি " অভিসপন " শব্দের পরিবর্তে পেতবত্থু অর্থকথায় পাঠ দৃষ্ট হয়, কিন্তু এই পাঠ ভূল) । = অভিসপন। = অভিসপন। 
অভিসম্পরায ( পুং)  [ অভি + সস্পরায ] ভবিষৎ জন্ম, পরবর্তী ভব, মৃত্যুর পর উৎপত্তি ভব, ভবিষৎ অদৃষ্ট বা ভাগ্য, অদৃষ্ট, ভাগ্য, নিয়তি, চরম ফল, মৃত্যুর পরবর্তী অবস্থা,  পুনর্জন্মে ভবিষৎ অবস্থা।  ( বি - ভা) কা গতি কো অভিসম্পরাযো -- কোথায় গতি হইবে আর কোথায়ই বা পুনরায় জন্মগ্রহণ করিতে হইবে অথবা অদৃষ্টজনক উৎপত্তি ভব কিরুপ হইবে?  ( দীর্ঘনিকায়) । এবং গতি এবং অভিসস্পরায — ( বিণ)  এরুপ গতিতে গমনকারী আর এরুপ স্থানে বা অবস্থায় পুনর্বার জন্মগ্রহণকারী, এরুপ এরুপ গতির আর এরুপ এরুপ ভবিষৎ ভবিষ্যৎ জন্মাবস্থার ( দীর্ঘনিকায়) ।  = এবং - বিধা পরলোকা'তি ( দীর্ঘনিকায় অর্থকথা) । 
অভিসম্পরাযং ( ক্রি - বিণ) ভবিষ্যতে, মৃত্যুর পর। = পুনভবে ( পেতবত্থু অর্থকথা) ।  ( অঙ্গুত্তর নিকায়)  দিট্ঠে চেব ধস্মে অভিসস্পরাযঞ্চ — এই সংসারে এবং মৃত্যুর পর ভবিষ্যৎ জন্মে।  ⇒ দিট্ঠ।  ( পেতবত্থু অর্কথায় " অদৃষ্ট বা ভাগ্য" অর্থেও ইহার ব্যবহার দৃষ্ট হয়। 
অভিসম্বাবাধতা = অতিসম্বাধতা।  ⇒ অতিসম্বাধতা। 
অভিসম্বুজ্ঝতি ( ক্রি)  [ অভি + সং + √বুধ + য ] অতি উচ্চজ্ঞান লাভ করা, সম্বোধি জ্ঞান লাভ করা, সম্যক রুপে বোধিজ্ঞান লাভ করা, চরম বা পরম জ্ঞান লাভ করা, উচ্চতম সম্যক সম্বোধিজ্ঞান লাভ করা, সম্পূর্ণরুপে জাগ্রত হওয়া, পরম জ্ঞানে জাগ্রত হওয়া।  ( যৌগিক)  অভিসম্বুজ্ঝতি অভিসমেতি ( সংযুক্ত নিকায়)। ( অতীত)  অভিসস্বুজ্ঝি। ( অ- বিণ)  অভিসম্বুদ্ধ। ( ব- বিণ)  অভিসস্বুদ্ধান।  ( করণে)  অভিসস্বোধেতি। 
অভিসম্বুজ্ঝন ( ক্লী)  অভিসম্বোধি। ⇒ অভিসম্বোধি। 
অভিসম্বুদ্ধ ( বিণ)  জ্ঞাত,  অভিজ্ঞাত, বুঝিয়াছে বা হৃদয়ঙ্গম করিয়াছে এমন,  উপলব্দ, সাধিত, সাধনকৃত, সম্পূর্ণরুপে
জ্ঞাত, জ্ঞানশ্রেষ্ঠ জ্ঞানপ্রাপ্ত, জ্ঞানের সাধনায় চরমে পৌছিয়াছেন এমন,  সর্ববিষয়ে সম্পূর্ণরুপে জাগ্রত, বুদ্ধত্বপ্রাপ্ত, সাধনায় পরিপূর্ণ, মহাজ্ঞানী, ((জ্যোতিষ্মান)) ,  সম্যক রুপে বোধিজ্ঞান প্রাপ্ত।  ⇒ অভিসম্বুজ্ঝতি। ( পুং)  মহাজ্ঞান, মহাজ্ঞানী বুদ্ধ,  সম্বোধিজ্ঞানে পূর্ণতালাভ,  সম্যক রুপে বোধিজ্ঞান লাভ। 
অভিসম্বুদ্ধত্ত ( ক্লী)  পূর্ণজ্ঞান, মহাজ্ঞান, চরমজ্ঞান, পূর্ণাঙ্গ হৃদয়ঙ্গম, সাধনায় পূর্ণতা, অনর্গল উপলব্দি,  বুদ্ধত্ব জ্ঞান।
অভিসম্বুধান ( বিণ)  জাগ্রত হইয়া, সাধনা করিয়া, বুঝিয়া, জ্ঞাত হইয়া, হৃদয়ঙ্গম করিয়া,  জানিয়া। বুজ্ঝন্তো আনন্তো'তি অত্থো ( ধর্মপদ অর্থকথা)।  ⇒ অভিসম্বুজ্ঝতি। 
অভিসস্বোধি ( স্ত্রী)  [ অভি + সম্বোধি ] পরম জ্ঞান প্রাপ্তির অবস্থা, চরম জ্ঞানালোক প্রাপ্তির অবস্থা, বুদ্ধত্ব লাভের অবস্থা, বুদ্ধত্ব লাভ, সম্যক জ্ঞানের অধিকার লাভ, জ্ঞান পর্যায়ের চরম সীমায় উপস্থিতি।  ( জাতক)  পরমাভিসম্বোধি। ( চরিয়া পিটক)  অভিসম = ম্বুজ্ঝন, ( সম্মা)  সম্বোধি। 
অভিসস্বোধিত ( বিণ)  চরম জ্ঞানে জাগিয়াছে বা জাগ্রত হইয়াছে এমন, চরম জ্ঞানে জাগাইয়া তোলা হইয়াছে বা জাগ্রত করা হইয়াছে এমন ,  জ্ঞানবান বা ((জ্যোতিম্মান)) করা হইয়াছে এমন, পরম জ্ঞানে উন্নীত করা হইয়াছে এমন, চরম জ্ঞানে জাগ্রত বা ভূষিত, পরম জ্ঞানে জ্ঞাত বা অভিজ্ঞ। ( পেতবত্থু অর্থকথা)  ভগবা অভিসম্বোধিত। 
অভিসম্বোধেতি ( ক্রি)  পরম জ্ঞানে জাগাইয়া তোলা বা জাগ্রত করা,  জ্ঞানবান বা ((জ্যোতিষ্মান))  করা। ( অতীত)  অভিসস্বোধেসি। ( অ- বিণ)  অভিসস্বোধিত। ⇒ অভিসস্বুজ্ঝতি।
অভিসম্ভব — সম্ভব, সম্ভাবনা।  ( কেবল " দুর" শব্দের সহিত যৌগিকে)  দুরাভিসম্ভব — কষ্টসাধ্য। 
অভিসম্ভবতি ( ক্রি) [ অভি + সম্ভবতি ] ( পর্যন্ত আস' অর্থাৎ পাইতে,  দাড়াইতে অথবা জয় করিতে সমর্থ হওয়া — এই অর্থে)  লাভ করা, সাধন করা, চেষ্টা করিয়া অর্জন করা, পাওয়া, পৌছানো, উপস্থিত হওয়া, প্রসারিত করা বা হওয়া, হাত বাড়াইয়া কোনো কিছু পাওয়া বা স্পর্শ করা, নাগাল পাওয়া, উপার্জন করা, প্রাপ্ত হওয়া, ধারণ করা,  বহন করা। ( অতীত)  অভিসম্ভবি,  অভিসম্ভোসি। ( অ- বিণ)  অভিসম্ভুত।  ( ভবিষ্যতে)  অভিসম্ভোস্সং = সহিস্সামি, অধিবাস্সেসামি। ( ক্রি - বিণ)  অভিসম্ভুত্বা,  অভিসম্ভবিত্বা। ( ব - বিণ)  অভিসম্ভবনীয।  ⇒ অভিসম্ভুণাতি। 
অভিসম্ভুণাতি ( ক্রি)  [ অভি + √সম্ভ + উণা ] নাগাল পাইতে বা পৌছিতে সমর্থ হওয়া, লাভ করিতে সমর্থ হওয়া,  সমর্থ হওয়া, লাভ করা । ⇒ অভিসম্ভবতি। ( অতীত)  অভিসম্ভুণি। 
অভিসম্ভুনন্তো ( বিণ)  সমর্থ হইয়া,  সমর্থ। ⇒ অভিসম্ভুনাতি।
অভিসম্ভুনাতি ( ক্রি)  = অভিসম্ভুণাতি। ⇒ অভিসম্ভুণাতি। 
অভিসম্ভূ ( বিণ)  [ অভি + সম + ভূ ] প্রাপ্ত হইয়াছে বা লাভ হইয়াছে এমন, পাইয়া, লাভ করিয়া। ( দীর্ঘনিকায়)  লোমহংস অভিসম্ভু।  ( পুং)  প্রাপ্তি,  প্রাপ্ত বস্তু, লাভ। 
অভিসম্ভূত ( বিণ)  লব্দ,  প্রাপ্ত, সাধিত, অর্জিত। ⇒ অভিসম্ভবতি। 
অভিসম্বোতি ( ক্রি)  = অভিসম্ভবতি।  ⇒ অভিসম্ভবতি।  ⇒ অভিসম্ভবতি। 
অভিসম্মতি ( ক্রি)  [ অভি + √সম + অ; সংস্কৃত -- অভিশম্যতি)  শেষ হওয়া,  ক্ষান্ত হওয়া, ত্যাগ করা, শেষ করা, বন্ধ করা, রুদ্ধ করা, প্রতিরোধ করা,  বাধা দেওয়া, গতি রুদ্ধ করা,  দমন করা, নিবৃত্ত করা, বন্ধ হওয়া, থামা, অল্পকাল থাকা বা বাস করা, উপশম করা, প্রশমিত করা, শান্ত করা, সান্ত্বনা দেওয়া, ঠান্ডা করা, স্থির করা, চুপ করানো, নিবৃত্ত করা।  ( অতীত)  অভিসম্মি।  ( অ- বিণ)  অভিসম্মন্তো। 
অভিসম্মন্তো ( বিণ)  শান্ত,উপশান্ত, রুদ্ধ, অল্পকাল স্থায়ী, ত্যক্ত, পরিত্যক্ত, সমাপ্ত, দমিত। ( জাতকে " অভিসম্মেন্তো শব্দের পরিবর্তে?  ইহা অনিশ্চিত পাঠ)। ⇒ অভিসম্মতি। 
অভিসম্মেন্তো = অভিসম্মন্তো। ⇒ অভিসম্মন্তো। 
অভিসর — অনুসরণকারী লোকগণ, বড়লোকের অনুচরবর্গ, পরিষদবর্গ, সঙ্গীগণ। 
অভিসরতি ( ক্রি)  [ অভি + সরতি ] ১ ( জলস্রোত)  বেগে প্রবাহিত হওয়া, একত্রে আসা বা একস্থানে মিলিত হওয়া। = সন্নিপততি। ২ অনুসরণ করা,  অনুগমন করা, যাইয়া বা আসিয়া সাক্ষাৎ করা,  দেখিতে যাওয়া বা আসা, দেখাশুনা করা, তদারক করা, পরিদর্শন করা, সাক্ষাৎ করা, সাক্ষাৎ করিবার জন্য যাওয়া, ( অপ্রচলিত)  দগু দেওয়া, সীমাপবর্তী হওয়া, নিকটে আসা, তুল্য হওয়া, নিকটে যাইয়া বলা।  ( অতীত)  অভিসরি। ( অ- বিণ)  অভিসেট। 
অভিসল্লেখিক ( বিণ)  [ অভি + সল্লেখ + ইক ] কঠোর, রুঢ়, উগ্র, অত্যন্ত সাদাসিদে,  বাহুল্য বর্জিত,  কঠিন, দারুণ, নির্দয়, দৃঢ়,  শক্ত, স্থির, অপরিবর্তনীয়, কটু। 
অভিসল্লেখিকা ( স্ত্রী)  কঠোরতা, রুঢ়তা, আত্ননিগ্রহ, তপশ্চর্যা, কঠিনতা,  কর্কশতা, কটুতা।  এই শব্দটি সচরাচর " কথা" সম্বন্ধে ব্যবহৃত হইয়া থাকে। 
অভিসবতি ( ক্রি)  [ অভি + সবতি ] ( কিছুর)  দিকে বা মধ্যে প্রবাহিত হওয়া।  ( জাতক)  নজ্জো গঙ্গং অভিবসতি। 
অভিসসি ( অতীত ক্রিয়া)  ⇒ অভিসংসতি।। 
অভিসেকমঙ্গল ( ক্লী)  অভিষেক মঙ্গল, উৎসর্গ, পূত বা পবিত্র করার কার্য। ⇒ মঙ্গল ( ক্লী) । 
অভিস্সবতি ( ক্রি)  = অভিসবতি। ⇒ অভিসবতি। 
অভিসহতি ( ক্রি)  = অভিসম্ভুণাতি।  ⇒ অভিসম্ভুণাতি।
অভিসহমানো ( বিণ)  = অসম্পাপুণন্তো। ⇒ অসম্পাপুণন্তো। 
অভিসাধেতি ( ক্রি)  [ অভি + সাধেতি ] সম্পন্ন করা, শ্রেণীবদ্ধ করা,  সাজানো,  বন্দোবস্ত করা, নিষ্পত্তি করা, পাওয়া,  লাভ করা, জন্ম দেওয়া, শিক্ষা করা, প্রবর্তিত করা, ঘটানো,  আহার্য প্রস্তুত করা বা ভোজন করা, রোগাক্রান্ত হওয়া, কোনো কিছু সম্পন্ন করানো, ( চলতি)  শাস্তি বা দগু পাওয়া,  জব্দ করা, বুঝিতে পারা, উপস্থিত হওয়া, কোনো অবস্থায় উপস্থিত হওয়া,স্বচেষ্টায় প্রাপ্ত হওয়া,  সাধন করা, চেষ্টা করিয়া অর্জন করা, পৌছানো।  ( অতীত)  অধিসাধেসি। 
অভিসাপ ( পুং)  অভিশাপ, অমঙ্গলেচ্ছা, অমঙ্গলজনক বস্তু, ধর্মযাজকের অভিসম্পাতগ্রস্ত ব্যাক্তি বা বস্তু, অভিসম্পাত, ধর্মযাজকের অভিসম্পাত। ⇒ অভিলাপ। ( অঙ্গুত্তর নিকায়ে অক্কোস রুপেও ইহাকে বর্ণনা করা হইয়াছে। 
অভিসাপাযং — অভিশাপ ( আক্রোশজনিত শব্দবিশেষ) । ⇒ অভিসাপ, অভিলাপ। 
অভিসারিকা ( স্ত্রী)  অভিসারিকা, বেশ্যা, গণিকা, বারবনিতা,  বারাঙ্গনা ।  ⇒ অভিসারিযা। 
অভিসারিযা ( স্ত্রী)  প্রণয়ীর উদ্দেশ্যে গৃহ হইতে সঙ্কেতস্থানে গমনকারিণী নারী। = অভিসারিকা। 
অভিসারেতি ( স্ত্রী)  [ অভি + সারেতি ] সমীপবর্তী হওয়া,  নিকটে আসা, তুল্য হওয়া, নিকটে যাইয়া বলা, উত্ত্যক্ত করা, নির্যাতন করা, নিগৃহীত করা, হয়রান করা,  উৎপীড়ন করা। ( অতীত)  অভিসারেসি।
অভিসিংসতি ( ক্রি)  [ অভি + সিংসতি ] পবিত্র ধর্মসংক্রান্ত বাক্য উচ্চারণ করা অথবা ইচ্ছা প্রকাশ করা,  শ্রদ্ধারুপ ভয়মিশ্রিত বাক্য উচ্চারণ করা অথবা ইচ্ছা প্রকাশ করা। ( তুলনীয়) অভিসংসতি।  ( অতীত)  অভিসিংসিসি,  অভিসিংসীসি ; ( বিমানবত্থু অর্থকথা বর্ণনা)  ইচ্ছি সম্পটিছি। 
অভিসিঞ্চতি ( ক্রি)  [ অভি + √সিচ + ং + অ ] অভিসিঞ্চন করা, সিঞ্চন করা, ( কিছুর উপর)  ছিটাইয়া দেওয়া; ( চরিত্রে)  অভিষেক করা, বিন্দু বিন্দু বারিপাত করা, দেবসেবায় উৎসর্গ করা, পবিত্র করা। ( অঙ্গুত্তর নিকায়) খত্তিযাভিসেকেন অভিসিঞ্চতি। ( অতীত)  অভিসিঞ্চি। ( মিলিন্দ প্রশ্ন)  অমতেন লোকং অভিসিঞ্চি ভগবা। ( পেতবত্থু অর্থকথায় পাঠ)  অভিসিঞ্চি চিমিল্লিক ঞ্চ। ( অ - বিণ)  অভিসিত্ত ।  ( ক্রি - বি)  অভিসিঞ্চিত্বা। ( কর্মবাচ্যে)  অভিসিঞ্চতি। ( করণে)  অভিসিঞ্চেতি,  অভিসেচেতি। 
অভিসিঞ্চেতি ( ক্রি)  অভিসিঞ্চন করা, সিঞ্চন করা, অভিষেক করা। ( অতীত)  অভিসিঞ্চেতি। ( পঞ্চমী)  অভিসেচযস্সূ। ⇒ অভিসিঞ্চতি।
অভিসিত্ত ( বিণ)  ১ অভিষিক্ত, স্নাত, বিধিপূর্বক পবিত্র জলদি দ্বারা স্নাপিত, কৃতাভিষিক্ত, শাস্ত্রমতে যথাবিধি
দীক্ষিত, কোনো পদে যথা — বিধি প্রতিষ্ঠিত, পূত, উৎসর্গ বা পবিত্র করা হইয়াছে এমন। ( অঙ্গুত্তর নিকায়)  খত্তিযো খত্তিযেহি খত্তিযাভিসেকেন অভিসিত্ত। অঙ্গুত্তর নিকায়ে " অবসিত্ত " শব্দের পরিবর্তে " অভিসিত্ত" শব্দের ব্যবহার দৃষ্ট হয়। ⇒ অবসিত্ত। ২ জল বা তৈল ছিটানো দেওয়া হইয়াছে এমন। ( সূত্রনিপাত)  অভিসিত্ত মনসা। ( মিলিন্দ প্রশ্ন)  অমতেন লোক অভিসিত্ত। ⇒ অভিসিঞ্চতি। 
অভিসেক ( পুং)  ( রাজ)  অভিষেক, স্লান, যথাবিধি মন্ত্র উচ্চারণপূর্বক পবিত্র জলাদি দ্বারা স্লাপন,  রাজার প্রথম সিংহাসনে আরোহণকালীন অনুষ্ঠানবিশেষ, রাজ্যাধিকারার্থ অভিষেচন, কর্মে নিয়োগ, স্লাপন, উৎসর্গ, পবিত্র বা পূত করার কার্য, পুণ্যস্লান, সাধারণ স্লান বা গাত্র ধোয়া, প্রতিষ্ঠা।  ( তুলনীয়)  অভিসিত্ত।  ( অঙ্গুত্তর নিকায়) অভিসেক — অনভিসিত্ত। ( ব্যাকরণে)  অভিসেকিক। 
অভিসেকিক ( বিণ)  = অভিসিত্ত। ⇒ অভিসিত্ত,  অভিসেক। 
অভিসেচন ( ক্লী)  অভিষেচন,  অভিষেক, যথাবিধি তীর্থজলাদি দ্বারা স্লান,  স্লাপন, কর্মে নিয়োগ, পুণ্য স্লান, পবিত্র করার কার্য, ধৌতকরণ, উৎসর্গ, সাধারণ স্লান বা গাত্র ধৌতকরণ। = অভিসেক। ( থেরীগাথা)  উদক অভিসেচন। 
অভিসেবনা ( স্ত্রী)  [ অভি + সেবন ] পশ্চাদ্ধাবন, অনুসরণ, সাধনা,  কার্য, ব্যবসায়, প্রসাদ, চরিতার্থতা সাধন, প্রশয়দান, পাপের জন্য দগুের ক্ষমা, অমিতাচার। ( সদ্ধম্মোপাযন)  পাপকম্ম অভিসেবন। 
অভিস্সর ( বিণ)  [ অভি + ইস্সর ] ঈশ্বর, আশ্রয়, দাতা, প্রভু, কর্তা। ⇒ অনভিস্সর। 
অভিহংসতি ( ক্রি)  [ অভি + হংসতি ] ১ ( সকর্মে)  হৃষ্ট করা, প্রফুল্ল করা, উল্লসিত করা, সন্তুষ্ট করা, পরিতৃপ্ত করা, প্রত্যয় জন্মানো,  নিঃসন্দেহ করা, সন্দেহ ভঞ্জন করা, পরিশোধ করা, পূর্ণ করা, মেটানো। ( সংযুক্ত নিকায়)  অভিহট্ঠুং অভিহংসতি। ২ ( অকর্মে)  ( কোনো বিষয়ে)  আনন্দিত হওয়া, সুখভোগ করা, কেবল হাসিতে থাকা, আনন্দ অনুভব করা, উপভোগ করা। ( সংযুক্ত নিকায়)  রুপং মনাপং অভিহংসতি -- মনোজ্ঞ রুপে ( সৌন্দর্যে)  আনন্দ অনুভব করা। ( ব- বিণ)  অভিহিসমান, অভিহংসমান। 
অভিহংসমান ( বিণ)  আনন্দ অনুভব করে বা করিতেছে এমন, হাস্যমান, কেবল হাসিতে থাকে এমন।  ( অঙ্গুত্তর নিকায়)  অভিহংসমান ঝানং — যে ধ্যান অবস্থায় আনন্দ অনুভব করিতে থাকে অথবা কেবল হাসিতে থাকে। ⇒ অভিহংসতি। 
অভিহট ( বিণ)  আনীত, প্রদত্ত, দেওয়া হইয়াছে এমন, পুরস্কৃত,  বিক্রীত, প্রাপ্ত, আকর্ষিত। = পুরেতং গহেত্বা আহতন ভিক্খং ( পুগ্গল পঞ্ঞতি অর্থকথা) । ⇒ অভিহরতি।

অভিহট্ঠুং ( ক্রি - বি)  নিকটে আনা। = অভিহরিতুং। ( বি- ভা)  অভিহট্ঠুং পবারেতি — আনিয়া দেওয়া, আনিয়া দান করা ( মধ্যম নিকায়) । ⇒ অভিহরতি। 
অভিহত ( বিণ)  আক্রান্ত, প্রহৃত, নিগৃহীত, আহত, অগ্রবতী, সমীপবতী, স্পশিত, আঘাতপ্রাপ্ত।  ⇒ অভিহনতি। ( ক্লী)  প্রহার, আঘাত। 
অভিহনতি ( ক্রি)  [ অভি + √হন + অ ] ১ প্রহার করা, আঘাত করা, মারা, নিগৃহীত করা। ২ বলে বশীভূত করা, জয় করা, পরাভূত করা, বধ করা, ধ্বংস করা।  ( অতীত)  অভিহনি। ( অ - বিণ)  অভিহত। ( তুম)  " অভিহন্তুং" শব্দের পরিবর্তে জাতকে " অভিহন্তু"। 
অভিহরতি ( ক্রি)  [ অভি + √হর + অ; তুলনীয় সংস্কৃত -- অভ্যাহরতি, বৈদিক -- আহরতি, আভরতি ] ১ নিকটে আনা, প্রদান করা, দান করা, যাইয়া আনয়ন করা, প্রাপ্ত হওয়া, ( মূল্যে)  বিক্রয় হওয়া, আকর্ষণ করা। ২ অভিশাপ দেওয়া, গালি দেওয়া, ( উপাসনাগার হইতে)  বঞ্চিত করা, অভিশাপ হইয়া দাঁড়ানো, গালাগালি করা, অপযশ করা, অপব্যয় করা, নিয়মভঙ্গ করা, বিকৃত করা। সংস্কৃত — অনুব্যাহরতি, অভিব্যাহরতি। ⇒ আভত। ( অতীত)  অভিহরি। ( অ - বিণ)  অভিহট। ( ক্রি - বি)  অভিহট্ঠুং। ( কর্মবাচ্য)  অভিহরিযতি। ( তুম)  অভিহরিতুং। ( করণে)  অভিহারেতি। 
অভিহরিতুং ( তুম)  নিকটে আনিতে,  আনিতে। = অভিহট্ঠুং। ⇒ অভিহরতি। 
অভিহার ( পুং)  নিকটে আনয়নকরণ, যাইয়া আনয়নকরণ,  প্রদান, উপহার, বলি, পূজা, নৈবেদ্য, প্রস্তাবন দান, প্রকৃতিদত্ত গুণ,  উপঢৌকন। ( জাতক)  আসনাভিহার। 
অভিহারযং  ( ক্রি - বি)  আনিবার কারণ হওয়া, প্রাপ্ত হওয়া, লাভকরণ, অর্জনকরণ। এই শব্দটি টীকাকার " অভিভারযিন " শব্দরুপেও গ্রহণ করিয়াছেন। অভিহারেতি (১) শব্দে উদাহরণ দ্রষ্টব্য। 
অভিহারেতি ( ক্রি)  ১ আনিবার কারণ করা বা হওয়া, প্রাপ্ত হওয়া, লাভ করা, উপার্জন করা, অর্জন করা। ( জাতক)  অভিহারযং ( টীকায়)  অভিভারযিন। ২ নিজে ( কোনো উপায়)  অবলম্বন করা অথবা আশ্রয় গ্রহণ করা, গিয়া বা আসিয়া সাক্ষাৎ করা, দেখিতে যাওয়া বা আসা, দেখাশুনা করা, তদারক করা, পরিদর্শন করা, সাক্ষাৎ করা, সাক্ষাৎ করিবার জন্য যাওয়া, ( অপ্রচলিত)  দগু দেওয়া, লওয়া, লইয়া যাওয়া,ধরা, গ্রহণ করা, বাছিয়া বাছিয়া লওয়া, মনোনীত করা, প্রয়োগ করা, ব্যবহার বুঝিতে পারা, বিবেচনা করা, প্রয়োজন হওয়া, দরকার করা, বিমোহিত করা, বন্দী করা, দখল করা, সম্মত হওয়া, বোধ করা, অধিকারভুক্ত করা, কাড়িয়া লওয়া,  হরণ করা, বাদ দেওয়া, বাহির করা, সরানো,  সহ্য করা, ভক্ষণ করা, পান করা, শরীরে গ্রহণ করা, ( কোনো সংক্রামক রোগ দ্বারা)  আক্রান্ত হওয়া, সম্পাদন করা, ধারণা করা, গ্রাহক হওয়া, অতিক্রম করা, উদ্ধৃত করা, ফলপ্রদ হওয়া, গমন
করা, ঘটা, চলা, প্রস্থান করা, অগ্রসর হওয়া, প্রস্তুত হওয়া, যন্ত্রের সাহায্য চলা, রহিত হওয়া, দেহত্যাগ করা, অন্য অবস্থায় পরিণত হওয়া, কোনো বিশেষ অবস্থার অধীন হওয়া, উপযোগী হওয়া,বিস্তৃত হওয়া, কোনো লক্ষ্যের দিকে অগ্রসর হওয়া, বিক্রয় করা, প্রচলিত হওয়া, অতিবাহিত হওয়া।  ( সূত্রনিপাত)  অভিহারেসি — পাগুবশৈলে আরোহষ করিল; = আরুহি ( সূত্রনিপাত অর্থকথা) ।  ( সূত্রনিপাত)  বনন্তং অভিহারযে = বনং গচ্ছেয্য ( সূত্রনিপাত অর্থকথা) ।  ৩ ( ডাকে)  দেওয়া, রাখা।  ( অতীত)  অভিহারেসি,  অভিহারযিং; ( থেরীগাথা অর্থকথা) । ( ভবিষ্যৎ)  অভিহেস্সতি ; ( জাতক)  কবচং অভিহেস্সতি ( জাতক অর্থকথা বর্ণনায় ভুলবশত গ্রহণ করা হইয়াছে " অভিহনিস্সতি ভিন্দিস্সতি,  সুতরাং প্রত্যক্ষরুপে " অভিভবিস্সতি" গ্রহণ করা যাইতে পারে); ( জাতক) কিং যোব্বানেন চিণ্ণেন যং জরা অভিহেস্সতি? অভিহিংসতি ( ক্রি)  " অভিহিসতি " শব্দের পরিবর্তে অঙ্গুত্তর নিকায়ে পাঠ হইয়াছে " অভিহংসতি"। ⇒ অভিহংসতি।
অভিহিংসনা ( স্ত্রী)  হ্রেষা রব, ঘোড়ার ডাক। " অভিহেসনা " শব্দের পরিবর্তে ব্যবহার শব্দ; সংস্কৃত — হেষা, হেসিতং ; প্রাকৃত — হেসা। বিমানবত্থু অর্থকথার টীকায় " অভিহেসন"।  ⇒ অভিসংসনা।

কণ্টকি ( স্ত্রী) কাঁটাগাছের বেড়া, কাঁটাযুক্ত, কণ্টকময়।

কণ্ঠ ( পুং)  ১ কণ্ঠ, গলা,  টুটি, কণ্ঠধ্বনি, গলদেশ। ( মিলিন্দ প্রশ্ন)  কণ্ঠো আকুরতি — গলাভাঙ্গা শব্দ করা। ( পেতবত্থু অর্থকথা)  অক্খরানি মহতা কণ্ঠেন উচ্চরিতানি। ( প্রেতগণের চিক্কণ এবং পিপাসায় শুষ্ক কণ্ঠের অবস্থা বর্ণনায়ঃ —) কণ্ঠ— ওট্ঠ — তালুনংতস্সিত; সূচিকণ্ঠ; সূচিকট্ঠ; বিসুক্খ কণ্ঠ— ওট্ঠ - জিহবা ( পেতবত্থু অর্থকথা) ।  ২ গ্রীবা, গলা, ঘাড়, গলার লম্বা সঙ্কীর্ণ অংশ, গলার দীর্ঘ অপ্রশস্ত অংশ। = গ্রীবা। ( ধর্মপদ)  কাসাবকণ্ঠ। ( বিমানবত্থু অর্থকথা বর্ণনায়)  গীবূপগ সীসূপগাদি অভরণানি। ⇒ কণ্ঠে, কণ্ঠেবর।
