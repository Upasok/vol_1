অ_৭ 
( বিনয় )। 
অভিরুলহ ( বিণ)  আরুঢ়, আরোহিত, আরোহণ করিয়াছে এমন, চড়িয়াছে বা উঠিয়াছে এমন। ⇒ অভিরুহতি। 
অভিরুহতি ( ক্রি)  [ অভি + √রুহ + অ ] আরোহণ করা,উপরে উঠা, চড়া, পর্বতারোহণ করা, বৃক্ষরোহণ করা,  অশ্বারোহণ করা, উর্ধ্বে উত্তোলন করা, অশ্বপৃষ্ঠে স্থাপন করা, উভয়পার্শ্বে পা দিয়া উপবেশন করা, যথাযথভাবে সজ্জিত করা, বাধানো কিছুর উপর স্থাপন করা, উর্ধ্বে গমন করা,  অত্যন্ত হওয়া, ওঠা, উচুতে ওঠা,  চেষ্টা দ্বারা উচুতে ওঠা,  উপরে যাওয়া। ( জাতক)  নাবং অভিরুহতি।( অতীত) অভিরুহি, অভিরুচ্ছি।  ( অ বিণ)  অভিরুলহ। ( পষ্ণমী)  অভিরুহ। ( ক্রি - বি)  অভিরুযহ, অভিরুহিত্বা; ( পেতবত্থু) অভিরুযহিত্ব। ( পেতবত্থু অর্থকথা) নাবং অভিরুযহিত্ব; ( জাতক)  পব্বতং অভিরুহিত্বা।
অভিরুহন ( ক্লী)  আরোহণ, চড়ন, উপরে ওঠা, অধিরোহণ, উর্ধ্বে গমন।
অভিরোচেতি ( ক্রি) [ অভি + √রুচ + এ ] ১ পছন্দ করা,অনুমোদন করা,  মনোনীত করা, উপভোগ করা, ( কিছুতে)  সন্তুষ্ট হওয়া বা আনন্দিত হওয়া,  তৃপ্ত হওয়া,  ইচ্ছা করা, আকাঙ্কা করা, পাইতে চাওয়া,  তৃষ্ণা উৎপন্ন করা, আসক্ত হওয়া।  ( বিমানবত্থু)  বতং অভিরোচযি = অভিরোচেসি রুচ্চিত্বা পূরেসী'তি অত্থো; অভিরাধযী'তি পি পাঠো ; সাধেসি নিপ্ফাদেসী'তি অত্থো ( বিমানবত্থু অর্থকথা) ।  ২ সন্তুষ্ট করা, পরিতৃপ্ত করা, প্রত্যয় জন্মানো,  নিঃসন্দেহ করা, সনদেহ ভঞ্জন করা, পরিশোধ করা,  পূর্ণ করা, মেটানো, পোষণ করা, আতিথ্য প্রদর্শন করা, আমোদিত করা,  হৃদয়ে পোষণ করা,হৃষ্ট করা, প্রফুল্ল করা, উল্লাসিত করা।  ৩ " অতিরোচেতি" শব্দের পরিবর্তে বিমানবত্থুতে " অভিরোচেতি" ব্যবহার দৃষ্ট হয়।  ( বিমানবত্থু অর্থকথা)  অভিভবিত্বা বিজ্জোততি ( সমার্থক) ।  ( অ— বিণ) অভিরুচিত।
অভিরোপন (ক্লী)  অভিরোপণ, মনের একাগ্রতা সাধন,  মন সমাবেশকরণ, মনকে কেন্দ্রীভূত করণ, মনকে সমাহরণকরণ,  মনকে ঘনীভূত করণ, মনোযোগ, মনোসংযোগ, মনোনিবেশ।  ( পরিসম্ভিদা মার্গ)  অভিনিরোপন; বুদ্ধি অভিরোপন; অভিরোপণ বিরাগ; অভিরোপন বিমুত্তি ; অভিরোপন অভিসময। ⇒ অভিনিরোপন। 
অভিরোপেতি ( ক্রি)  [ অভি + রোপেতি; সংস্কৃত— অভিরোপয়তি)  মনের একাগ্রতা সাধন করা,  মনোনিবেশ করা, মনোযোগ দেওয়া, মনোসংযোগ করা, মনকে কেন্দ্রীভূত করা,  মনকে সমাহরণ করা, মনকে একত্রীভূত করা বা ঘনীভূত করা, অভিরোপণ করা, মন সমাবেশ করা, ভক্তি দেখানো, শ্রদ্ধা প্রদর্শন করা, মর্যাদা দেওয়া,  সম্মান প্রদর্শন করা,  সম্মান করা, পূজা করা, পদোন্নতি করা, মহিমান্বিত করা, (( হুন্ডি)) প্রভৃতি)  গ্রাহ্য করিয়া
সময়মত টাকা দেওয়া। ( অতীত)  অভিরোপযি, অভিরোপেসি = পূজং কারেসি, পূজেসি। 

অভিলকখিত ( বিণ)  চিহ্নিত, নির্দিষ্ট, লক্ষ্য স্থাপিত, অঙ্কিত, প্রতিষ্টিত, অভিষিক্ত, শুভলক্ষণ বলিয়া চিহ্নিত। ⇒ অভিলকখেতি। 
অভিলকখিতত্ত (ক্লী)  চিহ্নিত বা দাগ দেওয়া হইয়াছে এরুপ বিষয় বা বস্তু, বিশেষ গুণ দ্বারা বর্ণিত ব্যক্তি বা বিষয়,  চিহ্নিত বস্তু, প্রভেদ করা হইয়াছে এমন ব্যক্তি বা বস্তু, বিশেষ গুণসমূহ, লক্ষণসমূহ। 
অভিলকখেতি ( ক্রি)  [ অভি + √লকখ + এ ] অভিষেক করা,  প্রতিষ্ঠা করা,  আরম্ভ করা,  নক্সা করা, অঙ্কিত করা, দাগ কাটা, প্যাটার্ন তোলা,ফুল-পাতা কাটা বা বোনা, অভিসন্ধি করা, চিহ্নিত করা, দাগ দেওয়া, দেখা, লক্ষ্য করা, মনোযোগ দেওয়া, দৃঢ় করা, সংলগ্ন করা, স্থাপন করা, প্রতিষ্ঠিত করা, আবদ্ধ করা, দাম দেওয়া, তারিখ স্থির বা নির্দিষ্ট করা।  ( অতীত)  অভিলকখেসি।  ( অ - বিণ) অভিলকখিত। 
ভিলঙ্ঘতি ( ক্রি) [ অভি + লঙ্ঘতি ] আরোহণ করা, উপরে উঠা, উঠা, চড়া, ফাঁপিয়া উঠা, জন্মানো, উত্থিত হওয়া, লম্ফ দেওয়া, দৃষ্টিগোচর হওয়া, আসিয়া পড়া, উত্তেজিত হওয়া, বিদ্রোহী হওয়া, বৃদ্ধি পাওয়া,  পর্যটন করা, ভ্রমণ করা,  গমন করা, ভ্রমণ করিয়া অতিক্রম করা, লঙ্ঘন করা, উপেক্ষা করা, পার হইয়া যাওয়া,  ছাড়িয়া যাওয়া। 
অভিলম্বতি ( ক্রি)  [ অভি + লম্বতি ] ( কিছুর উপর)  ঝুলিয়া পড়া। = অভিলম্বতি অজঝোলম্বতি  ( নেত্তি) । ( জাতক) পপাতং অভিলম্বতি ; বেতরনিং অভিলম্বতি। ( অ — বিণ)  অভিলম্বিত।
অভিলম্বিত ( বিণ)  নীচের দিকে ঝুলিয়া পড়িয়াছে এমন। ( জাতক)  নীলদুম অভিলম্বিত।  ⇒ অভিলম্বতি। 
অভিলাপ ( পুং)  কথা, বাক্য, বিশেষ ভাষা, কথাবার্তা,  আলাপ, ভাষণ,  কথা কথনের ধারা, বর্ণনা ।  ( সূত্রনিপাত)  বাচাভিলাপ —— আপত্তিকর কথা, বাজে কথা; তিরচ্ছানকথা ( চুলনির্দেশ)  সূত্রনিপাতে " অভিলাপাযং" পাঠ অনিশ্চিত; অভিপাযং, অভিপাপাযং, অভিসাপাযং, অভিসপাযং, অতিসপ্পাযং পাঠও দৃষ্ট হয়। সংযুক্ত নিকায়ে এই সম্পর্কিত পাঠ আছে " অভিসপযং" যাহার অর্থ " অভিশপ; সংযুক্তনিকায় অর্থকথায় বর্ণনা দেওয়া হইয়াছে " অবিলাপো'তি অক্কোসো।  " অধিবচন" এর শব্দান্তরে অর্থ প্রকাশক শব্দস্বরুপ ধর্মসঙ্গণী ও চুলনির্দেশে উল্লিখিত সমার্থবাচক যৌগিক শব্দ " অভিলাপ ব্যঞ্জন" এর অর্থ Mrs. Rh. Davids এর মতে " কথাবার্তা বা আক্রোশজনিত বাক্যে পার্থক্যমূলক কোনো চিহ্ন অথবা যে বিশিষ্ট চিহ্ন নিয়া আলাপ করা হয়।
অভিলাপাযং ( অনিশ্চিত পাঠ)  অপবাদ। ইহা

আক্রোশজনিত শব্দবিশেষ। ⇒ অভিলাপ। 
অভিলাস ( সংস্কৃত — অভিলাষ)  অভিলাষ, ইচ্ছা, লালসা, অনুরাগ, লোভ, তৃষ্ণা, বাসনা। 
অভিলিখতি ( ক্রি)  [ অভি + লিখতি)  লেখা, খোদাই করা।  ( অ — বিণ) অভিলিখতি।  ( করণে)  অভিলেখতি। 
অভিলিখিত ( বিণ)  লিখিত, খোদিত, খোদাইকৃত। ⇒ অভিলিখতি,  অভিলেখতি। 
অভিলেখ — অভিলেখ, আলেখ্য, খোদিত লিপি, লিখিত লিপি।
অভিলেখতি ( ক্রি)  লেখা, অঙ্কিত করা, খোদিত করা, অঙ্কন করা, খোদানো।  ( দাঠাবংস)  চারিত্র — লেখং অভিলেখযি। 
অভিলেপন ( ক্লী)  [ অভি + লেপন ] চর্বি ইত্যাদি লেপন, দাগ, চিহ্ন, কলঙ্ক, নিন্দা, লজ্জা, অপবিত্রকরণ, অবিশুদ্ধতা, অপরিছন্নতা।  = লগ্গন, বন্ধন, উপক্কিলেস। 
অভিবগ্গ [ অভি + বগ্গ ] মহাজনতা, অধিকতর সৈন্য। (বি - ভা)  অভিবগ্গেন ওমদ্দতি — অধিকতর সৈন্য দ্বারা বা ক্ষমতাবলে মর্দিত করা বা দমন করা ( মধ্যম নিকায়)। 
অভিঞ্চন ( ক্লী)  [ অভি + বঞ্চ ] প্রবঞ্চনা, চাতুরী, কপটতা, ছল, ছলনা, প্রবঞ্চক। 
অভিবট্ট = অভিবট্ঠ।  ⇒ অভিবট্ঠ। 
অভিবট্ঠ ( বিণ)  অভিবর্ষিত, বৃষ্টিপাত হইয়াছে এমন,  ঢালিয়াছে এরুপ, ( কিছুর উপর)  ঢালিয়াছে বা বর্ষিত হইয়াছে এরুপ।  ( ধর্মপদ অর্থকথার টাকার)  অভিবুট্ঠ।⇒ অভিবস্সতি। 
অভিবড্ঢ ( বিণ)  অভিবর্ষিত, বর্ষিত, ছাড়াইয়া উঠিয়াছে এমন।  ( ধর্মপদ)  অভিবড্ঢ বীরণতিণ — ধারণতৃণ বা বীণাঘাস বৃদ্ধিপ্রাপ্ত হইয়াছে এরুপ। ⇒ অভিবড্ঢতি। 
অভিবড্ঢতি ( ক্রি)  [ অভি + √বড্ঢ + অ; বৈদিক -- অভিবর্ধতি ]  বৃদ্ধি পাওয়া বা হওয়া,  বাড়া, বর্ধিত হওয়া, (অন্যান্য বৃক্ষাদিকে ছাড়াইয়া বা আবৃত করিয়া তৃণাদি) অতিরিক্ত বাড়িয়া উঠা, ছাড়াইয়া উঠা,  অতিশয় বর্ধিত হওয়া, অতিরিক্ত বৃদ্ধি পাওয়া, বৃদ্ধি পাইয়া অতিক্রম করা।  (বিপ)  হায়তি। (অঙ্গুত্তর নিকায়)  ভোগা অভিবড্ঢতি।  ( জাতক)  বনস্পতিং অভিবড্ঢতি। ( অতীত)  অভিবড্ঢি। ( অ— বিণ)  অভিবড্ঢ, অভিবুড্ঢ।
অভিবড্ঢন ( ক্লী)  বৃদ্ধি, ক্রমবৃদ্ধি, ক্রমিক বৃদ্ধি, উন্নতি, পুষ্টিসাধন, জন্মানো, লাভ, বংশ, উৎপন্ন দ্রব্য, সন্তান সন্ততি, ক্রমবর্ধমান, বর্ধন।  ( বিণ)  বর্ধিত, উন্নত, পুষ্ট, লব্দ।
অভিবড্ঢি ( স্ত্রী)  বৃদ্ধি, জন্মানো। = অভিবড্ঢন।

অভিবণ্ণযি ( অতীত ক্রিয়া)  প্রশংসা করিলেন, গুণকীর্তন করিলেন, ব্যাখ্যা করিলেন।  ⇒ অভিবণ্ণেতি। 
অভিবণ্ণিত (বিণ)  প্রশংসিত, গুণকীর্তিত, ব্যাখ্যাত, গুণবর্ণিত।  ⇒ অভিবণ্ণেতি। 
অভিবণ্ণেতি ( ক্রি)  [ অভি + √বণ্ণ + এ ] প্রশংসা করা, গুণকীর্তন করা, গুণবর্ণনা করা, ব্যাখ্যা করা।  ( অতীত)  অভিবণ্ণেসি, অভিবণ্ণযি।  ( অ — বিণ)  অভিবণ্ণিত। 
অভিবত্ত (বিণ)  = অভিবট্ঠ। ⇒ অভিবট্ঠ।  অভিবর্ষিত। 
অভিবত্থ ( বিণ)  = অভিবট্ঠ। ⇒ অভিবট্ঠ।  অভিবর্ষিত। 
অভিবদতি ( ক্রি) [ অভি + বদতি ]  ১ কাহার ও উদ্দেশ্যে বলা বা লেখা, বলিয়া প্রকাশ করা, বলা, কহা, জ্ঞাপন করা, প্রচার করা, প্রকাশ করা, বক্তব্য প্রকাশ করা, অঙ্গীকার করা, স্বীকার করা, আশা দেওয়া,  দিতে প্রতিশ্রুত হওয়া।  ২ অনুগ্রহ করিয়া বলা, সাদরে অভ্যর্থনা করা, সাদর সম্ভাষণ করা, 'আসুন আসুন' বলা, 'আসিতে আজ্ঞা হউক' বলা, 'স্বাগত স্বাগত' বলা,  অভিবাদন করা, অভ্যর্থনা করা, চুম্বন করা, সম্ভাষণ করা,।  ( প্রায়শ যৌগিকে)  অভিবদতি অভিনন্দতি ( মধ্যম নিকায়) ।  (অতীত) অভিবদি।  ( করণে)  অভিবাদেতি। 
অভিবদ্ধ ( বিণ)  = \ অভিবড্ঢ।  ⇒ অভিবড্ঢ, অভিবদ্ধতি। 
অভিবদ্ধতি ( ক্রি)  [ অভি + বদ্ধতি ] = অভিবড্ঢতি। ⇒ অভিবড্ঢতি।  ( অ — বিণ)  অভিবদ্ধ। 
অভিবদ্ধন ( ক্লী)  = অভিবড্ঢন। ⇒ অভিবড্ঢন। 
অভিবদ্ধি ( স্ত্রী)  = অভিবড্ঢি।  ⇒ অভিবড্ঢি। 
অভিবন্দতি ( ক্রি)  [ অভি + √বন্দ + অ ] বন্দনা করা, নমস্কার করা, প্রণাম করা, প্রণিপাত করা, সম্যক আরাধানা করা, অভিবন্দনা করা, সম্মুখে বসিয়া পূজা করা, অভিবাদন করা, সম্মান  প্রদর্শন করা।  ( অতীত)  অভিবন্দি। ( অ— বিণ)  অভিবন্দিত।  ( ব — বিণ)  অভিবন্দনীয়।  ( ক্রি — বি) অভিবন্দিয, অভিবাদিয।  ( করণে)  অভিবাদেতি,  অভিবাদাপেতি। 
অভিবন্দনা ( স্ত্রী)  = অভিবাদন।  ⇒ অভিবাদন। 
অভিবন্দিত ( বিণ)  বন্দনা করা হয় এমন,  অভিবাদিত, অভ্যর্থিত, বন্দিত,  পূজিত, পূজ্য,  আরাধিত,  সম্মানিত,  সম্মান প্রদর্শিত।  ⇒ অভিবন্দতি। 
অভিবন্দিয ( ক্রি বি)  বন্দনা করিয়া,  পূজা করিয়া,  অভিবাদন করিয়া, আরাধনা করিয়া, সম্মান প্রদর্শন করিয়া,  অভ্যর্থনা করিয়া।  = অভিবাদিত। ⇒ অভিবন্দতি। 
অভিবসতি ( ক্রি)  [ অভি + বসতি ] বাস করা,  থাকা, অবস্থান করা। 
অভিবস্সক ( বিণ)  বৃষ্টিপাত করে বা হয় এমন,  বর্ষণকারী, খসিয়া পড়ে বা ঝড়িয়া পড়ে এমন ; ( চরিত্রে)  রক্তপাত

করে বা হয় এমন, অশ্রুবর্ষণকারী, রক্তপাতকারী, পরিপূর্ণ করে বা ঢালে এমন, নমনীয়, একগুয়ে নয় এমন,বাধ্য,বশ্যতা স্বীকারকারী।  ( বিমানবত্থু অর্থকথা) পুপ্ফাভিবস্সক। 
অভিবস্সতি (ক্রি)  [ অভি + √বস্স + অ ] অভিবর্ষণ করা, অতিবর্ষণ করা, বর্ষণ করা, অতিবৃষ্টি হওয়া, প্রচুর বৃষ্টিপাত হওয়া, অধিক পরিমাণে জল ঢালা, পূর্ণ করিয়া দেওয়া; ( চরিত্রে)  রক্তপাত করা, অশ্রুবিসর্জন করা, বৃষ্টি পড়া, পূর্ণ করা, ঢালা। ( দীর্ঘনিকায় )  আভিবস্সং। ( জাতক)  পুপ্ফাভিবস্সতি — স্রোতের ন্যায় পুষ্প বর্ষিত হওয়া৷ ( তুলনীয়)  অতিবস্সতি। ( অতীত)  অভিবস্সি৷ ( অ— বিণ)  অভিবট্ট, অভিবট্ঠ, অভিবুট্ঠ, অভিবুত্থ, অভিবত্ত, অভিবত্থ৷ ( করণে)  অভিবস্সাপেতি। 
অভিবস্সাপেতি (ক্রি) ( আকাশে)  বৃষ্টিপাতের কারণ করা। ⇒ অভিবস্সতি। 
অভিবস্সিন (বিণ)  = অভিস্সক।  ⇒ অভিবস্সক৷ সব্বত্থ অভিবস্সিন। 
অভিবাদন ( ক্লী)  অভিবাদন, " নমো " উচ্চারণপূর্বক নমস্কার, পদদ্বয় স্পর্শ করিয়া প্রণতি, বন্দনা,  শ্রদ্ধাভিবাদন, সাদর অভ্যর্থনাকরণ, স্বাগত জানানো, সম্মান প্রদর্শনকরণ৷ ( ধর্মপদ)  অভিবাদনসীলিন। ( সদ্ধম্মোপাযন)  অভিবাদনসীল। 
অভিবাদনসীল ( বিণ)  অভিবাদনশীল,  ধর্মনিষ্ঠ, ন্যায়নিষ্ঠ, শ্রদ্ধাশীল, শ্রদ্ধাবিশিষ্ট,  ভক্ত, শ্রদ্ধাপ্রবণ। ⇒ অভিবাদন। 
অভিবাদনসীলিন ( বিণ)  অভিবাদনশীল, ধর্মনিষ্ঠ স্বভাবসম্পন্ন, ন্যায়নিষ্ঠ স্বভাবশীল, শ্রদ্ধাযুক্ত আচারবিশিষ্ট। = অভিবাদনশীল। 
অভিবাদয ( পঞ্চমী)  অভিবাদন করা, নমস্কার করা। = বন্দ। ⇒ বন্দতি, অভিবাদেতি। 
অভিবাদাপেতি ( ক্রি)  কাহারও দ্বারা নমস্কার করানো অথবা সম্ভাষণ করানো।  ⇒ অভিবাদেতি। 
অভিবাদিত ( বিণ)  অভিবাদিত, পূজিত,  নমস্কারকৃত, অভ্যর্থিত, সম্ভাষিত, সম্মান প্রদর্শিত,  সাদরে অভ্যর্থিত। ⇒ অভিবাদেতি। 
অভিবাদিয = অভিবন্দিয। ⇒ অভিবন্দতি। 
অভিবাদেতি (ক্রি)  অভিবাদন করা, প্রণাম করা, নমস্কার করা, ভূলুন্ঠিত নমস্কার করা, হাত জোর করিয়া নমস্কার করা, করজোড়ে নমস্কার করা, কৃতাঞ্জলি নমস্কার করা, পূজা করা, সম্মান করা, সাদর অভ্যর্থনা করা, সম্ভাষণ করা। ⇒ অভিবন্দতি।  ( বিদায়কালীন সম্ভাষণে)  অভিবাদেতি পদকখিণং করোতি ( দীর্ঘনিকায়) । ( অতীত)  অভিবাদেসি,  অভিবাদযিং = অভিবাদনং কারেসিং বন্দিং ( বিমানবত্থু অর্থকথা) ।  ( অ — বিণ)  অভিবাদিত। ( ভ— বিণ)  অভিবাদেতব্ব।  (পঞ্চমী) অভিবাদয।  ( করণে)  অভিবাদাপেতি।

অভিবাদেতব্ব  ( বিণ)  পূজার যোগ্য,  অভিবাদনের যোগ্য,  সম্মানের যোগ্য,  নমস্কার করা উচিত,  অভ্যর্থনা করা উচিত। ⇒ অভিবাদেতি। 
অভিবাযতি (ক্রি)  [ অভি + বাযতি; সংস্কৃত — অভিবাতি ] জোরে প্রবাহিত হওয়া,  প্রবল বায়ু বহা,  তুফান হওয়া,  ঝড় বহা, ব্যাপ্ত করা,  অনুপ্রবেশ করা। = অতিবাযতি। 
অভিবারেতি ( ক্রি)  [ অভি + বারেতি ] ফিরাইয়া ধরা, বারণ করা, দিতে বা লইতে অমত করা,  ' না ' বলা,  অস্বীকার করা, অসত্য বা মিথ্যা বলিয়া প্রতিবাদ করা, ( কিছু হইতে)  বঞ্চিত করা। = নিবারেতি। 
অভিবাহ ( বিণ)  বাহিত, পরিচালিত।। 
অভিবাহেতি ( ক্রি)  [ অভি + বাহেতি ] নড়ানো,  সরাইয়া ফেলা, অপসারিত করা, তুলিয়া লওয়া,  কর্মচ্যুত করা,  পদচ্যুত করা, প্রস্থান করা, ঘর বদল করা, সরাইয়া রাখা, লুক্কাইয়া রাখা, গোপন করা। 
অভিবিজযতি ( ক্রি)  [ অভি + বিজযতি ] বলে বশীভূত করা,  জয় করা, পরাভূত করা, পরাজিত করা, বশীভূত করা, অতিক্রম করা, অধ্যবসায় দ্বারা লাভ করা, জয়ী হওয়া ।  ( অতীত)  অভিবিজযি।  ( অ — বিণ)  অভিবিজিত। ( ক্রি — বি)  অভিবিজিয,  অভিবিজিত্বা। 
অভিবিজিত ( বিণ)  বিজিত, জয়কৃত, ।  ⇒ অভিবিজযতি, অভিবিজিনাতি। 
অভিবিজিনাতি ( ক্রি)  [ অভি + বি + √জি + না ] অভিবিজযতি ।  ⇒ অভিবিজযতি।  ( বর্তমান,  প্রথম পুরুষ  বহুবচন)  অভিবিজিনন্তি।  ( অতীত)  অভিবিজিনি।  ( অ — বিণ)  অভিবিজিত।  ( ক্রি — বি)  অভিবিজিনিত্বা। 
অভিবিঞ্ঞাপেতি ( ক্রি)  [ অভি + বিঞ্ঞাপেতি ] কাহারও মন আকর্ষণ করা অথবা পরিবর্তন করিয়া দেওয়া, ( কাহাকেও)  প্রবৃত্ত করা, প্রবর্তিত করা, বিজ্ঞাপিত করা, ইশারা করা, ইঙ্গিত করা। ( অতীত)  অভিবিঞ্ঞাপেসি।  ( বিনয়)  পুরাণ - দুতিযিকায মেথুনং ধম্মং অভিবিঞ্ঞাপেসি। 
অভিবিতরতি  ( ক্রি)  [ অভি + বিতরতি ] ( কিছুর)  নীচে যাওয়া, তলিয়া যাওয়া,  ভিতরে ঢুকাইয়া দেওয়া, মনোযোগ দেওয়া,  লক্ষ্য করা, যত্ন করা, অবলোকন করা, অবধান করা, পালন করা, মন্তব্য প্রকাশ করা। বিনয়ের বর্ণনায় " সঞ্চিচ্চ " শব্দ পাঠ দৃষ্ট হয়। 
অভিবিনয ( পুং)  [ অভি + বিনয় ] উচ্চশিক্ষা, উচ্চ ধরনের নিয়মানুবর্তিতা,  উচ্চ ধরনের অনুশাসন বা বিনয় অথবা শৃঙ্খলা,  অতিশয় আজ্ঞাধীনত্ব,  উচ্চ ধরনের আইন বা আদেশের অধীনতা,  উচ্চ ধরনের শাসন বা শাস্তি অথবা দন্ড, শিক্ষার বিশুদ্ধিতা, বিনয়ের বিশোধন।  ( যৌগিকে) অভিধম্ম অভিবিনয ( দীর্ঘ নিকায়) ; ( বিনয়)  বিনয অভিবিনয। 
অভিবন্দতি ( ক্রি)  [ অভি + বিন্দতি ] খুঁজিয়া পাওয়া, দেখিতে পাওয়া,  অনুভব করা,  অভিজ্ঞতা বা পরীক্ষার

অভিবিসিট্ঠ ( বিণ)  [ অভি + বিসিট্ঠ ] অতিবিশিষ্ট, অতি উত্তম,  অতি চমৎকার, অতি প্রসিদ্ধ, বিশেষরুপে চিহ্নিত।
অভিবিস্সজ্জতি ( ক্রি)  [ অভি + বিস্সজ্জতি ] বাহিরে প্রেরণ করা,  বাহিরে পাঠানো, প্রক্ষেপ করা, ছোড়া, কোনো সংবাদ বা সংবাদবাহক প্রেরণ করা, শীঘ্র পাঠানো, দেওয়া। 
অভিবিস্সত্থ ( বিণ)  অতি বিশ্বস্ত, বিশ্বাসে আনীত। ⇒ অভিবিস্সসতি।  ( মধ্যম নিকায়)  অভিবিসট্ঠ। 
অভিবিস্সসতি (ক্রি)  [ অভি + বিস্সসতি ] অতি বিশ্বাস করা, অতি বিশ্বাসের মধ্যে আনা। ( অতীত)  অভিবিস্সসি। 
অভিবিহচ্চ ( ক্রি — বি)  নষ্ট করিয়া, বিনাশ করিয়া,  সরাইয়া বা বহিষ্কৃত করিয়া,  তাড়াইয়া, বিদূরিত করিয়া, নির্বাসিত করিয়া, অপসারিত করিয়া, সূর্যের আলোতে অন্ধকার বিদূরিত করিয়া। 
অভিবিহনতি (ক্রি)  [ অভি + বিহনতি ] নষ্ট করা, বিনাশ করা, দূরে সরাইয়া দেওয়া, বহিষ্কার করা, তাড়াইয়া দেওয়া, বিদূরিত করা, নির্বাসিত করা, অপসারিত করা, সূর্যের অন্ধকার বিনষ্ট করা। 
অভিবুট্ঠ ( বিণ)  = অভিবট্ট।  ⇒ অভিবট্ট। 
অভিবুড্ঢ ( বিণ)  বর্ধিত,  বাড়ন্ত, বৃদ্ধিপ্রাপ্ত, ধনাঢ্য, সুশোভিত,  উর্বরা ।  ⇒ অবিবড্ঢতি। 
অভিবুড্ঢি ( স্ত্রী)  অভিবৃদ্ধি,  শ্রীবৃদ্ধি, বৃদ্ধি, উন্নতি, জন্মানো, উৎপাদন, সাফল্য, সৌভাগ্য।  ⇒ অভিবড্ঢি। 
অভিবুত্থ ( বিণ)  ঢালা হইয়াছে এরুপ, প্রবাহিত করা হইয়াছে এরুপ,  বেগে ধাবিত, প্রবাহিত, ঝড়িত, পূরিত, নিক্ষিপ্ত। = অভিবুট্ঠ। ⇒ অভিবস্সতি। 
অভিবুদ্ধ ( বিণ)  বুদ্ধিপ্রাপ্ত। = অভিবুড্ঢ।  ⇒ অভিবুড্ঢ, অভিবদ্ধতি।
অভিবুদ্ধি ( স্ত্রী)  = অভিবড্ঢি, অভিবুড্ঢি।  ⇒ অভিবড্ঢি, অভিবুড্ঢি। 
অভিবেঠেতি ( ক্রি)  [ অভি + বেঠেতি ] ( এই শব্দটি " অতিবেঠেতি " শব্দের পরি জাতকে ব্যবহার দৃষ্ট হয়, কিন্তু জাতক অর্থকথা বর্ণনার সহিত ইহার সামঞ্জস্য নাই) ।  চারিদিকে প্যাঁচাইয়া দেয় অথবা পাক দিয়া রাখে, গুটাইয়া ফেলে,  জড়াইয়া ফেলে।  ⇒ অতিবেঠেতি। 
অভিবেদেতি ( ক্রি)  [ অভি + বেদেতি ] ১ জানানো, সংবাদাদি প্রেরণ করা, পত্র ব্যবহার করা, সংবাদাদি আদান প্রদান করা,  সংলগ্ন থাকা। ২ জানা,  জ্ঞাত হওয়া।  = জানাতি।

অভিসংবিসতি ( ক্রি)  [ অভি + সংবিসতি ] প্রবেশ করা। ⇒ সংসবিসতি। 
অভিসংবিস্সেয্যগত্তং ( " অভিসং বিস্সেয্য — ভন্তং" অথবা অভিসংবিস্সেয্যসন্তং " ?)  Rh Davids বলেন এই শব্দটি এবং ইহার ব্যুৎপত্তি নির্ণয় ও অর্থ সন্দেহজনক, কিন্তু ধর্মপাল থেরীগাথায় ইহার অর্থ করিয়াছেন — " গলিত মাংসপূর্ণ চামড়ার থলি"।

অভিসঙ্গ ( পুং)  ১ অভিসঙ্গ,  সংলগ্নতা, আসক্তি,  অনুরক্তি, দৃঢ় অনুরাগ।  ( ধর্মসঙ্গণী অর্থকথা)  অভিসঙ্গ হেতুকং দুকখৎ; অভিসঙ্গরস। ( তুলনীয়)  অভিসজ্জতি; ( সংস্কৃত)  অভিসঙ্গ।  ২ বদমেজাজ, ক্রোধ,  গালাগালি, অভিশাপ। 
অভিসঙ্গিন ( বিণ)  অভিসঙ্গিন,  সংলগ্ন, আসক্ত, অনুরক্ত। 
অভিসজে ( ক্রিয়া সপ্তমী)  ক্রোধের সংলগ্ন হইলে,  ক্রোধের বশীভূত হইলে,  ক্রোধান্বিত হইলে।  = কুজঝাপন বসেন লগ্গপেয্য ( ধর্মপদ অর্থকথা) ।  ⇒ অভিসজ্জতি। 
অভিসজ্জতি ( ক্রি)  [ অভি + √সদ + অ ] বদমেজাজি হওয়া, ক্রুব্দ হওয়া, অভিশাপ দেওয়া,  গালাগালি করা, আসক্ত হওয়া, সংলগ্ন হওয়া,  অনুরক্ত হওয়া, অমঙ্গল প্রার্থনা করা।  ⇒ অভিসঙ্গ। = কোধবসেন লগ্গতি ( দীর্ঘ নিকায় অর্থকথা) ।  ( জাতক)  অভিসজ্জতি কুপ্পতি।  = কোপেতি ( সংযুক্ত নিকায় অর্থকথা) । ( অতীত)  অভিসজ্জি ; ( জাতক)  অভিসজ্জি কুপ্পি ব্যাপাজ্জি; ( তুলনীয়)  বৌদ্ধ সংস্কৃত অবদান শতক — অভিষজ্যতে বুপ্যতি ব্যাপাদ্যতে। ( সপ্তমী) অভিসজে; কুজঝাপনবসেন লগ্গপেয্য ( ধর্মপদ অর্থকথা) ।  ( করণে)  অভিসজ্জেতি। ⇒ অভিসজ্জন,  অভিসজ্জনা।
অভিসজ্জন ( ক্লী)  ক্রোধ, বদমেজাজ। ⇒ অভিসঙ্গ ( ২)। ( বিণ)  ক্রোধী, ক্রুদ্ধ, বদমেজাজি।
অভিসজ্জনং ( ক্রি — বিণ)  ক্রুদ্ধভাবে, ক্রোধান্বিত হইয়া। 
অভিসজ্জনা ( স্ত্রী)  গালাগালি,  মন্দ কথা, অভিশাপ,  দূর্বাক্য, বদমেজাজ।  ( তুলনীয়)  অভিসজ্জতি। ( যৌগিকে)  অভিসজ্জনা বাচাভিলাপ; কিন্তু চুলনির্দেশের বর্ণনায় এবং বুদ্ধঘোষ কর্তৃক বর্ণনার ইহার অর্থ করা হইয়াছে — " সংলগ্নতা, আসক্তি, আনুরক্তি,  দৃঢ় অনুরাগ,  তীব্র আকাঙ্খা, তৃষ্ণা, ইচ্ছ ; = তণ্হা। ( সূত্রনিপাত অর্থকথা)  সিনেহবসেন অভিসজ্জনা।  অভিসজ্জতি শব্দে বুদ্ধঘোষ কর্তৃক ব্যাখ্যাত দীর্ঘ নিকায় অর্থকথার উদাহারণ " কোধবসেন লগ্গতি " তুলনীয়। 
অভিসজ্জনী ( স্ত্রী)  ( "বাচা " সম্বন্ধে বর্ণনায়)  গালাগালি, মন্দ কথা, দুর্বাক্য,  অভিশাপ, বদমেজাজ।  ⇒ অভিসজ্জনা। 
অভিসজ্জেতি ( ক্রি)  = অভিসজ্জতি।  ⇒ অভিসজ্জতি। 
অভিসঞ্চযতি ( ক্রি)  [ অভি + সঞ্চযতি ] = অভিসঞ্চিনাতি। ⇒ অভিসঞ্চিনাতি। 
অভিসঞ্চিনাতি ( ক্রি)  [ অভি + সঞ্চিনাতি ] রাশীকৃত করা বা হওয়া,  স্তূপীকৃত করা বা হওয়া, সঞ্চয় করা,  জমানো,  সংখ্যায় বা পরিমাণে বাড়ানো বা বেশি হওয়া,  সংগ্রহ করা, একত্র করা বা হওয়া,  অনুমান করা, টাকা আদায় করা, পুণ্য বৃদ্ধি করা।  ( অতীত)  অভিসঞ্চিনি। ( সপ্তমী)  অভিসঞ্চেয্যং অভিসঞ্চিনেয্যং ( বিমানবত্থু অর্থকথা) । 
অভিসঞ্চেতযিত ( বিণ)  আত্নজ্ঞানে উৎপন্ন, চেতনায় অবধারিত, প্রত্যক্ষজ্ঞানে জ্ঞাত, চিন্তিত, ইচ্ছাকৃত, বাগদত্ত, সন্কল্পিত, অভিপ্রেত, পরিকল্পিত।  ⇒অভিসঞ্চেতেতি।  ( ক্লী)   অভিপ্রায়, সন্কল্প, পরিকল্পনা, সিদ্ধান্ত, চিন্তা করিয়া অবধারণ। 
অভিসঞ্চেতেতি ( ক্রি)  [ অভি + সং + √চিত + এ ] আত্নজ্ঞানের মধ্যে আনা, চিন্তা করিয়া বাহির করা, অভিসন্ধি করা, ফন্দি করা, কৌশল করা, কল্পনা করা, আবিস্কার করা, উইল করিয়া দান করা, নক্সা করা, মনে মনে অঙ্কন করা।  ( অতীত)  অভিসঞ্চেতেসি।  ( অ- বিণ)  অভিসঞ্চেতযিত।
অভিসণ্ছন্ন ( বিণ)  = অভিচ্ছন্ন।  ⇒ অভিচ্ছন্ন। 
অভিসঞ্ঞা ( স্ত্রী)  অতি সংজ্ঞা, পূর্ণচেতনা,  পূর্ণসংজ্ঞা বা জ্ঞানশক্তি। ⇒ সঞ্ঞা ।  ( যৌগিকে)  অভিসঞ্ঞা নিরোধ — পূর্ণমাত্রায় সংজ্ঞালোপ ( দীর্ঘ নিকায়)  ; ( ইহা কিন্তু বৌদ্ধ মত নহে,  কোনো কোনো বিভ্রান্ত সাধকেরই  অভিমত) ।  ( বৌদ্ধমতে)  সঞ্ঞা - বেদযিত - নিরোধ ( যথাস্থানে অর্থ দ্রষ্টব্য) । 
অভিসঞ্ঞূহতি ( ক্রি)  [ অভি + সঞ্ঞূহতি অর্থাৎ সং + নি + উহতি ] রাশীকৃত করা, স্তূপীকৃত করা, একগ্রতা সাধন করা, ধ্যানে মগ্ন হওয়া।  ( তুলনীয়)  অভিসঙ্খিপতি। 
অভিসট ( বিণ)  ১ স্রোতবেগে প্রবাহিত, একত্রে আগত বা একস্থানে মিলিত।  = সন্নিপতিত।  ২ সমীপবর্তী হইয়াছে এমন,  নিকটে আগত, সাক্ষাৎকৃত, পরিদর্শনকৃত, দেখাশুনা হইয়াছে এরুপ, দেখিতে গিয়াছে বা আসিয়াছে এমন।  ⇒ অভিসরতি। 
অভিসন্ঠান ( বিণ)  = পচ্চগ্ঘ।  ⇒ পচ্চগ্ঘ। 
অভিসত্ত ( বিণ)  অভিশপ্ত, ঘৃণার্হ, তিরস্কৃত, গালিবর্ষিত,  অপযশকৃত,  নিন্দিত, ভৎসিত।  = অক্কুট্ঠ।  ⇒ অভিসপতি। 
অভিসত্থ ( বিণ)  অভিশপ্ত, ঘৃণার্হ।  = অভিসত্ত।  ⇒ অভিসংসতি।
অভিসদ্দহতি ( ক্রি)  [ ক্রি + সং + √দহ + অ ; সংস্কৃত — অভিশ্রদ্দধাতি ]  বিশ্বাস রাখা বা করা,  ( কিছুতে)  বিশ্বাস করা, সত্য বলিয়া ধারণা করা, আস্থা রাখা, অনুমান করা। ( অতীত)  অভিসদ্দহি।  ( সপ্তমী)  অভিসদ্দহেয্য ( পেতবত্থু) ; = পটিঞ্ঞেয্য ( পেতবত্থু অর্থকথা) । 
অভিসন্তপতি ( ক্রি)  [ অভি + সন্তপতি অথবা সন্তপ্পতি ] অতি সন্তপ্ত হওয়া, অত্যন্ত উত্তপ্ত হওয়া, অতিশয় শোকান্বিত হওয়া,  ঝলসাইয়া যাওয়া, শুষ্ক হওয়া, নষ্ট করা, ধ্বংস করা, ঈষৎ দগ্ধ করা।  ( অতীত)  অভিসন্তাপি। ( করণে)  অভিসন্তাপেতি। ⇒ সন্তপ্পতি। 
অভিসন্তাপেতি ( ক্রি)  [ অভি + সন্তাপেতি ] ঝলসাইয়া দেওয়া, শুকাইয়া দেওয়া, নষ্ট করানো,  ধ্বংস করানো, ঈষৎ দগ্ধ করানো, জ্বালাইয়া দেওয়া,  ভাজা,  ঝলসানো। ⇒ অভিসন্তপতি। ( অতীত)  অভিসন্তাপেসি। 
অভিসন্দ ( পুং)  ( তুলনীয় বৌদ্ধসংস্কৃত মহাবস্তু — অভিস্যন্দ)  ( পূর্ণ হইয়া)  বাহিয়া পড়া, উথলিয়া পড়া, জলপ্লাবন, আধিক্য,  উৎপাদ, উৎপন্ন দ্রব্য, নিঃসরণ, বহিপ্রবাহ, প্রবাহ,  সন্তানসন্তুতি, লাভ, প্রচার, সংস্করণ, বিচার্য বিষয়, ( কার্যের)  ফল, সমস্যা, ফলাফল। ( বি— ভা)  চত্তারো পুঞ্ঞাভিসন্দা কুসলাভিসন্দা -- চতুর্বিধ পুণ্যপ্রবাহ সদগুণোৎপাদ ( বা সৎগুণের ফল)  ( সংযুক্ত নিকায়) । ( মিলিন্দ প্রশ্ন)  কষ্মাভিসন্দ — কর্মফল বা কর্মের ফল। ( তুলনীয়)  অভিসন্দন। 
অভিসন্দতি ( ক্রি)  [ অভি + √সন্দ + অ ] প্রবাহিত হওয়া, বহা, দ্রব হওয়া, চলাচল হওয়া, (( লম্বমান))  হওয়া, ঝুলিয়া পড়া, নির্গত হওয়া, জোয়ার হওয়া, প্লাবিত হওয়া, ধীরে ধীরে প্রবাহিত হওয়া, নিঃসরণ হওয়া, চুয়াইয়া পড়া, ক্ষরিত হওয়া, চুয়ানো, ক্ষরণ করানো।  ( অতীত)  অভিসন্দি। ( অ — বিণ)  অভিসন্ন। 
অভিসন্দন ( ক্লী)  প্রবাহ,প্লাবন, আধিক্য, ফলাফল, ( কার্যের)  ফল, পরিণাম, প্রভাব, গুরুত্ব। ( পটিসম্ভিদামার্গ)  সুখস্স অভিসন্দন। = অভিসন্দ। 
অভিসন্দহতি ( ক্রি)  [ অভি + সং + √দহ + অ ] একত্রিত করা, মিল করা, একত্রে রাখা, প্রস্তুত করিয়া রাখা, সংযুক্ত করা, ঠিক মিল করা, সন্ধিতে সন্ধিতে ছেদন করা, সম্পূর্ণ মিল হওয়া।  ( অতীত)  অভিসন্দহি। ( ক্রি - বি)  অভিসন্ধায। 

অভিসন্দায ( ক্রি - বি)  একত্রে মিল করিয়া বা রাখিয়া,  সংযুক্ত করিয়া,  প্রস্তুত রাখিয়া, ঠিক মিল করিয়া, সম্পূর্ণ মিল হইয়া; ( পদান্বয়ী অব্যয়ভাব যুক্ত হইলে)  কারণবশত, কারণে,  দরুণ,  যেহেতু।  = পটিচ্চ।  ⇒ অভিসন্দহতি। 
অভিসন্দেতি ( ক্রি)  [ অভি + সন্দেতি ] প্লাবিত করা, অভিভূত করা, উথলিয়া পড়া, ছাপাইয়া উঠা, অধিক হওয়া, পূর্ণ করা বা হওয়া, ভরা বা ভরণ করা, ব্যাপ্ত করা, অনুপ্রবেশ করা। 
অভিসন্ধায় = পটিচ্চ। ⇒ পটিচ্চ, অভিসন্দায। 
অভিসন্ন ( বিণ)  প্রচুর, প্লাবিত, উথলিত, ক্ষরিত, প্রবাহিত,পূরিত, পূর্ণকৃত, পরিপূর্ণ, নিঃসারিত। ( বিনয়)  অভিসন্ন কায। ( জাতক)  অভিসন্ন পীতিযা।  ( মিলিন্দ প্রশ্ন)  দুগ্গন্ধাভিসন্ন। ⇒ অভিসন্দতি। 
অভিসন্নকায — রসপূর্ণ দেহ। ⇒ অভিসন্ন। 
অভিসন্ন পীতিযা — পরিপূর্ণ বা নিঃসারিত প্রীতিসমূহ। ⇒ অভিসন্ন। 
অভিসপতি ( ক্রি)  [ অভি + √সপ + অ ] অভিশাপ দেওয়া,  অত্যন্ত ঘৃণা করা, গালি দেওয়া, ভৎসনা করা, অভিশাপ হইয়া দাঁড়ানো, শপথ গ্রহণ করা।  ( অ — বিন)  অভিসত্ত। 
অভিসপন ( ক্লী)  শপথ, হলফ, অভিশাপ, মঙ্গলেচ্ছা, অমঙ্গলজনক বস্তু। ইহা " অভিসম্পন্ন " শব্দের পরিবর্তে পাঠ। 
অভিসপযং ( আক্রোশজনিত ব্যবহার)  অভিশাপ দিব। ⇒ অভিলাপ। 
অভিসপযাং = অভিসপযং। = অভিসপযং। 
অভিসপ্পাযং ( আক্রোশজনিত ব্যবহার)  অভিশাপের যোগ্য,  অভিশাপ দেওয়া উচিত।  ⇒ অভিলাপ। 
অভিসময ( পুং)  [ অভি + সময; তুলনীয় বৌদ্ধসংস্কৃত দিব্যাবদান — অভিসময ] পূর্ণতাপ্রাপ্ত হইয়া আসা,  অন্তর্দৃষ্টি বা পরিজ্ঞান লাভ,  বোধশক্তি, অন্তর্ভুক্তির ক্ষমতা বা অন্তর্ভুক্তি, বাস্তবে পরিণতি, উপলব্দি, আদায়করণ, পরিস্কারভাবে বুঝানো বা নিঃসন্দেহজ্ঞান, হস্তমুষ্টি, গ্রহণ,  আয়ত্ত, ধীশক্তি, প্রবেশ, তীক্ষ্ণবুদ্ধি, বিচক্ষণতা।  ( তুলনীয়)  অভিসমেতি, সমেতি। ( সংযুক্ত নিকায়)  অত্থভিসময -- বিচক্ষণতা বা ব্যুৎপত্তি আয়ত্তকরণ। ( অঙ্গুত্তর নিকায়)  অরিযসচ্চানং অভিসময — চারি আর্য্যসত্য বিষয়ে পূর্ণজ্ঞান লাভ।  ( সূত্রনিপাত)  সচ্চাভিসময = সচ্চাববোধ ( সূত্রনিপাত অর্থকথা) ।  ( মিলিন্দ প্রশ্ন)  চতুসচ্চাভিসময। ( সদ্ধম্মোপাযন)  সচ্চানং অভিসময।  ( সংযুক্ত নিকায়)  ধম্মাভিসময -- ধর্মবিষয়ে জ্ঞানের পূর্ণতা লাভ।  ( সংযুক্ত নিকায়)  সম্মা- মানাভিসময — মিথ্যামান বা অহঙ্কার সম্বন্ধে পূর্ণজ্ঞান।  ( সংযুক্ত নিকায়)  পঞ্ঞায - অভিসময; অভিসময সংযুক্ত। ( সূত্রনিপাত)  ফস্সাভিসময; ( সূত্রনিপাত অর্থকথা বর্ণনায়)  " ফস্স- নিরোধ" দ্বারা বাহ্যিক ইন্দ্রিয় দমন হয়, কিন্তু

আভ্যন্তরিক ক্রিয়া দমিত হয় না।  = সচ্চপটিবেধ ( বিমানবত্থু অর্থকথা);  ( বিমানবত্থু অর্থকথা)  ভাবনা - ভিসময; সচ্ছিকিরিযাভিসময। 
অভিসমাগচ্ছতি ( ক্রি)  [ অভি + সম + আগচ্ছতি ] সম্পূর্ণরুপে জ্ঞানপর্যায়ে আসা, সম্পূর্ণরুপে বুঝিতে পারা, সম্পূর্ণরুপে আয়ত্ব করা,  প্রভূত্ব করা। ( তুলনীয়)  অধিগচ্ছতি।  ( অ - বিণ)  অভিসমেচ্চ। 
অভিসমাচারিক ( বিণ)  [ অভি + সমাচারিক ] যাহা ক্ষুদ্রতর বা নিকৃষ্টতর নীতিশাস্ত্র, নীতিতত্ত্ব অথবা নীতিবিজ্ঞান - সম্বন্ধীয় অভ্যাসের অধিকারী, পুনঃপুন অভ্যাস বা ব্যবহার করা হইতেছে এমন, সংসৃষ্ট অথবা যে সৎস্বভাবের ((অত্যষ্প)) গ্রহণ করিবার আশা করা হইয়াছে,  সঙ্গত, স্বকীয়, ব্যক্তিগত, উপযুক্ত, উচিত।  ( বিনয়)  অভিসমাচারিক সিক্খা — উপযুক্ত শিক্ষা।  ( মধ্যম নিকায়)  অভিসমাচারিক ধম্ম — স্বকীয় বা ব্যক্তিগত ধর্ম।
অভিসমিক্খতি ( ক্রি)  [ অভি + সমিক্খতি ] দেখা, অবলোকন করা,  দৃষ্টিপাত করা, দর্শন করা,  দেখা করা, দৃষ্টি দেওয়া, বিশেষ করিয়া দেখা, বিবেচনা করা, সম্মান করা, ব্যবহার করা,  লক্ষ্য করা, মনোযোগ দেওয়া,  বর্ণনা করা,  ভদ্র ব্যবহার করা, শিষ্টতা প্রদর্শন করা।  ( অতীত)  অভিসমিক্খি; ( মধ্যম পুরুষ একবচনে)  অভিসমেক্খসে = ওলোকেসি।  ( ক্রি — বি)  অভিসমিক্খ, অভিসমেক্খ ( বৌদ্ধসংস্কৃত জাতকমালায় -- অভিসমীক্ষা),  সঞ্চিক্খ = পস্সিত্বা ( জাতক অর্থকথা) ; = দিম্বা ( জাতক অর্থকথা) ।  ( করণে)  অভিসমেক্খতি। 
অভিসমিচ্চ = অভিসমেচ্চ। ⇒ অভিসমেচ্চ। 
অভিসমিত = অভিসমেত।  ⇒ অভিসমেত। 
অভিসমেক্খতি ( ক্রি)  অভিসমিক্খতি। ⇒ অভিসমিক্খতি। 
অভিসমেচ্চ ( বিণ)  সম্পূর্ণরুপে জ্ঞাত, অনর্গল জ্ঞাত।  ⇒ অভিসমেতি,  অভিসমাগচ্ছতি।  ( ক্রি — বিণ)  = অবেচ্চ। ⇒ অবেচ্চ। ( ক্রি — বি)  জ্ঞাত হইয়া। ⇒ অভিসমেতি। 
অভিসমেত ( বিণ)  সম্পূর্ণরুপে আয়ত্ব বা হৃদয়ঙ্গম করা হইয়াছে এরুপ, সম্পূর্ণরুপে জ্ঞাত, অধিকৃত। ⇒ অভিসমেতি।  ( সংযুক্ত নিকায়)  ধম্মাডিসমেত। 
অভিসমেতাবিন ( বিণ)  সস্পূর্ণরুপে জ্ঞান বা বিচক্ষণতার অধিকারী, সত্য সম্বন্ধে সম্পূর্ণরুপে অন্তদৃষ্টির অধিকারী,  সম্পূর্ণরুপে অন্তদৃষ্টি বা পরিজ্ঞান যাহার অধিগত বা আয়ত্তাধীন হইয়াছে। 
অভিসমেতাবী ( বিণ)  অভিসমেতাবিন।  ⇒ অভিসমেতাবিন।
অভিসমেতি ( ক্রি)  [ অভি + সমেতি ] পৌছা,  নাগাল পাওয়া, প্রাপ্ত হওয়া, আয়ত্ত করা, অধিগত করা, জ্ঞাত হওয়া, হৃদয়ঙ্গম করা, বোঝা, সাধন করা, কার্যে পরিণত করা,  আদায় করা, লাভ করা,  স্থাবর সম্পত্তিতে পরিণত

করা, পাওয়া, আকঁড়াইয়া ধরা, গ্রহণ করা, বুঝিতে পারা, জানা, ধরিয়া লওয়া।  ( তুলনীয়)  অধিগচ্ছতি।  ( তুলনীয় বৌদ্ধ — সংস্কৃত দিব্যাবদান)  অভিসমযতি ; চতুরার্য সত্যানি অভিসমযতি।  ( মিলিন্দ প্রশ্ন)  চতুসচ্চাভিসমযং অভিসমেতি) ।  ( অভ্যাসগত অর্থে যৌগিকে)  অভিসস্বুজ্ঝতি অভিসমেতি ; অভিসস্বুজ্ঝিত্বা অভিসমেত্বা ( সংযুক্ত নিকায়) ।  ( অতীত)  অভিসমেসি; ( প্রথম পুরুষ বহুবচনে)  অভিসমিংসু,  অভিসমেসং।  ( অ- বিণ)  অভিসমেত ( সমেত; সংস্কৃত - সমিত) । ( ভবিষৎ)  অভিসমেস্সতি।  ( ক্রি - বি)  অভিসমেত্বা, অভিসমাগন্ত্বা ( খুদ্দকপাঠ অর্থকথা)  ; " অভিসমিচ্চ" শব্দের পরিবর্তে " অভিসমেত্বা " শব্দের প্রভাবে ক্রিয়াবাচক বিশেষ্য " অভিসমেচ্চ" শব্দের ব্যবহারও দৃষ্ট হয়; ( অঙ্গুত্তর নিকায়)  সম্ম'ত্থ অভিসমেচ্চ — অনর্গলভাবে অথবা সস্পূর্ণরুপে প্রকৃত অর্থ জ্ঞাত হইয়া।
অভিসম্পন্ন ( এই শব্দটি " অভিসপন " শব্দের পরিবর্তে পেতবত্থু অর্থকথায় পাঠ দৃষ্ট হয়, কিন্তু এই পাঠ ভূল) । = অভিসপন। = অভিসপন। 
অভিসম্পরায ( পুং)  [ অভি + সস্পরায ] ভবিষৎ জন্ম, পরবর্তী ভব, মৃত্যুর পর উৎপত্তি ভব, ভবিষৎ অদৃষ্ট বা ভাগ্য, অদৃষ্ট, ভাগ্য, নিয়তি, চরম ফল, মৃত্যুর পরবর্তী অবস্থা,  পুনর্জন্মে ভবিষৎ অবস্থা।  ( বি - ভা) কা গতি কো অভিসম্পরাযো -- কোথায় গতি হইবে আর কোথায়ই বা পুনরায় জন্মগ্রহণ করিতে হইবে অথবা অদৃষ্টজনক উৎপত্তি ভব কিরুপ হইবে?  ( দীর্ঘনিকায়) । এবং গতি এবং অভিসস্পরায — ( বিণ)  এরুপ গতিতে গমনকারী আর এরুপ স্থানে বা অবস্থায় পুনর্বার জন্মগ্রহণকারী, এরুপ এরুপ গতির আর এরুপ এরুপ ভবিষৎ ভবিষ্যৎ জন্মাবস্থার ( দীর্ঘনিকায়) ।  = এবং - বিধা পরলোকা'তি ( দীর্ঘনিকায় অর্থকথা) । 
অভিসম্পরাযং ( ক্রি - বিণ) ভবিষ্যতে, মৃত্যুর পর। = পুনভবে ( পেতবত্থু অর্থকথা) ।  ( অঙ্গুত্তর নিকায়)  দিট্ঠে চেব ধস্মে অভিসস্পরাযঞ্চ — এই সংসারে এবং মৃত্যুর পর ভবিষ্যৎ জন্মে।  ⇒ দিট্ঠ।  ( পেতবত্থু অর্কথায় " অদৃষ্ট বা ভাগ্য" অর্থেও ইহার ব্যবহার দৃষ্ট হয়। 
অভিসম্বাবাধতা = অতিসম্বাধতা।  ⇒ অতিসম্বাধতা। 
অভিসম্বুজ্ঝতি ( ক্রি)  [ অভি + সং + √বুধ + য ] অতি উচ্চজ্ঞান লাভ করা, সম্বোধি জ্ঞান লাভ করা, সম্যক রুপে বোধিজ্ঞান লাভ করা, চরম বা পরম জ্ঞান লাভ করা, উচ্চতম সম্যক সম্বোধিজ্ঞান লাভ করা, সম্পূর্ণরুপে জাগ্রত হওয়া, পরম জ্ঞানে জাগ্রত হওয়া।  ( যৌগিক)  অভিসম্বুজ্ঝতি অভিসমেতি ( সংযুক্ত নিকায়)। ( অতীত)  অভিসস্বুজ্ঝি। ( অ- বিণ)  অভিসম্বুদ্ধ। ( ব- বিণ)  অভিসস্বুদ্ধান।  ( করণে)  অভিসস্বোধেতি। 
অভিসম্বুজ্ঝন ( ক্লী)  অভিসম্বোধি। ⇒ অভিসম্বোধি। 
অভিসম্বুদ্ধ ( বিণ)  জ্ঞাত,  অভিজ্ঞাত, বুঝিয়াছে বা হৃদয়ঙ্গম করিয়াছে এমন,  উপলব্দ, সাধিত, সাধনকৃত, সম্পূর্ণরুপে
জ্ঞাত, জ্ঞানশ্রেষ্ঠ জ্ঞানপ্রাপ্ত, জ্ঞানের সাধনায় চরমে পৌছিয়াছেন এমন,  সর্ববিষয়ে সম্পূর্ণরুপে জাগ্রত, বুদ্ধত্বপ্রাপ্ত, সাধনায় পরিপূর্ণ, মহাজ্ঞানী, ((জ্যোতিষ্মান)) ,  সম্যক রুপে বোধিজ্ঞান প্রাপ্ত।  ⇒ অভিসম্বুজ্ঝতি। ( পুং)  মহাজ্ঞান, মহাজ্ঞানী বুদ্ধ,  সম্বোধিজ্ঞানে পূর্ণতালাভ,  সম্যক রুপে বোধিজ্ঞান লাভ। 
অভিসম্বুদ্ধত্ত ( ক্লী)  পূর্ণজ্ঞান, মহাজ্ঞান, চরমজ্ঞান, পূর্ণাঙ্গ হৃদয়ঙ্গম, সাধনায় পূর্ণতা, অনর্গল উপলব্দি,  বুদ্ধত্ব জ্ঞান।
অভিসম্বুধান ( বিণ)  জাগ্রত হইয়া, সাধনা করিয়া, বুঝিয়া, জ্ঞাত হইয়া, হৃদয়ঙ্গম করিয়া,  জানিয়া। বুজ্ঝন্তো আনন্তো'তি অত্থো ( ধর্মপদ অর্থকথা)।  ⇒ অভিসম্বুজ্ঝতি। 
অভিসস্বোধি ( স্ত্রী)  [ অভি + সম্বোধি ] পরম জ্ঞান প্রাপ্তির অবস্থা, চরম জ্ঞানালোক প্রাপ্তির অবস্থা, বুদ্ধত্ব লাভের অবস্থা, বুদ্ধত্ব লাভ, সম্যক জ্ঞানের অধিকার লাভ, জ্ঞান পর্যায়ের চরম সীমায় উপস্থিতি।  ( জাতক)  পরমাভিসম্বোধি। ( চরিয়া পিটক)  অভিসম = ম্বুজ্ঝন, ( সম্মা)  সম্বোধি। 
অভিসস্বোধিত ( বিণ)  চরম জ্ঞানে জাগিয়াছে বা জাগ্রত হইয়াছে এমন, চরম জ্ঞানে জাগাইয়া তোলা হইয়াছে বা জাগ্রত করা হইয়াছে এমন ,  জ্ঞানবান বা ((জ্যোতিম্মান)) করা হইয়াছে এমন, পরম জ্ঞানে উন্নীত করা হইয়াছে এমন, চরম জ্ঞানে জাগ্রত বা ভূষিত, পরম জ্ঞানে জ্ঞাত বা অভিজ্ঞ। ( পেতবত্থু অর্থকথা)  ভগবা অভিসম্বোধিত। 
অভিসম্বোধেতি ( ক্রি)  পরম জ্ঞানে জাগাইয়া তোলা বা জাগ্রত করা,  জ্ঞানবান বা ((জ্যোতিষ্মান))  করা। ( অতীত)  অভিসস্বোধেসি। ( অ- বিণ)  অভিসস্বোধিত। ⇒ অভিসস্বুজ্ঝতি।
অভিসম্ভব — সম্ভব, সম্ভাবনা।  ( কেবল " দুর" শব্দের সহিত যৌগিকে)  দুরাভিসম্ভব — কষ্টসাধ্য। 
অভিসম্ভবতি ( ক্রি) [ অভি + সম্ভবতি ] ( পর্যন্ত আস' অর্থাৎ পাইতে,  দাড়াইতে অথবা জয় করিতে সমর্থ হওয়া — এই অর্থে)  লাভ করা, সাধন করা, চেষ্টা করিয়া অর্জন করা, পাওয়া, পৌছানো, উপস্থিত হওয়া, প্রসারিত করা বা হওয়া, হাত বাড়াইয়া কোনো কিছু পাওয়া বা স্পর্শ করা, নাগাল পাওয়া, উপার্জন করা, প্রাপ্ত হওয়া, ধারণ করা,  বহন করা। ( অতীত)  অভিসম্ভবি,  অভিসম্ভোসি। ( অ- বিণ)  অভিসম্ভুত।  ( ভবিষ্যতে)  অভিসম্ভোস্সং = সহিস্সামি, অধিবাস্সেসামি। ( ক্রি - বিণ)  অভিসম্ভুত্বা,  অভিসম্ভবিত্বা। ( ব - বিণ)  অভিসম্ভবনীয।  ⇒ অভিসম্ভুণাতি। 
অভিসম্ভুণাতি ( ক্রি)  [ অভি + √সম্ভ + উণা ] নাগাল পাইতে বা পৌছিতে সমর্থ হওয়া, লাভ করিতে সমর্থ হওয়া,  সমর্থ হওয়া, লাভ করা । ⇒ অভিসম্ভবতি। ( অতীত)  অভিসম্ভুণি। 
অভিসম্ভুনন্তো ( বিণ)  সমর্থ হইয়া,  সমর্থ। ⇒ অভিসম্ভুনাতি।
অভিসম্ভুনাতি ( ক্রি)  = অভিসম্ভুণাতি। ⇒ অভিসম্ভুণাতি। 
অভিসম্ভূ ( বিণ)  [ অভি + সম + ভূ ] প্রাপ্ত হইয়াছে বা লাভ হইয়াছে এমন, পাইয়া, লাভ করিয়া। ( দীর্ঘনিকায়)  লোমহংস অভিসম্ভু।  ( পুং)  প্রাপ্তি,  প্রাপ্ত বস্তু, লাভ। 
অভিসম্ভূত ( বিণ)  লব্দ,  প্রাপ্ত, সাধিত, অর্জিত। ⇒ অভিসম্ভবতি। 
অভিসম্বোতি ( ক্রি)  = অভিসম্ভবতি।  ⇒ অভিসম্ভবতি।  ⇒ অভিসম্ভবতি। 
অভিসম্মতি ( ক্রি)  [ অভি + √সম + অ; সংস্কৃত -- অভিশম্যতি)  শেষ হওয়া,  ক্ষান্ত হওয়া, ত্যাগ করা, শেষ করা, বন্ধ করা, রুদ্ধ করা, প্রতিরোধ করা,  বাধা দেওয়া, গতি রুদ্ধ করা,  দমন করা, নিবৃত্ত করা, বন্ধ হওয়া, থামা, অল্পকাল থাকা বা বাস করা, উপশম করা, প্রশমিত করা, শান্ত করা, সান্ত্বনা দেওয়া, ঠান্ডা করা, স্থির করা, চুপ করানো, নিবৃত্ত করা।  ( অতীত)  অভিসম্মি।  ( অ- বিণ)  অভিসম্মন্তো। 
অভিসম্মন্তো ( বিণ)  শান্ত,উপশান্ত, রুদ্ধ, অল্পকাল স্থায়ী, ত্যক্ত, পরিত্যক্ত, সমাপ্ত, দমিত। ( জাতকে " অভিসম্মেন্তো শব্দের পরিবর্তে?  ইহা অনিশ্চিত পাঠ)। ⇒ অভিসম্মতি। 
অভিসম্মেন্তো = অভিসম্মন্তো। ⇒ অভিসম্মন্তো। 
অভিসর — অনুসরণকারী লোকগণ, বড়লোকের অনুচরবর্গ, পরিষদবর্গ, সঙ্গীগণ। 
অভিসরতি ( ক্রি)  [ অভি + সরতি ] ১ ( জলস্রোত)  বেগে প্রবাহিত হওয়া, একত্রে আসা বা একস্থানে মিলিত হওয়া। = সন্নিপততি। ২ অনুসরণ করা,  অনুগমন করা, যাইয়া বা আসিয়া সাক্ষাৎ করা,  দেখিতে যাওয়া বা আসা, দেখাশুনা করা, তদারক করা, পরিদর্শন করা, সাক্ষাৎ করা, সাক্ষাৎ করিবার জন্য যাওয়া, ( অপ্রচলিত)  দগু দেওয়া, সীমাপবর্তী হওয়া, নিকটে আসা, তুল্য হওয়া, নিকটে যাইয়া বলা।  ( অতীত)  অভিসরি। ( অ- বিণ)  অভিসেট। 
অভিসল্লেখিক ( বিণ)  [ অভি + সল্লেখ + ইক ] কঠোর, রুঢ়, উগ্র, অত্যন্ত সাদাসিদে,  বাহুল্য বর্জিত,  কঠিন, দারুণ, নির্দয়, দৃঢ়,  শক্ত, স্থির, অপরিবর্তনীয়, কটু। 
অভিসল্লেখিকা ( স্ত্রী)  কঠোরতা, রুঢ়তা, আত্ননিগ্রহ, তপশ্চর্যা, কঠিনতা,  কর্কশতা, কটুতা।  এই শব্দটি সচরাচর " কথা" সম্বন্ধে ব্যবহৃত হইয়া থাকে। 
অভিসবতি ( ক্রি)  [ অভি + সবতি ] ( কিছুর)  দিকে বা মধ্যে প্রবাহিত হওয়া।  ( জাতক)  নজ্জো গঙ্গং অভিবসতি। 
অভিসসি ( অতীত ক্রিয়া)  ⇒ অভিসংসতি।। 
অভিসেকমঙ্গল ( ক্লী)  অভিষেক মঙ্গল, উৎসর্গ, পূত বা পবিত্র করার কার্য। ⇒ মঙ্গল ( ক্লী) । 
অভিস্সবতি ( ক্রি)  = অভিসবতি। ⇒ অভিসবতি। 
অভিসহতি ( ক্রি)  = অভিসম্ভুণাতি।  ⇒ অভিসম্ভুণাতি।
অভিসহমানো ( বিণ)  = অসম্পাপুণন্তো। ⇒ অসম্পাপুণন্তো। 
অভিসাধেতি ( ক্রি)  [ অভি + সাধেতি ] সম্পন্ন করা, শ্রেণীবদ্ধ করা,  সাজানো,  বন্দোবস্ত করা, নিষ্পত্তি করা, পাওয়া,  লাভ করা, জন্ম দেওয়া, শিক্ষা করা, প্রবর্তিত করা, ঘটানো,  আহার্য প্রস্তুত করা বা ভোজন করা, রোগাক্রান্ত হওয়া, কোনো কিছু সম্পন্ন করানো, ( চলতি)  শাস্তি বা দগু পাওয়া,  জব্দ করা, বুঝিতে পারা, উপস্থিত হওয়া, কোনো অবস্থায় উপস্থিত হওয়া,স্বচেষ্টায় প্রাপ্ত হওয়া,  সাধন করা, চেষ্টা করিয়া অর্জন করা, পৌছানো।  ( অতীত)  অধিসাধেসি। 
অভিসাপ ( পুং)  অভিশাপ, অমঙ্গলেচ্ছা, অমঙ্গলজনক বস্তু, ধর্মযাজকের অভিসম্পাতগ্রস্ত ব্যাক্তি বা বস্তু, অভিসম্পাত, ধর্মযাজকের অভিসম্পাত। ⇒ অভিলাপ। ( অঙ্গুত্তর নিকায়ে অক্কোস রুপেও ইহাকে বর্ণনা করা হইয়াছে। 
অভিসাপাযং — অভিশাপ ( আক্রোশজনিত শব্দবিশেষ) । ⇒ অভিসাপ, অভিলাপ। 
অভিসারিকা ( স্ত্রী)  অভিসারিকা, বেশ্যা, গণিকা, বারবনিতা,  বারাঙ্গনা ।  ⇒ অভিসারিযা। 
অভিসারিযা ( স্ত্রী)  প্রণয়ীর উদ্দেশ্যে গৃহ হইতে সঙ্কেতস্থানে গমনকারিণী নারী। = অভিসারিকা। 
অভিসারেতি ( স্ত্রী)  [ অভি + সারেতি ] সমীপবর্তী হওয়া,  নিকটে আসা, তুল্য হওয়া, নিকটে যাইয়া বলা, উত্ত্যক্ত করা, নির্যাতন করা, নিগৃহীত করা, হয়রান করা,  উৎপীড়ন করা। ( অতীত)  অভিসারেসি।
অভিসিংসতি ( ক্রি)  [ অভি + সিংসতি ] পবিত্র ধর্মসংক্রান্ত বাক্য উচ্চারণ করা অথবা ইচ্ছা প্রকাশ করা,  শ্রদ্ধারুপ ভয়মিশ্রিত বাক্য উচ্চারণ করা অথবা ইচ্ছা প্রকাশ করা। ( তুলনীয়) অভিসংসতি।  ( অতীত)  অভিসিংসিসি,  অভিসিংসীসি ; ( বিমানবত্থু অর্থকথা বর্ণনা)  ইচ্ছি সম্পটিছি। 
অভিসিঞ্চতি ( ক্রি)  [ অভি + √সিচ + ং + অ ] অভিসিঞ্চন করা, সিঞ্চন করা, ( কিছুর উপর)  ছিটাইয়া দেওয়া; ( চরিত্রে)  অভিষেক করা, বিন্দু বিন্দু বারিপাত করা, দেবসেবায় উৎসর্গ করা, পবিত্র করা। ( অঙ্গুত্তর নিকায়) খত্তিযাভিসেকেন অভিসিঞ্চতি। ( অতীত)  অভিসিঞ্চি। ( মিলিন্দ প্রশ্ন)  অমতেন লোকং অভিসিঞ্চি ভগবা। ( পেতবত্থু অর্থকথায় পাঠ)  অভিসিঞ্চি চিমিল্লিক ঞ্চ। ( অ - বিণ)  অভিসিত্ত ।  ( ক্রি - বি)  অভিসিঞ্চিত্বা। ( কর্মবাচ্যে)  অভিসিঞ্চতি। ( করণে)  অভিসিঞ্চেতি,  অভিসেচেতি। 
অভিসিঞ্চেতি ( ক্রি)  অভিসিঞ্চন করা, সিঞ্চন করা, অভিষেক করা। ( অতীত)  অভিসিঞ্চেতি। ( পঞ্চমী)  অভিসেচযস্সূ। ⇒ অভিসিঞ্চতি।
অভিসিত্ত ( বিণ)  ১ অভিষিক্ত, স্নাত, বিধিপূর্বক পবিত্র জলদি দ্বারা স্নাপিত, কৃতাভিষিক্ত, শাস্ত্রমতে যথাবিধি
দীক্ষিত, কোনো পদে যথা — বিধি প্রতিষ্ঠিত, পূত, উৎসর্গ বা পবিত্র করা হইয়াছে এমন। ( অঙ্গুত্তর নিকায়)  খত্তিযো খত্তিযেহি খত্তিযাভিসেকেন অভিসিত্ত। অঙ্গুত্তর নিকায়ে " অবসিত্ত " শব্দের পরিবর্তে " অভিসিত্ত" শব্দের ব্যবহার দৃষ্ট হয়। ⇒ অবসিত্ত। ২ জল বা তৈল ছিটানো দেওয়া হইয়াছে এমন। ( সূত্রনিপাত)  অভিসিত্ত মনসা। ( মিলিন্দ প্রশ্ন)  অমতেন লোক অভিসিত্ত। ⇒ অভিসিঞ্চতি। 
অভিসেক ( পুং)  ( রাজ)  অভিষেক, স্লান, যথাবিধি মন্ত্র উচ্চারণপূর্বক পবিত্র জলাদি দ্বারা স্লাপন,  রাজার প্রথম সিংহাসনে আরোহণকালীন অনুষ্ঠানবিশেষ, রাজ্যাধিকারার্থ অভিষেচন, কর্মে নিয়োগ, স্লাপন, উৎসর্গ, পবিত্র বা পূত করার কার্য, পুণ্যস্লান, সাধারণ স্লান বা গাত্র ধোয়া, প্রতিষ্ঠা।  ( তুলনীয়)  অভিসিত্ত।  ( অঙ্গুত্তর নিকায়) অভিসেক — অনভিসিত্ত। ( ব্যাকরণে)  অভিসেকিক। 
অভিসেকিক ( বিণ)  = অভিসিত্ত। ⇒ অভিসিত্ত,  অভিসেক। 
অভিসেচন ( ক্লী)  অভিষেচন,  অভিষেক, যথাবিধি তীর্থজলাদি দ্বারা স্লান,  স্লাপন, কর্মে নিয়োগ, পুণ্য স্লান, পবিত্র করার কার্য, ধৌতকরণ, উৎসর্গ, সাধারণ স্লান বা গাত্র ধৌতকরণ। = অভিসেক। ( থেরীগাথা)  উদক অভিসেচন। 
অভিসেবনা ( স্ত্রী)  [ অভি + সেবন ] পশ্চাদ্ধাবন, অনুসরণ, সাধনা,  কার্য, ব্যবসায়, প্রসাদ, চরিতার্থতা সাধন, প্রশয়দান, পাপের জন্য দগুের ক্ষমা, অমিতাচার। ( সদ্ধম্মোপাযন)  পাপকম্ম অভিসেবন। 
অভিস্সর ( বিণ)  [ অভি + ইস্সর ] ঈশ্বর, আশ্রয়, দাতা, প্রভু, কর্তা। ⇒ অনভিস্সর। 
অভিহংসতি ( ক্রি)  [ অভি + হংসতি ] ১ ( সকর্মে)  হৃষ্ট করা, প্রফুল্ল করা, উল্লসিত করা, সন্তুষ্ট করা, পরিতৃপ্ত করা, প্রত্যয় জন্মানো,  নিঃসন্দেহ করা, সন্দেহ ভঞ্জন করা, পরিশোধ করা, পূর্ণ করা, মেটানো। ( সংযুক্ত নিকায়)  অভিহট্ঠুং অভিহংসতি। ২ ( অকর্মে)  ( কোনো বিষয়ে)  আনন্দিত হওয়া, সুখভোগ করা, কেবল হাসিতে থাকা, আনন্দ অনুভব করা, উপভোগ করা। ( সংযুক্ত নিকায়)  রুপং মনাপং অভিহংসতি -- মনোজ্ঞ রুপে ( সৌন্দর্যে)  আনন্দ অনুভব করা। ( ব- বিণ)  অভিহিসমান, অভিহংসমান। 
অভিহংসমান ( বিণ)  আনন্দ অনুভব করে বা করিতেছে এমন, হাস্যমান, কেবল হাসিতে থাকে এমন।  ( অঙ্গুত্তর নিকায়)  অভিহংসমান ঝানং — যে ধ্যান অবস্থায় আনন্দ অনুভব করিতে থাকে অথবা কেবল হাসিতে থাকে। ⇒ অভিহংসতি। 
অভিহট ( বিণ)  আনীত, প্রদত্ত, দেওয়া হইয়াছে এমন, পুরস্কৃত,  বিক্রীত, প্রাপ্ত, আকর্ষিত। = পুরেতং গহেত্বা আহতন ভিক্খং ( পুগ্গল পঞ্ঞতি অর্থকথা) । ⇒ অভিহরতি।

অভিহট্ঠুং ( ক্রি - বি)  নিকটে আনা। = অভিহরিতুং। ( বি- ভা)  অভিহট্ঠুং পবারেতি — আনিয়া দেওয়া, আনিয়া দান করা ( মধ্যম নিকায়) । ⇒ অভিহরতি। 
অভিহত ( বিণ)  আক্রান্ত, প্রহৃত, নিগৃহীত, আহত, অগ্রবতী, সমীপবতী, স্পশিত, আঘাতপ্রাপ্ত।  ⇒ অভিহনতি। ( ক্লী)  প্রহার, আঘাত। 
অভিহনতি ( ক্রি)  [ অভি + √হন + অ ] ১ প্রহার করা, আঘাত করা, মারা, নিগৃহীত করা। ২ বলে বশীভূত করা, জয় করা, পরাভূত করা, বধ করা, ধ্বংস করা।  ( অতীত)  অভিহনি। ( অ - বিণ)  অভিহত। ( তুম)  " অভিহন্তুং" শব্দের পরিবর্তে জাতকে " অভিহন্তু"। 
অভিহরতি ( ক্রি)  [ অভি + √হর + অ; তুলনীয় সংস্কৃত -- অভ্যাহরতি, বৈদিক -- আহরতি, আভরতি ] ১ নিকটে আনা, প্রদান করা, দান করা, যাইয়া আনয়ন করা, প্রাপ্ত হওয়া, ( মূল্যে)  বিক্রয় হওয়া, আকর্ষণ করা। ২ অভিশাপ দেওয়া, গালি দেওয়া, ( উপাসনাগার হইতে)  বঞ্চিত করা, অভিশাপ হইয়া দাঁড়ানো, গালাগালি করা, অপযশ করা, অপব্যয় করা, নিয়মভঙ্গ করা, বিকৃত করা। সংস্কৃত — অনুব্যাহরতি, অভিব্যাহরতি। ⇒ আভত। ( অতীত)  অভিহরি। ( অ - বিণ)  অভিহট। ( ক্রি - বি)  অভিহট্ঠুং। ( কর্মবাচ্য)  অভিহরিযতি। ( তুম)  অভিহরিতুং। ( করণে)  অভিহারেতি। 
অভিহরিতুং ( তুম)  নিকটে আনিতে,  আনিতে। = অভিহট্ঠুং। ⇒ অভিহরতি। 
অভিহার ( পুং)  নিকটে আনয়নকরণ, যাইয়া আনয়নকরণ,  প্রদান, উপহার, বলি, পূজা, নৈবেদ্য, প্রস্তাবন দান, প্রকৃতিদত্ত গুণ,  উপঢৌকন। ( জাতক)  আসনাভিহার। 
অভিহারযং  ( ক্রি - বি)  আনিবার কারণ হওয়া, প্রাপ্ত হওয়া, লাভকরণ, অর্জনকরণ। এই শব্দটি টীকাকার " অভিভারযিন " শব্দরুপেও গ্রহণ করিয়াছেন। অভিহারেতি (১) শব্দে উদাহরণ দ্রষ্টব্য। 
অভিহারেতি ( ক্রি)  ১ আনিবার কারণ করা বা হওয়া, প্রাপ্ত হওয়া, লাভ করা, উপার্জন করা, অর্জন করা। ( জাতক)  অভিহারযং ( টীকায়)  অভিভারযিন। ২ নিজে ( কোনো উপায়)  অবলম্বন করা অথবা আশ্রয় গ্রহণ করা, গিয়া বা আসিয়া সাক্ষাৎ করা, দেখিতে যাওয়া বা আসা, দেখাশুনা করা, তদারক করা, পরিদর্শন করা, সাক্ষাৎ করা, সাক্ষাৎ করিবার জন্য যাওয়া, ( অপ্রচলিত)  দগু দেওয়া, লওয়া, লইয়া যাওয়া,ধরা, গ্রহণ করা, বাছিয়া বাছিয়া লওয়া, মনোনীত করা, প্রয়োগ করা, ব্যবহার বুঝিতে পারা, বিবেচনা করা, প্রয়োজন হওয়া, দরকার করা, বিমোহিত করা, বন্দী করা, দখল করা, সম্মত হওয়া, বোধ করা, অধিকারভুক্ত করা, কাড়িয়া লওয়া,  হরণ করা, বাদ দেওয়া, বাহির করা, সরানো,  সহ্য করা, ভক্ষণ করা, পান করা, শরীরে গ্রহণ করা, ( কোনো সংক্রামক রোগ দ্বারা)  আক্রান্ত হওয়া, সম্পাদন করা, ধারণা করা, গ্রাহক হওয়া, অতিক্রম করা, উদ্ধৃত করা, ফলপ্রদ হওয়া, গমন
করা, ঘটা, চলা, প্রস্থান করা, অগ্রসর হওয়া, প্রস্তুত হওয়া, যন্ত্রের সাহায্য চলা, রহিত হওয়া, দেহত্যাগ করা, অন্য অবস্থায় পরিণত হওয়া, কোনো বিশেষ অবস্থার অধীন হওয়া, উপযোগী হওয়া,বিস্তৃত হওয়া, কোনো লক্ষ্যের দিকে অগ্রসর হওয়া, বিক্রয় করা, প্রচলিত হওয়া, অতিবাহিত হওয়া।  ( সূত্রনিপাত)  অভিহারেসি — পাগুবশৈলে আরোহষ করিল; = আরুহি ( সূত্রনিপাত অর্থকথা) ।  ( সূত্রনিপাত)  বনন্তং অভিহারযে = বনং গচ্ছেয্য ( সূত্রনিপাত অর্থকথা) ।  ৩ ( ডাকে)  দেওয়া, রাখা।  ( অতীত)  অভিহারেসি,  অভিহারযিং; ( থেরীগাথা অর্থকথা) । ( ভবিষ্যৎ)  অভিহেস্সতি ; ( জাতক)  কবচং অভিহেস্সতি ( জাতক অর্থকথা বর্ণনায় ভুলবশত গ্রহণ করা হইয়াছে " অভিহনিস্সতি ভিন্দিস্সতি,  সুতরাং প্রত্যক্ষরুপে " অভিভবিস্সতি" গ্রহণ করা যাইতে পারে); ( জাতক) কিং যোব্বানেন চিণ্ণেন যং জরা অভিহেস্সতি?  ( এখানে "অভিববিস্সতি " শব্দের পরিবর্তে অভিহেস্সতি " শব্দ গ্রহণ করা হইয়াছে টীকায় " অভিভুয্যতি" শব্দ গৃহীত হওয়ার ন্যায়। ⇒ অভিহরতি। 
অভিহিংসং = অভিহিংসনা। ⇒ অভিহিংসনা। 
অভিহিংসতি ( ক্রি)  " অভিহিসতি " শব্দের পরিবর্তে অঙ্গুত্তর নিকায়ে পাঠ হইয়াছে " অভিহংসতি"। ⇒ অভিহংসতি।
অভিহিংসনা ( স্ত্রী)  হ্রেষা রব, ঘোড়ার ডাক। " অভিহেসনা " শব্দের পরিবর্তে ব্যবহার শব্দ; সংস্কৃত — হেষা, হেসিতং ; প্রাকৃত — হেসা। বিমানবত্থু অর্থকথার টীকায় " অভিহেসন"।  ⇒ অভিসংসনা। 
অভিহিত ( বিণ)  অভিহিত, কথিত, নির্দিষ্ট, উল্লিখিত। সিংহলী সংযুক্ত নিকায়ে ইহাকে " অভিগীত " শব্দরুপেও পাঠ দেখা যায়। ⇒ অভিধাতি। ( ক্লী)  বাক্য, শব্দ, কথন। 
অভিহিসমান = অভিহংসমান। ⇒ অভিহংসমান। 
অভিহীত = অভিহিত।  ⇒ অভিহিত। 
অভিহেসন ( ক্লী)  = অভিহিংসনা। ⇒ অভিহিংসনা। 
অভিহেস্সতি ( ক্রি)  ⇒ অভিহারেতি। 
অভীত ( বিণ)  [ অ + ভীত ] অভীত, নির্ভয়, ভয়হীন, সাহসী।  ⇒ অভিদ (১)। 
অভীতনাদ ( পুং)  নির্ভীক ধ্বনি, বিজয়োল্লাস ধ্বনি। ⇒ সীহনাদ।
অভীনিন্দ্রিয = অভিনিন্দ্রিয। ⇒ অভিনিন্দ্রিয। 
অভীরু ( বিণ)  [ অ + ভীরু ] অভীরু, নির্ভীক, নির্ভয়, সাহসী, ভয়হীন, অভয়। ( যৌগিকে)  অভীরু অনুত্রাসিন।

কণ্টকাপস্সযিকা ( বিণ, স্ত্রী)  = কণ্টকাপস্সযিক। ⇒ কণ্টকাপস্সযিক, কণ্টকপস্সয। 
কণ্টকি ( স্ত্রী) কাঁটাগাছের বেড়া, কাঁটাযুক্ত, কণ্টকময়। 
কণ্টকিবাট = কণ্টকি। ⇒ কণ্টকি। 
কণ্ঠ ( পুং)  ১ কণ্ঠ, গলা,  টুটি, কণ্ঠধ্বনি, গলদেশ। ( মিলিন্দ প্রশ্ন)  কণ্ঠো আকুরতি — গলাভাঙ্গা শব্দ করা। ( পেতবত্থু অর্থকথা)  অক্খরানি মহতা কণ্ঠেন উচ্চরিতানি। ( প্রেতগণের চিক্কণ এবং পিপাসায় শুষ্ক কণ্ঠের অবস্থা বর্ণনায়ঃ —) কণ্ঠ— ওট্ঠ — তালুনংতস্সিত; সূচিকণ্ঠ; সূচিকট্ঠ; বিসুক্খ কণ্ঠ— ওট্ঠ - জিহবা ( পেতবত্থু অর্থকথা) ।  ২ গ্রীবা, গলা, ঘাড়, গলার লম্বা সঙ্কীর্ণ অংশ, গলার দীর্ঘ অপ্রশস্ত অংশ। = গ্রীবা। ( ধর্মপদ)  কাসাবকণ্ঠ। ( বিমানবত্থু অর্থকথা বর্ণনায়)  গীবূপগ সীসূপগাদি অভরণানি। ⇒ কণ্ঠে, কণ্ঠেবর। 
কণ্ঠক ( ক্লী)  ১ কণ্ঠক, কাঁটা। = কণ্টক (১)। ২ (পুং)  সিদ্ধার্থের ( গৌতমের)  অশ্বের নাম ছিল " কণ্টক " ( জাতকের ইহার বানান " কণ্ঠক ") = কণ্টক (৬)। 
কণ্ঠকূপ — কণ্ঠছিদ্র, গলনালী, গলার গহব্বর বা গর্ত। 
কণ্ঠজ ( বিণ)  কণ্ঠজ, কণ্ঠস্থানীয়, কণ্ঠ হইতে উৎপন্ন, কণ্ঠে উৎপন্ন।

কণ্ঠসুত্তক — কণ্ঠসূত্র, ( ব্রাহ্মণের)  পৈতা, গলায় বন্ধনী সুতা, গলার আলঙ্কারিক বা শোভাবর্ধক সুতা, গলার মালাবন্ধনী সুতা অথবা গলায় মালা পরিহিত সুতা। 
কণ্ঠহত্থ — কণ্ঠের সহিত সংলগ্ন হস্তাংশ অর্থাৎ বাহু, হস্তবাহু, হাতের বাহুদেশ। ⇒ কুণি। 
কণ্ঠে ( অধিকরণ কারকে)  গলদেশের বা গলার বেষ্টনীতে। ⇒ কণ্ঠেবর শব্দে উদাহরণ দ্রষ্টব্য। 
কণ্ঠেবর — গলাবন্ধনী, গলার ভিতর আটকানো দ্রব্য। ( অঙ্গুত্তর নিকায়)  কুণপং কণ্ঠে আসত্তং। ( জাতক)  কুণপং কণ্ঠে বদ্ধং ; কণ্ঠে মালা; কণ্ঠে বন্ধন্তি বড্ঢনং। ( পেতবত্থু অর্থকথায় পাপের শাস্তির জন্য প্রস্তুত দগু বিষয়ে —)  রত্তবণ্ণ- বিরল- মালায বন্ধকণ্ঠ; করবীর - মালা - বদ্ধ ( অন্যত্র পাঠান্তরে " সক্ত")  কণ্ঠেগুণ। 
কগু ( পুং, ক্লী)  ১ কাগু, বৃন্ত, নাল, গাছের গুঁড়ি, গুচ্ছ, ঝাড়, বাণ, অবসর,  খকড়াবিশেষ, সমূহ, জল, প্রশংসা, প্রস্তাব, প্রকারণ, নির্জন স্থান, ব্যাপার, অশ্ব, প্রন্তর, নাড়িসমূহ, ডাঁটার অংশ, দুই গিরার মধ্যবতী বেতের অংশ, সম্পূর্ণ ডাঁটা বা ডাঁটা, তীর, শরযষ্টি। চুল্লধনুগ্গহ জাতকেও " কগু" অর্থে "শর"। ২ গ্রন্হের ভাগ, গ্রন্হের খগু। ৩ ক্ষুদ্র অংশ , টুকরা, অত্যল্প পরিমাণ, পিগু মোট, সমগ্র সমষ্টি। ( ধর্মপদ অর্থকথা)  পূবকগু।

কগুক (পুং)  ১ পদ্মমূলের গোলাকার শালুক। = মুড়াল বটক। ২ = কণ্টক। ⇒ কণ্টক। ( পেতবত্থু অর্থকথা)  কগুকাকগুক।
কগুকবত্তিক ( বিণ)  কাঁটাযুক্ত,  কণ্টক আছে এমন, নিষ্কণ্টক নহে এমন । ⇒ বুত্তিক। 
কগুকাকগুক [ কগুক + অকগুক ] চোরের উপদ্রব মুক্ত, নিরাপদ, নির্বিঘ্ন, নিরুদ্বেগ, অক্ষুণ্ণ, অসাবধান, নিঃসন্দেহ, নিশ্চিত। 
কগুগমন — শর বা তীরের গমন অর্থাৎ ধনুর বাণ গমনাকালে যতদূর শরাচ্ছন্ন করিয়া যায়, তীর ছোঁড়া, তীরের পাল্লা, অল্প দূরবর্তী স্থান। ( তুলনীয়)  কগুু।

কগুবারণ ( বিণ)  ধনুর্ধারী দ্বারা পাহারা দেওয়া হয় এমন, ধনুর্বাণ হস্তে রক্ষাকারী বা পাহারাদার, রক্ষাকর্তা নিযুক্ত। ( ক্লী)  ঢাল, বর্ম, কুলচিহ্ন, আশ্রয়, রক্ষা, রক্ষাকর্তা। 
কগুিকেসযিক ( বিণ)  কণ্টকে শায়িত, কণ্টকতুল্য কেশযুক্ত অথবা কণ্টকতুল্য কেশকম্বলে শায়িত। ⇒ অপস্সাযিক। 
কগুিম — জাতকে এই শব্দটির মুদ্রণ ভুল, কিন্তু ইহার পাঠ — "কগুমস্স অত্থী'তি কগুী তং কগুিনং"। 
কগুিন ( বিণ)  শর সন্নিবেশিত, শর অন্তঃপ্রবিষ্ট,  শরসন্নিবিষ্ট, শরের  অগ্রভাগ ফলাতে শরযষ্টি সন্নিবিষ্ট, ( সল্ল)  বা শর বিদ্ধ বা প্রবিষ্ট। ( পুং)  ধনুর্ধর, তীরন্দাজ।
কগুু ( স্ত্রী)  ১ কগুূ, খাজুরী, গাত্রঘর্ষণ, চুলকানি, সিরসিরানি, খোস, পাঁচড়া, কগুূয়ণ, নিয়ত বিরক্তিকর অভিলাষ, কগুূয়নজনক বিষয়, কগুূয়ন সংক্রান্ত পদার্থ, নখ দ্বারা আঁচড়াইবার ইচ্ছা। ( জাতক)  কগুূং করোতি — চুলকানো, কগুূয়ন করা, কগুূং বিনেতি — চুলকানি উপশম করা, কগুূয়ন প্রশমিত করা, আঁচড়ানো বন্ধ করা। ২ ( চরিত্রে)  সংসার বিষয়ে আসক্তি,  কামপ্রবৃত্তির বশে উত্তেজনা, কামোত্তেজন, কামবশে প্রদাহ,  লোভ বশে ক্রোধ, কামনা বশে প্রদাহ, ক্রোধ বশে উত্তেজনা। ( ধ্যানফলস্বরুপ)  কগুূং সংহতি। ৩ নিক্ষিপ্ত শরের পাল্লা ( শর ছোঁড়ার দূরত্ব পরিমাণ স্বরুপ) । ( থেরগাথা)
সহস্স — কগু সত — ভেত্তু = সতভেদো ( বর্ণানায়)  সহস্সকগুো সহস্স ( সত?) ভূমকো; ( জাতক)  সহস্স কগু ব্বেধো'তি  পাসাদো সতভূমিকো অহোসি; ( পূর্ববর্তী পঙক্তিতে বর্ণানায়)  সহস্সকগু গমনং উচ্চং৷ ( তুলনীয়)  কগু। 
কগুুক ( ক্লী)  চুলকানি, কগুূয়ন, কগুূয়ন করার অনুভূতি বা অভিলাষ, খোস, পাঁচড়া, ক্রোধ, উত্তেজনা, প্রদাহ, সিরসিরানি। = কগুুতি, কগুুযন।
কগুুতি ( স্ত্রী)  কগুূতি, কগুূ,  চুলকানি, সিরসিরানি। = কগুুক, কগুু (১) 
কগুু পটিচ্ছাদি — কগুূয়ন বস্ত্র অর্থাৎ কোন ভিক্ষুর চুলকানি - খোস - পাঁচড়াদি চর্মরোগ হইলে যেইরুপ বস্ত্রখগু দ্বারা উহা বাঁধিয়া রাখার অনুমতি আছে।
কগুুপ্পল [ কগু + উগ্গল ] একপ্রকার ফুটন্ত পদ্মফুল। 
কগুরোগিন  ( বিণ,  পুং)  চুলকানি — খোস — পাঁচড়াদি চর্মরোগগ্রস্ত। 
কগুুবতি = কগুূবতি। ⇒ কগুূবতি। 
কগুুবন ( ক্লী)  ১ চুলকানি, কগুূয়ন, খোস — পাঁচড়াদি চর্মরোগ, চুলকাইবার অনুভূতি। ( তুলনীয়) কগুুবন ( ধাতুঞ্জুষা) । ২ নখাঘাতকরণ,  আঁচড়কাটা, দাগকাটা, ঘর্ষণ, মার্জন।  এই শব্দটি জাতকে খারাপ সঙ্গীতে প্রযুক্ত দৃষ্ট হয়। 
কগুুবাযতি ( ক্র)  = কগুূবাযতি। ⇒ কগুূবাযতি।। 
কগুুস ( ক্লী)  কঠিন চীবর চিহ্নিত করিবার ক্ষুদ্র বস্ত্রখগু, কঠিন চীবরের উপরে নীচে যে ছোট লম্বা এক টুকরা কাপড় লাগানো হয় এবং যাহাকে " গীবায" বলা হয়। 
কগুুসক = কগুুস। ⇒ কগুুস। 
কগুুসকরণ ( ক্লী)  " কগুুস" সেলাইয়ের কার্য, কঠিন চীবরের " গীবায" লাগানো। ⇒ কগুুস। 
কগুূযন ( ক্লী)  কগুূয়ন, চুলকানি,  সুরসুর করা, খোস, পাঁচড়া। = কগুুবন, কগুুক। 
কগুূবতি ( ক্রি)  ১ চুলকানো, কগুূয়ন করা, কোপান্বিত হওয়া,  বিরক্ত হওয়া, উত্তেজিত হওয়া, চুলকানি বা খাজুরি রোগ ভোগ করা। ( জাতক)  কগুুবাযতি। ( ধর্মপদ অর্থকথা)  কগুূবন্তি। ২ তীক্ষ্ম বস্তু দ্বারা চাঁচিয়া ফেলা, নখে আঁচড়ানো,  চাঁচা, ঘর্ষণে ক্ষয় করা, ঘষিয়া ফেলা, ছোলা, ঘর্ষণ করা ,  খুব ঘর্ষণ করা,  পরিস্কার করা ,  ঘসড়ানো, পালিশ করা,  মর্দন করা , লেপন করা, মালিশ করা। ( অতীত)  কগুূবি।
কগুূবন = কগুুবন।  ⇒ কগুুবন। 
কগুূবন্তি ( ক্রি)  = কগুূবতি। ⇒ কগুূবতি।
কগুূবাযতি ( ক্রি)  = কগুূবতি। ⇒ কগুূবতি।
কগুোলিকা ( স্ত্রী)  বেতের ঝুড়ি,  বাঁশের নির্মিত ঝুড়ি, কঞ্চি নির্মিত ঝুড়ি, বাঁশের মঞ্চ, কিছু রাখিবার জন্য বেত বা কঞ্চি নির্মিত মঞ্চ। 
কণ্ণ ( ক্লী)  ১ কোণ, নিভৃত কোণ, দৃষ্টি কোণ। ( বিনয়) চীবর কণ্ণ  — চীবরের প্রান্তভাগ।  চতুকণ্ণ — চতুষ্কোণ, ( বর্ণানায়)
নরকের চারিকোণ; ( পেতবত্থু বর্ণনায়)  চতুকোণ। ২ কর্ণ, কান, শ্রবণেন্দ্রিয়। ( ধর্মপদ অর্থকথা)  দসাকণ্ণ। ( বি - ভা)  কণ্ণং ছিন্দতি — কর্ণচ্ছেদন করা ( শাস্তিস্বরুপ) । ( অধিকরণ কারক)  কণ্ণে — কানের ভিতর অর্থাৎ নীচ স্বরে, কানে কানে কথা বলা, ফিসফিসানি, চাপাকথা, মৃদু বা খস খস শব্দ, গুজব, কানাঘুষা। ৩ চামচের অগ্রভাগ বা আগা। 
কণ্ণক ( বিণ)  কোণ আছে এমন, কান আছে এমন। 
কণ্ণকটক ( বিণ)  শুনিতে বিরক্তিকর, শ্রবণকটু। 
কণ্ণকিত ( বিণ)  ( এই শব্দটি " কণ্হকিত " হইবে; তুলনীয় — পুংসুকিত, মলগ্গকিত; কিত = কত)  ধূলিধূসরিত, ময়লাযুক্ত, মল বা ময়লা লাগিয়াছে এরুপ,  নষ্টকৃত, মরিচাধরা, মরিচা পড়া, মলাবৃত, নিস্তেজ, অশক্ত,  প্রাচীনপন্হী, স্পষ্টভাষী, ভোঁতা, স্থুলবুদ্ধি, কর্দমযুক্ত, অপরিস্কার,  নোংরা,  নীচ, অধম, ছাতাপড়া, দাগী, চিহ্নিত, দাগ পড়া,  দাগ লাগা। 
কণ্ণগৃথ ( পুং, ক্লী)  কর্ণমল, কানের ময়লা, কাণের খৈল। = কণ্ণমল। 
কণ্ণগৃথক = কণ্ণগৃথ। ⇒ কণ্ণগৃথ। 
কণ্ণচালন ( ক্লী)  কানামলা, কর্ণমর্দন, কাননাড়া, কর্ণচালনা।
কণ্ণচুলিকা ( স্ত্রী)  ১ কর্ণমূল, কানের মূল শিকর। ⇒ চুলিকা। ২ = কণ্ণচুল। ⇒ কণ্ণচুল। 
কণ্ণচুল ( ক্লী)  হস্তিকর্ণের মূল,  কানের ছাদ,  কর্ণপটহ, কর্ণপট, কর্ণের মধ্যস্থিত শব্দগ্রাহক সূক্ষ্ম পর্দা। = কণ্ণচুলিকা। ⇒ চুল। 
কণ্ণছিদ্দ = কণ্ণচ্ছিদ্দ। ⇒ কণ্ণচ্ছিদ্দ। 
কণ্ণচ্ছিদ্দ ( ক্লী)  কানের ছিদ্র, কর্ণছিদ্র, কর্ণরন্ধ্র, কর্ণবিবর, অলঙ্কার ধারণার্থ কর্ণে কৃত ছিদ্র, শ্রবণ — সম্বন্ধীয় পথ, শ্রবণপথ, শ্রবণ করিবার কর্ণচ্ছিদ্র, কর্ণপথ, কর্ণকুহর। ( তুলনীয়)  সূচি — ছিদ্র। ⇒ ছিদ্দ। 
কণ্ণচ্ছিন্ন ( বিণ)  কর্ণচ্ছিন্ন, কানাকাটা, যাহার কান ( শাস্তিস্বরুপ)  ছেদন করা হইয়াছে। 
কণ্ণচ্ছেদ — কর্ণচ্ছেদ, কান কাটিয়া বা ছিঁড়িয়া ফেলা। 
কণ্ণজপ্পক ( বিণ, পুং)  যে ব্যাক্তি কানে কানে কথা বলে, যে ব্যাক্তি গোপনভাবে কথা বলে, যে ব্যাক্তি চুপি চুপি অথবা ফিস ফিস করিয়া কথা বলে, যে ব্যাক্তি আস্তে আস্তে কথা বলে,  যে ব্যাক্তি কানাঘুষা করে।  সকণ্ণজপ্পক — কানে কানে কথা বলে অথবা কাণাঘুষা করে এমন। ( তুলনীয়)  উপকণ্ণক জপ্পিন। 
কণ্ণজপ্পন ( ক্লী)  কানে কানে কথা বলা, চুপি চুপি অথবা ফিস ফিস করিয়া কথা কথন, কর্ণসন্ত্র, কানাঘুষা।  ( তুলনীয়)  উপকণ্ণকজপ্পিন। 
কণ্ণজলূকা ( স্ত্রী)  এক প্রকারের ক্ষুদ্র বিছা, কেন্নো প্রভৃতি বহুপদ প্রাণী, শতপদী। 
কণ্ণতেল — কর্ণতৈল, কানে দিবার ঔষধরুপ তৈলযুক্ত মলম,  ঔষধরুপ কানে দিবার তৈল। ( দীর্ঘনিকায় অর্থকথা)
কণ্ণতেলনং।
কণ্ণতেলনং = কণ্ণতেল। ⇒ কণ্ণতেল। 
কণ্ণধার (পুং)  কর্ণধার, নাবিক,  কাগুারী। 
কণ্ণনাস — কর্ণ এবং নাসিকা,  নাক — কান। ( মিলিন্দ প্রশ্ন)  কণ্ণনাসচ্ছিন্ন — যাহার নাক কান ছেদন করা হইয়াছে,  নাক কান ছিন্ন বা কর্তিত।
কণ্ণনাসচ্ছিন্ন ( বিণ) ⇒ কণ্ণনাস। 
কণ্ণপত্ত — কানের নিম্নভাগ,  কর্ণপত্র, কানের লতি। ( থেরীগাথায়)  কণ্ণপন্ত। = কণ্ণপুচ্ছ। 
কণ্ণপন্ত = কণ্ণপত্ত। ⇒ কণ্ণপত্ত। 
কণ্ণপালি = কণ্ণপত্ত। ⇒ কণ্ণপত্ত। 
কণ্ণপিট্ঠী — কানের উপরি অংশ,  কানের উর্ধ্বাংশ। 
কণ্ণপুচ্ছ — কর্ণলেজ,  কানের শিথিলভাবে লম্বমান বস্তু। = কণ্ণপত্ত। 
কণ্ণবিল ( ক্লী)  = কণ্ণচ্ছিদ্দ। ⇒ কণ্ণচ্ছিদ্দ।
কণ্ণভিজপ্পন ( ক্লী)  কর্ণমন্ত্র জপকরণ। 
কণ্ণভূসা ( স্ত্রী)  কর্ণাভরণ, কানের অলঙ্কার,  কর্ণভুষণ। 
কণ্ণভেরি — একপ্রকার ঢোল,  কর্ণপটহ। 
কণ্ণমল ( পুং)  কর্ণমল, কানের ময়লা, কানের খৈল। = কণ্ণগূথ। 
কণ্ণমলহরণী ( স্ত্রী)  কর্ণমল সরাইবার যন্ত্র, কানের ময়লা বা খৈল ফেলিবার যন্ত্র। ⇒ হরণী। 
কণ্ণমালা ( স্ত্রী)  ( মন্দিরের)  এক কোণ হইতে অপর কোণের বা কোণ পর্যন্ত ফলের মালা। 
কণ্ণমুগু ( বিণ)  ( কুকুর- সস্বন্ধে পেতবত্থু অর্থকথায়)  কানকাটা, কর্ণছিন্ন,  কর্ণচ্ছেদিত, যাহার কান ছেদন করা হইয়াছে অথবা কাটিয়া ফেলা হইয়াছে। ( পেতবত্থু অর্থকথা)  ছিন্নকণ্ণ। ⇒ মুগু। 
কণ্ণমুগুক ( বিণ)  স্হুল কোণযুক্ত অর্থাৎ হিমালয়ের উপকথায় বর্ণিত ( সত্ত মহাসরা)  সপ্ত মহাসরোবরের মধ্যে প্রথমটির নাম " কণ্ণমগুক " ( জাতক, বিশুদ্ধিমার্গ) । ⇒ মুগুক, কণ্ণমুগু।
কণ্ণমূল ( ক্লী)  কর্ণমূল, কানের গোড়া, রোগবিশেষ অর্থাৎ কর্ণমূল নামক রোগ, ( সাধারণ কথায়)  কান, ( চরিত্রে ধর্মপদ অর্থকথানুসারে)  ক্ষীণস্বর, নিকটে,  খুব কাছে; মমকণ্ণমূল ( ধর্মপদ অর্থকথা) । 
কণ্ণরোগ ( পুং)  কর্ণরোগ, কানের পীড়া। 
কণ্ণবন্ত ( বিণ)  যাহার কান আছে অর্থাৎ নিপুণ,  দক্ষ, চতুর, বুদ্ধিমান, সতর্ক,  ধূর্ত, উগ্র, তীব্র,  কর্কশ, ব্যগ্র, সুনির্দিষ্ট। = কণ্ণচ্ছিদ্দং পন ন কস্সচি ন'ত্থি ( অর্থকথা) । 
কণ্ণবল্লী ( স্ত্রী)  কানের লতা, কানের লতি, কানের নিম্নভাগ। = কণ্ণপত্ত, কণ্ণপুচ্ছ। 
কণ্ণবিজ্ঝন ( ক্লী)  কর্ণবেধ, ( সংস্কারবিশেষে)  কান বিঁধানো, কান ফোঁড়ানো।  ( ধর্মপদ অর্থকথা)  কণ্ণবিজ্ঝন মঙ্গল — কর্ণবেধ উৎসব। ( তুলনীয়)  মঙ্গল। = কণ্ণবেদ।
কণ্ণবিজ্ঝন মঙ্গল ( ক্লী)  কর্ণবেধের মঙ্গলোৎসব। ⇒ মঙ্গল ( ক্লী) ,  কণ্ণবিজ্ঝন,  বিজ্ঝন।
কণ্ণবিল ( ক্লী)  কর্ণরন্ধ্র, কানের ছিদ্র। = কণ্ণচ্ছিদ্দ। 
কণ্ণবেঠন ( ক্লী)  একরকম কর্ণাভরণ, কানের একপ্রকার অলঙ্কারবিশেষ। 
কণ্ণবেদ ( পুং)  কর্ণবেধ,  কান ফোঁড়ানো,  কান বিঁধানো। = কণ্ণবিজ্ঝন। 
কণ্ণসক্খলি ( পুং)  কণ্ণসক্খলিকা। ⇒ কণ্ণসক্খলিকা।
কণ্ণসক্খলিকা ( স্ত্রী)  কর্ণচ্ছিদ্রের বহির্ভাগ,  শ্রবণ — সস্বন্ধীয় পথ,  কর্ণে প্রবেশ পথ বা দ্বার। ( ধর্মসঙ্গণী অর্থকথা)  কণ্ণসক্খলিকং পহরতি — বাতাস জোরে কানে ধাক্কা দেয় বা আঘাত করে, ( বাতাস যেন)  কান ফাটিয়া ফেলিতেছে; কণ্ণসক্খলিকং ভিন্দতি — ( বিরক্তিকর শব্দে)  কানে তালা লাগিতেছে অথবা কান বন্ধ হইয়া যাইতেছে, কান বিদীর্ণ হইয়া যাইতেছে; = ভিন্দন্তো বিষ পহরতি। ( ধর্মসঙ্গণী)  সঙ্খলিং, সক্খলিং। 
কণ্ণসঙ্খলি ( পুং)  কর্ণাভরণের ক্ষুদ্র শিকল, কানে অলঙ্কার ঝুলানের ছোট শিকল। 
কণ্ণসন্ধোবিক — কর্ণ ধৌতকরণ,  কানের ভিতরে ধুইয়া পরিস্কার করণ। 
কণ্ণসুক্ক ⇒ কণ্হ (২) 
কণ্ণসুখ ( বিণ)  কর্ণসুখ,  কানসুখ, শ্রবণে আনন্দদায়ক বা সুখজনক,  যাহা শুনিতে ভালো লাগে, স্বীকার্য, মনোজ্ঞ, রাজি, সম্মত,  উপযোগী। ⇒ সুখ ( বিণ) ।  ( ক্লী)  সুখকর বা আনন্দজনক কথা ,  মনোজ্ঞ কথা, আমোদজনক আলাপ। ( বিপ)  কণ্ণসুল। 
কণ্ণসুত্ত — কর্ণাভরণের সুতা, কানের অলঙ্কারস্বরুপ সুতা অথবা অলঙ্কার ঝুলাইয়া দিবার সুতা। 
কণ্ণসুত্তক — এক কোণ হইতে অন্য কোণে আবদ্ধ সুতা বা রজ্জু, এক কোণ হইতে অপর কোণ পর্যন্ত আবদ্ধ ফিতা বা বস্ত্র,  এক কোণ হইতে অপর কোণ পর্যন্ত ফিতা বা বস্ত্র টানা। 
কণ্ণসূল ( ক্লী)  ১ কর্ণশূল,  কর্ণবেদনা,  কানের ব্যাথা,  কানের যাতনা। ২ কর্কশ বাক্য,  অমনোজ্ঞ কথা, যাহা শুনিতে ইচ্ছা করে না।  ( বিপ)  কণ্ণসুখ। 
কণ্ণসোত ( পুং,  ক্লী)  কর্ণ, কান, শ্রবণেন্দ্রিয়, শ্রবণ — সম্বন্ধীয় পথ। ( দীর্ঘনিকায়)  কণ্ণ — নাসিক সোতানি, যেমন - উভো সোতানি অর্থাৎ হেট্ঠাউপরিমা সোতানি। ⇒ সোত ( ২খ)  
কণ্ণযত = কণ্ণপত্ত। ⇒ কণ্ণপত্ত। 
কণ্ণাযত মুত্ত — মুক্তাখচিত কর্ণাভরণ, কানে লাগানো মুক্তা। 
কণ্ণালঙ্কার — কর্ণাভরণ,  কানের অলঙ্কার। 
কণ্ণিক ( বিণ)  = কণ্ণক। ⇒ কণ্ণক। কাড়কণ্ণিক। ( পুং)  কণিক, কর্ণধার,  মাঝি, নাবিক। 
কণ্ণিকমগুল = গহকূট। ⇒ গহকূট,  কণ্ণিকা (৩) ।

কণ্ণিকা ( স্ত্রী)  ১ কণিকা,  কর্ণভূষণ,  কানতড়কা, কর্ণাভরণ৷।  ( দীর্ঘনিকায়)  কণ্ণিকা লক্খণ। ২ পদ্মোর ফলত্বক, পদ্মের মধ্যস্থ বীজকোষ, বোঁটা, লেখনী, কুট্টিনী, হস্তের মধ্যাঙ্গুলি, অগ্নিমস্থন বৃক্ষ,  অজশৃঙ্খী বৃক্ষ, অপ্সরাবিশেষ, গোলাপ ফুল, যোনিরোগবিশেষ। ( বিশুদ্ধিমার্গ)  পদুমকণ্ণিকা। ৩ মন্দির অথবা প্রাসাদের উপরিস্থ সিঁড়ির কোণ, গৃহের উপরিভাগ বা ঘরের তুলি। ( ধর্মপদ অর্থকথা)  কুটাগার কণ্ণিকা। = গহকূট। ৪ শস্যাদির আঁটি, গুচ্ছ, মন্দিরের চূড়া আকারে গঠিত গুচ্ছ বা আঁটি। ( যৌগিকে)  কণ্ণিক কণ্ণিকা। ৫ = গিরিকণ্ণিকা। ⇒ গিরিকণ্ণিকা। 
কণ্ণিকাবদ্ধ ( বিণ)  আঁটিবদ্ধ, গুচ্ছে আবদ্ধ; ( চরিত্রে)  চিন্তার বিষয়পুঞ্জ, ভাবনার বিষয়পুঞ্জ ( ধর্মপদ অর্থকথা) । 
কণ্ণিকামগুল ( ক্লী)  ঘরের ছাদের বা চালের অংশ, ঘরের উপরিস্থ চালের বা ছাদের যেখানে বরগাগুলি আসিয়া  পরস্পরের সহিত মিলিত হয়, ঘরের চালের কোণালি। 
কণ্ণিকার ( পুং)  ১ কর্ণিকার, গনিয়ারি গাছ, ছোট সোঁদার, পদ্মের পাপড়ির অগ্রভাগ। ( ক্লী)  স্বর্ণবর্ণ পুষ্পবিশেষ। = কণিকার। 
কণ্ণিকারুক্খ ( পুং)  ঘরের মূল পাইরের খুঁটি, গাছ বা কাঠের গুঁড়ি যাহা ঘরের চাল বা চূড়া নির্মাণ করিতে ব্যবহার করা হয়। 
কণ্ণিকালক্খণ ( ক্লী)  ঘরের তুলি বা চূড়ার লক্ষণ দেখিয়া অথবা কানে পরিহিত অলঙ্কারের লক্ষণ দেখিয়া ভবিষ্যদ্বাণী করার বিদ্যা। = পিলন্ধন কণ্ণিকা পি গেহ কণ্ণিকা পি বসেন ( দীর্ঘনিকায় অর্থকথা) । 
কণ্ণে ⇒ কণ্ণ (২) । 
কণ্হ ( বিণ)  ১ কালো বর্ণবিশিষ্ট, কৃষ্ণবর্ণ, কৃষ্ণবর্ণের,  কালো রঙের,  কৃষ্ণবর্ণবিশিষ্ট, মন্দ, অসৎ, অমঙ্গল বা দুর্ভাগ্যসূচক, অনিষ্টকর। ( বিপ)  সুক্ক। ⇒ পঞ্চবণ্ণ। ২ আঁধার, অন্ধকার।  ( বিপ)  আলোক। ( দৃষ্টিজ্ঞানে)  কণ্ণসুক্ক — আঁধার, অন্ধকার। ( বিপ)  আলোক। ( দৃষ্টিজ্ঞানে)  কণ্ণসুক্ক — আঁধার এবং আলো, ( বর্ণ ভিন্নতায়)  শ্বেত — কৃষ্ণ। ( কালো এবং বিষধর অর্থেঃ—)  কণ্হসপ্প = কালসপ্প — বিষধর সর্প ( জাতক অর্থকথা) । কণ্হ - সীসা — কালো বর্ণের মস্তকযুক্ত, যাহার মাথার চুল কালো বর্ণের। ( জাতক প্রাচুর্য অর্থেঃ—)  কালকেস — কালো বর্ণের চুল; সুকণ্হসীস — গাঢ় কালো বর্ণের চুল; সুকণ্হ - কণ্হ- সীস- অতি গাঢ় কালো বর্ণের চুল; ( তুলনীয়)  সুসুকাল। ( সংযুক্ত নিকায়ে অগ্নির কালো শিখা- সম্বন্ধে)  কণ্হ বত্তনিন। ( বিনয়ে কালো বর্ণের উর্বর মাটি- সম্বন্ধে)  কণ্হ- উত্তর। ( অন্ধকার বা দুঃখ - সম্বন্ধে "কণ্হ" অর্থেঃ —) নিঃস্ব, দুঃস্থ, ঘৃণ্য, শোচনীয়, হতভাগ্য, অভাগা, সমাজে পতিত। ( দীর্ঘনিকায়)  ব্রহ্মণো ব সুক্কো বণ্ণো,  কণ্হো অঞ্ঞো বণ্ণো। ( অন্ধকার বা অসৎ কার্য বা গুণ- সম্বন্ধে)  কণ্হধম্ম ( অঙ্গুত্তর নিকায়) ।  ( জাতক)  কণ্হপটিপদা। ( অঙ্গুত্তর নিকায়)  কণ্হমগ্গ। (পুং) ১ একটি কুকুরের নাম। ⇒ সুনখ। ২ কালো, কালো বর্ণ, কালো রং,  বিষধর সর্প।
কণ্হকম্ম ( ক্লী)  পাপকর্ম, অসৎকার্য, নিন্দনীয় কার্য,  হীনকার্য, নীচকর্ম। ( বিপ)  সুক্ককম্ম। ⇒ কণ্হবিপাক। 
কণ্হকিত ( বিণ)  = কণ্ণকিত। ⇒ কণ্ণকিত। 
কণ্হজটি ( পুং)  কালো বর্ণের মসৃণ জটাধারী ঋষি। ( তুলনীয়)  সুকণ্হ জটিল। 
কণ্হজাত ( বিণ)  পাপস্থানে বা দুঃখময় স্থানে উৎপন্ন, নরকাদি অপায়ে উৎপন্ন। 
কণ্হঞ্জন [ কণ্হ + অঞ্জন ] কালো বর্ণের অঞ্জন, মসৃণ চাকচিক্য বা উজ্জলতা। ( বর্ণনায়)  সুখুমকণ্হ, লোমাচিতত্তা ( জাতক)  
কণ্হতি কণ্হ ( বিণ)  [ কণ্হ + অতিকণ্হ ] ঘোর অন্ধকার, গাঢ় কৃষ্ণবর্ণের,  অতিশয় পাপী, অত্যন্ত অসৎ প্রকৃতির। 
কণ্হধম্ম ( পুং)  পাপ, অসৎ বা অধর্ম কার্য, অসৎ স্বভাব, দুষ্টামি, নষ্টামি। ⇒ কণ্হ। 
কণ্হপক্খ ( পুং)  ১ কৃষ্ণপক্ষ, কৃষ্ণপক্ষীয় প্রতিপদ তিথি হইতে অমাবস্যা পর্যন্ত ১৫ দিন। ( বিপ)  সুক্কপক্খ,  জুন্হপক্খ। ⇒ পক্খ ( ১গ)। ২ সর্বপ্রকার অন্ধকার শক্তি এবং উহাদের চক্র - সম্বন্ধীয় যে কোন বস্তু বা বিষয় অর্থাৎ মার বা পাপরাজা ( সূত্রনিপাত)  ( = নামুচি) ,  দৈত্য,  পিশাচ, ভূত, প্রেত, উপদেবতা ( দীর্ঘনিকায়ে "কণ্হাযন" বা কৃষ্ণজাতীদের পূর্বপুরুষ সম্পর্কে বর্ণিত) । ( তুলনীয়)  কালবণ্ণ ( ধর্মপদ)। 
কণ্হপটিপদা — অসৎ মার্গ, অসৎ পথ, অসদুপায়, অসৎ নীতি, দুর্নীতি, হীন নীতি, অসৎ নিয়ম, অমার্গ, কুমার্গ, বিপথ, বিপরীত পথ, প্রতিকূল পথ বা উপায়। ( তুলনীয়)  কণ্হমগ্গ। 
কণ্হভাবকর ( বিণ)  তমপরায়ণ কার্যকারী, পাপকর্মরুপ কার্যকারী, পাপচিন্তাকারী, পুনর্বার হীনকূলে জন্মগ্রহণ করিবার কারণজনক, তমপরায়ণ ভাবাপন্ন। ( জাতক)  কণ্হ পাপকম্মানি। 
কণ্হভিজাতি [ কণ্হ + অভিজাতি ] গোসালের মতে বা ধর্মানুসারে মানুষের মধ্যে স্বতন্ত্র জাতি অথবা বিশেষভাবে উচ্চতর শ্রেণীর অন্তর্গত শ্রেণী অথবা স্বতন্ত্র প্রকার শ্রেণী ( দীর্ঘনিকায় অর্থকথা)। = নীলভিজাতি। 
কণ্হভিজাতিক ( বিণ)  কৃষ্ণজাতীয়, নীচজাতীয়, ( দীর্ঘনিকায়); অসৎ বা হীন প্রকৃতির ( জাতকে "কাড়ক সভাব " বর্ণনায়)। ⇒ অভিজাতিক। 
কণ্হমগ্গ ( পুং)  অন্ধকারময় পথ, অমার্গ,  কুমার্গ, বিপথ, কুপথ, বিপরীত পথ বা উপায়। = কণ্হ পটিপদা। 
কণ্হবত্তনিন ( বিণ)  কালো শিখা উত্তোলনকারী। তুলনীয় বৈদিক — কৃষ্ণবর্তনিং অগ্নিং ( রঘুবংশ) । ⇒ কণ্হ। 
কণ্হবত্তনী ( পুং)  অগ্নি, আগুন। ⇒ অগ্গি। ( বিণ)  কালো চিহ্ন বা লেজুর রাখিয়া যায় এমন, কালো পদচিহ্ন রাখিয়া যায় এমন ( জাতক বর্ণনায় অগ্নি - সম্বন্ধীয়) । 
কণ্হবিপাক ( ক্লী)  কুফল, হীনফল, পাপের ফল, পাপের শাস্তি কুকর্মের ফল। চারিপ্রকার কার্য এবং চারিপ্রকার
ফল, যথা — ১ ( কার্য সম্বন্ধে)  কণ্হকম্ম ( পাপকর্ম), সুক্ককম্ম ( পূণ্যকর্ম), কণ্হ- সুক্ককম্ম ( পাপ ও পূণ্য কর্ম), অকণ্হ - অসুক্ককম্ম ( পাপহীন ও পুণ্যহীন কর্ম) ; ২ ( ফল সম্বন্ধে)  কণ্হবিপাক ( পাপের ফল) ,  সুক্কবিপাক ( পুণ্যের ফল),  কণ্হ - সুক্ক - বিপাক ( পাপ ও পুণ্যের ফল),  অকণ্হ- অসুক্কবিপাক ( পাপহীন ও পুণ্যহীন কর্মের ফল) ।  ( বিপ) সুক্ক বিপাক। 
কণ্হ সপ্প (পুং)  কৃষ্ণবর্ণের সর্প বা কালন্তর সাপ, বিষাক্ত কালসর্প, গোখরা সাপ,  একজাতীয় বিষধর সর্প। = কালসপ্প। ⇒ সপ্প, কণ্হ (২) । 
কণ্হসীসা ⇒ কণ্হ ( ২) । 
কণ্হ - সুক্ক ( বিণ)  ( কার্য বা জাতি — সম্পর্কে)  হীন এবং উচ্চ, আঁধার এবং আলোকপূর্ণ, কৃষ্ণবর্ণের এবং শ্বেতবর্ণের, কৃষ্ণ এবং শুভ্র, পাপ এবং পুণ্যের,  কালো এবং শ্বেতবর্ণ মিশ্রিত। ⇒ কণ্হ ( ২) ,  কণ্হবিপাক। 
কণ্হ - সুক্ক বিপাক ( ক্লী)  পাপ এবং পুণ্যের ফল, সদাসদ কার্যের ফল। ( বিপ)  অকণ্হ - অসুক্ক বিপাক। ⇒  কণ্হ বিপাক। 
কণ্হস্সপক্খ ( বিণ)  কৃষ্ণপক্ষীয়, কৃষ্ণ বা দুষ্টের পক্ষীয়। = মারপক্খ। ⇒ পক্খ [ ১খ (বিণ) ]। 
কণ্হা ( স্ত্রী)  কালো,  কৃষ্ণবর্ণ, বিষধর সর্প। = কণ্হ ( পুং) । ( বিণ)  কালো রঙের, কৃষ্ণবর্ণের, অন্ধকারময়, অন্ধকার। = কণ্হ ( বিণ) । 
কণ্হ- সুক্ক ( বিণ)  অসৎ এবং সৎ, দুষ্ট এবং ভালো, কৃষ্ণ এবং শুভ্র, কালো এবং সাদা,  অন্ধকার এবং উজ্জল। = কণ্হ- সুক্ক। ⇒ সুক্ক ( বিণ) । 
কণ্হাভিজাতিক  ( বিণ)  নীচবংশীয়, হীনজাতিসম্ভূত, অকৌলিক, অকুলীন। ( সমার্থবাচক)  কাড়ক - সভাব। 
কণ্হুত্তর ( বিণ)  [ কণ্হ + উত্তর ] উপরিভাগে বা বহির্ভাগে কালো বর্ণের,  পৃষ্ঠদেশ বা গোত্র কালো রঙের, তলদেশ বা বাহ্যাকার কালো বর্ণের। ⇒ কণ্হ। 
কত ( বিণ)  ১ কৃত, পরিপূরিত, সমাপিত, তৈরিকৃত,  নির্মিত, করিয়াছে এরুপ, প্রস্তুত করিয়াছে এমন, শেষ করিয়াছে এমন, পরিপূর্ণ করিয়াছে এমন। ⇒ করোতি। কতপাপং -- কৃত পাপ। কত কুসলং -- কৃতপুণ্য। কতকম্মং — কৃতকর্ম। কতনগর -- নির্মিত নগর বা শহর। ( ধর্মপদ)  পাপং মে কতং -- আমার দ্বারা পাপ করা হইয়াছে ; তষ্ণ কম্মং কতং; অট্ঠীনং নগরং কতং -- অস্থি দ্বারা নগর নির্মাণ করা  হইয়াছে; যস্স পাপং কতং কম্মং ( কর্মবাচ্য) ।  ( দীর্ঘনিকায়, মধ্যেম নিকায়, সূত্রনিপাত)  কতং করণীযং — যাহা করিবার করা হইয়াছে ( তুলনীয়)  অঙ্গুত্তর নিকায়ে " অহরত " শব্দে) ।  ( বিনয়)  কতং মযা কল্যাণং অকতং মযা পাপং। ( পেতবত্থু)  অমহাকং কতা পূজা — আমাদের দ্বারা পূজা করা হইয়াছে। ( বাক্য)  নহাপকেহি কত পরিকম্মতা - নাপিত কর্তৃক প্রস্তুতি সমাপ্ত করা হইয়াছে  ( জাতক) ; তেন কত পরিচয় — তাহার সহিত প্রত্যক্ষ
পরিচয় করা হইয়াছে ( বিমানবত্থু অর্থকথা) ; তত্থ কত পরিচযতা — তথায় প্রত্যক্ষ পরিচয় হইয়াছে ( বিমানবত্থু অর্থকথা) ; ( দীর্ঘনিকায়)  তেসং কতপুব্ব — তৎসমস্ত পূর্বে করা হইয়াছে; ( জাতক)  কেন কত — কে করিয়াছে অথবা কাহার দ্বারা করা হইয়াছে? ( থেরগাথা)  কতত্তম — মাতাল বা প্রমত্ত করা হইয়াছে এমন; ( জাতক)  চিরকত সংসগ্গ - দীর্ঘদিনের সংসর্গযুক্ত, দীর্ঘদিন একত্রে বাস করা হইয়াছে এমন। ২ তৈরি, প্রস্তুতকৃত, কর্ষিত, শিক্ষিত, নিপুণ। ( সু, দু সংযোগে)  সুকত, দুক্কত। ( সময় জ্ঞাপনেঃ — বিশেষত আহারের সময় জ্ঞাপনে)  কত- অন্ন - কিচ্চ — ভোজন করিয়া, আহার কার্য শেষ করিয়া। ( জাতক)  কত - ভত্ত - কিচ্চ-- আহারের পর,  খাওয়ার পর। ( দীর্ঘনিকায় অর্থকথা)  কত - পুরেভত্ত - কিচ্চ - প্রাতে পূর্বকৃত্য শেষ করিয়া, প্রাতঃকৃৃত্য সমাপন করিয়া। ( অঙ্গুত্তর নিকায়)  কতপাতরাস — প্রাতরাশ আহার করিয়া, প্রাতঃকালীন ভোজন করিয়া; প্রাতরাশ আহার করিয়া, প্রাতঃকালীন ভোজন করিয়া; অকত পাতরাস — প্রাতঃকালীন ভোজন না করিয়া। ( জাতক)  কত পাতরাস ভত্ত; কত অনুমোদন করিয়া। ( পেতবত্থু অর্থকথা)  কত - ভত্ত - অনুমোদন — ভুক্তানুমোদন বা আহার - সম্বন্ধীয় দানের উপকারিতা ও সন্তুষ্টি বর্ণনা করিয়া। কত ওকাস। 
কতং ( ক্লী)  যাহা করা হইয়াছে তাহা, কৃতকর্ম, যে কাজ করা হইয়াছে। কতস্স অপ্পটিকারক — পরস্পরের মধ্যে আদান প্রদান না করার কার্য, বিনিময় না করার কার্য, পরিবর্তন না করার কার্য। কতং অনুকরোতি — পূর্বকার্যের অনুকরণ করা, ডে কাজ করা হইয়াছে উহার অনুকরণ করা, কৃতকার্যের অনুরুপ কার্য করা। 
কত অন্নকিচ্চ = কত ভত্তকিচ্চ। ⇒ কত ভত্তকিচ্চ, কত ( ২ )। 
কতক ( ক্লী)  ( স্লানের পূর্বে বা পরে ব্যবহৃত)  সম্মার্জনী,  ঘর্ষণকারী। 
কতকম্ম ( বিণ)  অভ্যাসকৃত, অভ্যস্ত, নিপুণ, দক্ষ, কাজ করিয়াছে এমন, কার্য বা নিয়ম করিয়াছে এমন, অভিজ্ঞতা লাভ করিয়াছে এমন। কতকম্ম চোর — যেই চোর বা ব্যাক্তি চুরি করিয়াছে ; অকতকম্ম চোর — যেই চোর চুরি কার্য করিতে পারে নাই ( বিশুদ্ধিমার্গ) । , ( ক্লী)  কর্মকৌশল, কৃতকর্ম, পূর্ব পূর্বজন্মে সম্পাদিত কার্য,  যেই কাজ করা হইয়াছে, করা হইয়াছে এরুপ কর্মের বা কার্যের ফল। ( জাতক)  কতকম্ম চোর -- এইমাত্র চুরি করিয়া আসিয়াছে এমন চোর। ( ধর্মপদ অর্থকথা)  কতকম্মা চোরা অকতকম্মা চোরা — চুরি করিয়াছে চোরগণ এবং চুরি করে নাই চোর গণ। চোরেহি কতকম্ম — চোরগণ কর্তৃক কৃতকর্ম। 
কত করণীয — যাহা করিবার তাহা করিয়া, করণীয় কর্তব্য সম্পাদন করিয়া, পূর্ণজ্ঞান বা অর্হত্ব প্রাপ্তির  অবস্থা। অরহং খীণাসবো বুসিত ওহিতভারো ( মধ্যম নিকায়) । 
কতকল্যাণ ( ক্লী)  যাহা মঙ্গলজনক করা হইয়াছে, ধার্মিক
চরিত্র, সাধুচরিত্র। ( বি - ভা)  কত কল্যাণ কতকুসল ইত্যাদি। = কতুপকার। ⇒ কল্যাণ। ( বিণ)  উপকারী, হিত্যাকারী, কল্যাণকারী, (( মঙ্গলাকাঙ্কী)) । 
কতকাল ( বিণ)  সময় গত করিয়াছে এমন, ( প্রেতগণের) সময় অতীত করা হইয়াছে এরুপ, মৃত, মরিয়া গিয়াছে এমন। পেতে কতকালে — প্রেতগণ গত হইলে বা মরিয়া গেলে। ( তুলনীয়)  কাল। 
কতকিচ্চ( ক্লী)  ১ কৃতকার্য, সম্পাদিত কার্য। ২ সাধনা - সম্বন্ধীয় যাহা করিবার আছে তাহা সম্পাদন করিয়াছেন এমন মহাপুরুষ অর্থাৎ অর্হৎ। ( বিণ)  কৃতকৃত্য, কৃত সম্পাদিত, যাহা কর্তব্য তাহা সম্পাদিত, কর্তব্য কর্ম সম্পাদিত, পূর্ণতাপ্রাপ্ত। ( সাধারণত যৌগিকে)  কতকিচ্চ অনাসব — কৃতকৃত্য অনাসব অর্থাৎ আসক্তি - হীন অর্হৎ। ( নেত্তি)  কতকিচ্চানি হি অরহতো ইন্দ্রযানি। 
কতকিচ্চতা ( স্ত্রী)  অর্হত্বের পূর্ণতা, অর্হত্ব, কৃতকার্যতা। 
কতকিব্বিস ( বিণ)  অসৎ, অসৎকর্মী, দুশ্চরিত্রের।  ( পুং)  দোষীব্যক্তি, অপরাধী,  পাপীলোক,  প্রেতলোকে যন্ত্রণাপ্রাপ্ত সত্ত্বগণ। 
কতকুসল ( বিণ)  পুণ্যবান, সৌভাগ্যবান। ⇒ কতকল্যাণ। 
কতগ্গহ ( বিণ, পুং)  [ কত + গহ ] কৃতলোভী, কৃতার্জনকারী, সুকৃতকর্মের সুফল অর্জনকারী বা লোভা। = কটগ্গহ। 
কতচিত্ত কম্ম ( বিণ)  সজ্জিত, ভূষিত, চিত্রবিচিত্র,  চিত্রকর্ম করা হইয়াছে এরুপ। 
কতচীবর — চীবর তৈরি করিয়া, চীবরের শেষ কার্য হইবার পর। 
কতজাতি হিঙ্গুলিক ( বিণ)  সমাপ্ত, পবিত্র সিন্দুর দ্বারা সজ্জিত বা চিত্রবিচিত্র। 
কতজানিয = কতরুপ। ⇒ কতরুপ। 
কতঞাণ ( ক্লী)  কৃতজ্ঞান, করা হইয়াছে বলিয়া জানা। ⇒ অরিয সচ্চ। 
কতঞ্জলি ( বিণ)  [ কত + অঞ্জলি ] ( নমস্কার - সম্বন্ধীয়)  কৃতাঞ্জলি, বদ্ধাঞ্জলি,  নমস্কার করিবার জন্য দুই হাত উঠাইয়া জোড় করিয়াছে এমন। 
কতঞ্জলিন = কতঞ্জলি। ⇒ কতঞ্জলি। 
কতঞ্জলী = কতঞ্জলি। = কতঞ্জলি। 
কতঞ্ঞু ( বিণ)  কৃতজ্ঞ,  কেহ উপকার করিলে যে ব্যাক্তি তাহা চিরকাল মনে রাখিয়া উপকারী ব্যাক্তির অনিষ্ট চিন্তা না করিয়া প্রত্যুপকারের চেষ্টা করে, উপকারকের সহিত সদ্ব্যবহারকারী, উপকারেচ্ছু, দয়ালু, ভদ্র। ( যৌগিকে)  কতঞ্ঞু কতবেদিন -- উপকারের জন্য অথবা অনুগ্রহ প্রদর্শনের জন্য কৃতজ্ঞ এবং মনোযোগী ( সংযুক্ত নিকায়) । 
কতঞ্ঞুতা ( স্ত্রী)  কৃতজ্ঞতা, উপকার স্মরণ বা স্বীকারকরণ, সাধ্যনুসারে উপকারীর হিতচিন্তা বা অনিষ্ট নিবারণের চেষ্ট,  উপকারকের সহিত সদ্ব্যবহার করণ। ( খুদ্দকপাঠ অর্থকথা)  কতস্স জাননতা।  ( খুদ্দকপাঠ)  কতঞ্ঞুনা কতঞ্ঞুতা। ( যৌগিকে)  কতঞ্ঞুতা
কতবেদিতা ( সংযুক্ত নিকায়) ।  ( জাতক)  কতঞ্ঞূ কতবেদিতা। 
কতঞ্ঞূ ( বিণ)  = কতঞ্ঞু। কতঞ্ঞু। 
কতত্ত ( ক্লী)  কর্মজত্ব, কৃতকর্ম, যাহা করিয়াছে তাহার ফল, কর্মফল, যাহা করিয়াছে,  কৃতকার্য। ( অপদান কারকে)  কতত্তা। ( বিণ)  আত্ন অধিকারভুক্ত,  শিক্ষাপ্রাপ্ত, প্রশান্ত ,  স্থির, ধীর, শান্ত, বিনীত,  দমিত।  ( ক্রি - বিণ)  জন্য, হেতু,  যে কোন হেতুতে বা কারণে। 
কতত্তা ⇒ কতত্ত। 
কতত্থ ( বিণ)  [ কত + অত্থ ] অর্থ উপার্জিত, ধনলব্ধ। ( ক্লী)  উপার্জিত অর্থ, লব্ধধন, প্রাপ্ত টাকা পয়সাদি সম্পত্তি। 
কতথব্ধ ( বিণ)  স্তব্ধকৃত, দৃঢ়ীকৃত, নিষ্ঠুর, নির্দয়, সহানুভূতিহীন। ⇒ অকত কল্যাণ। 
কত দগুকম্ম ( বিণ)  দগুকর্ম করা হইয়াছে এমন, দগুকর্ম দ্বারা পীড়িত বা শাস্তি প্রাপ্ত। = দগুাযিত। 
কতন ( ক্লী)  মন্দকার্য,  অসাদচরণ, অনিষ্টসাধন,  অন্যায়করণ বা অহিতকরণ, ক্ষতিকরণ। ( তুলনীয়)  কটন। ( জাতক)  যম মে অক্খাসি কতনং কতং। 
কত নামধেয ( বিণ)  নামের,  নামপ্রাপ্ত, নামের ডাকা হইয়াছে এমন, নামকৃত, নাম আছে এমন, নাম প্রদত্ত, নাম দেওয়া হইয়াছে এমন, নামধেয়। 
কতনিক্খিমন ( বিণ)  অভিনিস্ক্রান্ত,  নিস্ক্রমণ বা অভিনিস্ক্রমণ করিয়াছে এমন,  সংসারধর্ম ত্যাগ করিয়া সন্ন্যাসব্রত গ্রহণ করিয়াছে এমন। = নিক্কামিন। ⇒ অভিনিস্ক্রমন। 
কতনিস্সম ( বিণ)  অপরিশ্রাস্ত, সাহসী, অবিশ্রাস্ত, সাহসিক, নির্ভীক,  বীরোচিত। 
কতপচ্চুগ্গমন ( বিণ)  সহসা সাক্ষাৎ করিতে গিয়াছে এরুপ, প্রত্যুদগমন করিয়াছে এমন, অভ্যর্থনার জন্য অগ্রে গমন করিয়াছে এমন, অগ্রে গমনপূর্বক অভ্যর্থনা করিয়াছে এমন। 
কত পটিসস্থার ( বিণ)  অনুগ্রহের সহিত প্রাপ্ত, দয়াবশত প্রাপ্ত। 
কতপণিধান ( বিণ)  প্রণিধান করিয়া, মনোনিবেশের সহিত, মনের একাগ্রতার সহিত, ধ্যান করিয়া, প্রযত্ন করিয়া, সমাধি দ্বারা দৃষ্টি করিয়া, যোগ বা সমাধি করিয়া, শ্রদ্ধার সহিত কর্মের ফল ত্যাগ করিয়া, ( বুদ্ধত্ব লাভের জন্য)  স্থির সময় হইতে প্রচেষ্টা করিয়া। 
কত পরপ্পবাদ ( বিণ)  অপরের সহিত কলহে অভ্যস্ত, বিতর্কে অভ্যস্ত,  বিবাদপ্রিয়। 
কত পরিকম্মতা ( স্ত্রী)  প্রস্তুতি, যাহা সমাপ্ত করা হইয়াছে। ⇒ কত।
কতপরিগ্গহ ( বিন)  দার পরিগ্রহ করিয়া,  বিবাহ করিয়া, বিবাহ করিয়াছে এমন,  বিবাহিত, স্ত্রীপরিগৃহীত। ⇒ পরিগগ্গহ ( ৪) । 
কত পরিচয ( বিণ)  পরিচয়কৃত, পরিচয় করিয়াছে বা
পরিচিত হইয়াছে এমন, পরিচয় করিয়া, পরিচিত, প্রত্যক্ষ পরিচয় করা হইয়াছে এমন। ⇒ কত, পরিচয ( বিণ)  
কত পরিচযতা ( স্ত্রী)  প্রত্যক্ষ পরিচয়, সাক্ষাৎ পরিচয়, পরিচয়, জানাশোনা। 
কত পরিত্ত ( বিণ)  রক্ষামন্ত্র বা রক্ষাকবচ গ্রহণ বা ধারণ করা হইয়াছে এমন, কবচধারী, পরিত্রাণ বা মুক্তির জন্য অথবা আত্নরক্ষার জন্য করণীয় কার্যসমূহ সসাপ্ত করিয়াছে এমন।
কত পরিযোসিত ( বিণ)  সম্পাদিত, সমাপিত,  সমাপ্ত, প্রস্তুত অর্থাৎ সম্পূর্ণরুপে তৈরি হওয়ার পর। 
কতপাতরাস ( বিণ)  প্রাতরাশ সমাপ্ত করিয়া, প্রাতঃকালীন ভোজন করিয়া,  প্রাতরাশ ভোজন করার পর, সকালে ভোজনের পর। ( জাতক)  কতপাতরাসভত্তন।⇒ পাতরাস। 
কতপাপ ( বিণ)  কৃতপাপ, পাপী, পাপকার্যকারী, পাপকর্মী। ( পেতবত্থু)  কতপাপ অকত কল্যাণ।
কতপাপকম্ম ( ক্লী)  কৃত পাপকর্ম,  পাপকার্য করিয়া, পাপকর্ম করিবার পর, পূর্ব পূর্ব জন্মের সম্পাদিত বা সঞ্চিত পাপকর্ম।
কতপিট্ঠ — চাউলের গুঁড়ায় বা আটা দ্বারা প্রস্তুত পিঠা বা রুটি পরটাদি। 
কত - পীতি - পবোধন ( বিণ)  হৃদয়ঙ্গম কৃত, জ্ঞাত। = পবোধন ( বিণ) । 
কতপুঞ্ঞ ( বিণ)  কৃতপুণ্য, পুণ্যবান, পুণ্যকার্য করিয়াছে এমন ( ব্যক্তি),  পুণ্যকার্য করিয়া, সৎকার্য করিয়া। ( পুং)  সৎলোক,  ধার্মিক পুরুষ। 
কতপুঞ্ঞতা ( স্ত্রী)  কৃতপুণ্য, সৎকর্মের ফল, কৃতপুণ্যের ফল, পূর্ব পূর্ব জন্মে সম্পাদিত পুণ্যের ফল। 
কতপুব্ব ( বিণ)  কৃতপূর্ব, পূর্বে করা হইয়াছে এমন। ⇒ কত।

কতবুদ্ধ কিচ্চ ( বিণ)  বুদ্ধকৃত্য সম্পাদন করিয়া, বুদ্ধ হইবার জন্য যাহা করণীয় কর্তব্য তাহা সম্পাদন করিয়া অথবা সম্পাদন করিয়াছেন এমন, বুদ্ধকৃত্য সম্পাদনকারী। 
কতবুদ্ধি ( বিণ)  বুদ্ধিকৃত, বুদ্ধি করিয়াছে এমন,চালাক, চতুর, নিপুণ, দক্ষ, বুদ্ধিমান,  দক্ষতা সহকারে সম্পাদিত। 
কত ভত্তকিচ্চ ( বিণ) আহার করিয়া, আহারের পর,  ভাত৷ খাইয়াছে এমন,যে ব্যক্তি ভোজনকৃত সমাপণ করিয়াছে,  আহার করিয়াছে এমন,  আহারের পর, খাওয়ার কার্য শেষ করিয়া। = কত অন্নকিচ্চ, ভুত্তবন্ত। ⇒কত (২)  ভত্তকিচ্চ।
কতভদ্দক ( বিণ)  উপকার করা হইয়াছে এমন, মঙ্গল কামনা করা হইয়াছে এমন, সৎকার্য করা হইয়াছে এমন। 
কতভাব ( পুং)  কৃতকার্য, করা হইয়াছে এরুপ অবস্থা, করা হইয়াছে এরুপ কার্য,  কৃতভাব, সমাপ্তি,  ঘটনা, সম্পাদন, কৃতি, সম্পন্নকার্য। 
কতভিনীহার ( বিণ)  [ কত + অভিনীহার ] (( উচ্চাকাঙ্কাকৃত))। ⇒ অভিনীহার। 
কতভীরুট্ঠান ( বিণ)  নির্ভরযুক্ত স্থান করিয়াছে এমন, নির্ভয়যুক্ত স্থান করিয়াছে এমন, নিরাময় স্থান করিয়াছে এমন, উপকারী, আশ্রয়দাতা। ⇒ কতকল্যণ। ( ক্লী)  আশ্রয়, নিরাপত্তা, নিরাপদ স্থান,  নিরাসঙ্ক স্থান। 
কতভূমিকম্ম -- পুণ্যার্থে পরিস্কারাদি ভূমিকার্য করা হইয়াছে এরুপ, পুণ্য কামনায় ভূমি পরিস্কারাদি আবশ্যকীয় কার্য করিয়াছে এরুপ।

কতমংপণ — কয় প্রকার?  কয়টি?  কতটি? ⇒ পন ( ২) ।। 
কতমচ্চুকম্ম ( বিণ)  দাড়ি কামাইয়া, দাড়ি কামাইয়াছে এমন, দাড়ি কামাইবার পর। ( জাতক)  কত মচ্চুকম্মকরণ। 
কতমত্ত ( বিণ)  মাতাল করা হইয়াছে এমন, মাতাল করিয়া। ⇒ কত। 
কতমত্তে ( অধিকরণ কারক, ক্রি - বিণ)  ( কিছু)  সমাপ্ত হওয়ার পর, শেষ করা মাত্রই, সমাধা করিবার পরমুহূর্তে,  করিবার পর, করিবার মাত্রই। 
কতমরণ ( বিণ)  মরিবার পর, মরিয়া, মরিলে, মৃত।
কতমল্লক ( বিণ)  সংগঠিত দাঁতের, পিঠ বা পৃষ্ঠদেশ আঁচড়ানি, যাহার দ্বারা পিঠ আঁচড়ানো বা চুলকানো যায়। 
কতমেন ( ক্রি - বিণ)  কোন পথে?  কোন পথ দিয়া?  কিরুপে? কেমনে?  কতমেন মগ্গেন — কোন পথ ধরিয়া বা কোন পথে?  
কতযোগ্গ ( বিণ)  উত্তমরুপে অভ্যস্ত, উত্তমরুপে শিক্ষাপ্রাপ্ত, কাজে লাগাইবার যোগ্য করা হইয়াছে এমন, শিক্ষিত, কর্মক্ষম, কার্যকর, উপকারী। ⇒ যোগ্গা। 
কতর ( বিণ)  ( দুইটির মধ্যে)  কোনটি?  ( জাতক)  কতরং উপদ্দবং ন করেয্য?  ( যৌগিকে)  কতরগেহ? কতরগন্ধং?  কতরাদিবসং?  কতরনগরতো — কোন নগর হইতে?  ( ধর্মপদ অর্থকথা)  কতর নাম?  ( বিণ)  কতরং নাম —  কি নামের?  ( জাতক)  কতরস্মিং মগ্গে — কোন পথে?  কিরুপে?  কেমনে?  
কতরুপ ( বিণ)  স্বভাবকৃত, স্বাভাবিককৃত, স্বতঃস্ফূর্তভাবে করা হইয়াছে এরুপ, স্বভাবসিদ্ধভাবে করা হইয়াছে
এরুপ। ( জাতক বর্ণনায়)  কতজানিয, কত সভাব। 
কতলুদ্দ ( বিণ)  নিষ্ঠুর, হিংস্র, ভীষণ,  ভয়ঙ্কর, নির্দয়, ভীতিজনক, ভয়ানক। 
কতলুদ্দ ( বিণ)  লোভী, লোভপরায়ণ, আসক্ত, অনুরক্ত।
কতবিনাসক ( বিণ)  বিনষ্টকারী, অনুপমকারী, অনিষ্টকারী। 
কতবিস্সাস ( বিণ)  বিশ্বস্ত, আস্থাবান, বিশ্বাসকৃত,  বিশ্বাস্য, বিশ্বাসযোগ্য, নির্ভরযোগ্য। 
কতবেণী ( বিণ)  কৃতবেণী, বেণী করা হইয়াছে এমন, চুল আঁচড়াইয়া বেণী করা বা খোপা বাঁধা হইয়াছে এমন। 
কতবেদিতা ( স্ত্রী)  কৃতজ্ঞতা, উপকার স্মরণ বা স্বীকারকরণ, সাধ্যানুসারে উপকারী হিতচিন্তা বা অনিষ্ট  নিবারণের চেষ্টাকরণ, উপকারকের সহিত সদ্ব্যবহার - করণ। ⇒ কতঞ্ঞুতা। 
কতবেদিন ( বিণ)  কৃতজ্ঞ, মনোযোগী। = কতঞ্ঞু, কতবেদী। 
কতবেদী ( বিণ)  কৃতজ্ঞ, উপকারকের সহিত সদ্ব্যবহারকারী, কেহ উপকার করিলে যে ব্যক্তি তাহা চিরকাল মনে রাখে এবং উপকারীর অনিষ্ট নিবারণের এবং প্রত্যুপকারের চেষ্টা করে। = কতবেদিন। 
কতসংসগ্গ ( বিণ) কৃতসংসর্গ, সংসর্গ করা হইয়াছে এমন, একত্রে বাস করা হইয়াছে এমন, ঘরোয়া, বন্ধুত্বপূর্ণ, সংসর্গযুক্ত, সুপরিচিত, সচরাচর ঘটিত। ⇒ কত। 
কতসক্কার ( বিণ)  সৎকারপ্রাপ্ত, অতিথি সৎকার করা হইয়াছে এরুপ, সম্মানপ্রাপ্ত, সম্মান করা হইয়াছে এরুপ, সৎকারকৃত, সম্মানিত, পূজিত, পূজাপ্রাপ্ত, পূজা করা হইয়াছে এমন। 
কতসঙ্কেত ( বিণ)  পূর্বের সঙ্কেতমতো, সঙ্কেত করা হইয়াছে এরুপ, চুক্তিবদ্ধ, ঐক্যবদ্ধ, একমতের। ( ক্লী) পূর্বে যেরুপ সঙ্কেত করা হইয়াছে,  পূর্বপ্রদত্ত সঙ্কেত, পূর্বসূত্র,  পূর্বনিয়ম, পূর্বপ্রথা, পূর্বচিহ্ন, পূর্বলক্ষণ। 
কতসঙ্কত ( বিণ)  সংগ্রহকৃত, সংগ্রহ করিয়া একস্থানে সমাবেশ করা হইয়াছে এমন, সংগৃহীত। ( পুং)  যে ব্যক্তি ধর্মসম্বন্ধীয় পুস্তক লেখার অংশ গ্রহণ করিয়াছে। 
কতসভাব = কতরুপ। ⇒ কতরুপ। 
কতসল্লাহ ( বিণ)  বর্ম পরিহিত, বর্ম ধারণ করিয়াছে এমন,  যুদ্ধসজ্জা পরিহিত, যুদ্ধসজ্জা পরিধান করিয়াছে এমন।
কতসিক্খ ( বিণ)  শিক্ষিত, শিক্ষাপ্রাপ্ত। 
কতস্সম ( বিণ)  শ্রমকৃত, পরিশ্রম করিয়া করা হইয়াছে এমন, শ্রম করা হইয়াছে এরুপ। 
কতহত্থ ( বিণ)  কৃতহস্ত, সিদ্ধহস্ত, অভ্যস্ত হস্ত, ( তীরুন্দাজের)  অভ্যস্ত, হাতের অভ্যাস করিয়াছে এমন, চতুর, দক্ষ, নিপুণহস্ত। ( তুলনীয়)  কতূপাসন। ⇒ হত্থ (১)। 
কতহত্থিক — কৃত্রিম হস্তী,  খেলার পুতুলস্বরুপ হস্তী।। 
কতাকত ( বিণ)  [ কত + অকত ] কৃত এবং অকৃত, তৈরিকৃত এবং স্বাভাবিক, যাহা করা হইয়াছে আর যাহা করা হয়
নাই এমন। ( বিনয়)  কতাকত যূস; কতাকতং। 
কতাকতং [ কতং + অকতং ] যাহা করা হইয়াছে আর যাহা করা হয় নাই,  যাহা করা হইয়াছে আর যাহা ফেলিয়া রাখিয়াছে। ( ধর্মপদ)  কতানি অকতানি চ — যেই কার্য করা হইয়াছে আর যেই কার্য করা হয় নাই, কৃতকার্য আর অকৃতকার্য, সম্পন্ন এবং অসম্পন্ন কার্য। 
কতাগস ( বিণ)  পাপী, পাপকারী, পাপকর্মী। 
কতাধিকার ( বিণ)  [ কত + অধিকার ] উদ্যেগী,  যত্নশীল,  প্রেচেষ্টাকারী, আর্য অষ্টাঙ্গিক পথে গমনে উদ্যমশীল, সত্যপথে গমনে চেষ্টাকারী,  অধিকারপ্রাপ্ত, সঙ্কল্পিত, উচ্চাকাঙ্কিত। = কতাভিনীহার। 
কতানুগ্গহ ( বিণ)  [ কত + অনুগ্গহ ] অনুগ্রহকৃত, সাহায্যকৃত, অনুগ্রহ বা দয়া প্রদর্শন করা হইয়াছে এমন, সাহায্য প্রদত্ত, সহায়তা প্রদত্ত, অনুগৃহীত। 
কতাপরাধ ( বিণ)  দোষী, অপরাধকারী, অপরাধী, অপরাধ করিয়াছে এমন।
কতাভিনিবেস ( বিণ)  [ কত + অভিনিবেস ] অভিনিবেশ সহকারে যে ব্যক্তি শিক্ষা করে, একাগ্র মনে বা মনোযোগের সহিত যে ব্যক্তি শিক্ষা করে, যে ব্যক্তি দৃঢ়প্রতিজ্ঞ বা স্থিরসঙ্কল্প, মনোযোগী, মনোযোগ দেওয়া হইয়াছে এরুপ। = কতাভিনীহার।

কতিন্দ্রিয ( বিণ)  ইন্দ্রিয় বিষয়ে দমিত বা শিক্ষিত, শান্ত - ইন্দ্রিয় সম্পন্ন, যাহার ইন্দ্রিয়সমূহ শান্ত বা দমিত হইয়াছে, শান্তিন্দ্রিয়। 
কতিপয ( বিণ, বহুবচন) কতিপয়, কিয়ৎ সংখ্যক, কতকগুলি, কতেক, কতক, কিছু,  মাঝারি। = অপ্পকে ( পেতবত্থু); ( একবচনে)  অপ্পক। 
কতিপযবারে — কয়েকবারে, কয়েক সময়ে, কতিপয় সময়ে। 
কতিপাহ ( ক্লী)  কয়েকদিন, সামান্য কয়েকদিন। 
কতিপাহং ( ক্রি - বিণ)  [ কতিপয + অহং ] কতিপয় দিবসের জন্য, কয়েক দিনের জন্য। ( করণ কারকে)  কতিপাহেন -- কয়েক দিনের মধ্যে। ( জাতক)  কতিপাহ'চ্চযেন। 
কতিপাহ'চ্চযেন — কয়েক দিন পর, কয়েক দিন থাকিয়া। = দ্বীহ তীহ অচ্চযেন। দ্বীহ তীহ। 
কতিপাহেন ( ক্রি - বিণ)  ⇒ কতিপাহং। 
কতিম ( পুং)  ( যৌগিকে)  কতিম পক্খস্স — পক্ষকালের মধ্যে যে কোন দিন, অর্ধমাসের মধ্যে কোন দিন, পক্ষকাল বা পনের দিনের মধ্যে কোন দিন ( বিনয়) । 
কতিমী ( স্ত্রী)  = কতিম। ⇒ কতিম।  কতিমী পক্খস্স। 
কতিবস্স ( বিণ)  ১ কয়েক বৎসর ধরিয়া,  কত বৎসর?  কত বৎসর বয়স? ২ ( ভিক্ষুদের উপসম্পাদা লাভের বয়স সম্বন্ধে)  কত বর্ষা, কত বর্ষাবাস গত হইয়াছে? কত বর্ষাবাসের ( ছোট বা বড়) ?  কত বর্ষার ( ছোট বা বড়) ?

কতূপকার ( পুং)  কৃতজ্ঞতা, উপকারীর উপকার স্বীকার, কৃত উপকার, কৃত্যুপকার, কাহাকে সাহায্য করা, যাহাকে যে উপকার করা হইয়াছে। = কতকল্যাণ। ( বিণ)  উপকার করা হইয়াছে এমন, সাহায্য করা হইয়াছে এমন, সহায়তা প্রদত্ত, অনুকম্পাকৃত, মিত্রের ন্যায় সাহায্য করা হইয়াছে এমন।

কতমল্লক ( বিণ)  সংগঠিত দাঁতের, পিঠ বা পৃষ্ঠদেশ আঁচড়ানি, যাহার দ্বারা পিঠ আঁচড়ানো বা চুলকানো যায়। 
কতমেন ( ক্রি - বিণ)  কোন পথে?  কোন পথ দিয়া?  কিরুপে? কেমনে?  কতমেন মগ্গেন — কোন পথ ধরিয়া বা কোন পথে?  
কতযোগ্গ ( বিণ)  উত্তমরুপে অভ্যস্ত, উত্তমরুপে শিক্ষাপ্রাপ্ত, কাজে লাগাইবার যোগ্য করা হইয়াছে এমন, শিক্ষিত, কর্মক্ষম, কার্যকর, উপকারী। ⇒ যোগ্গা। 
কতর ( বিণ)  ( দুইটির মধ্যে)  কোনটি?  ( জাতক)  কতরং উপদ্দবং ন করেয্য?  ( যৌগিকে)  কতরগেহ? কতরগন্ধং?  কতরাদিবসং?  কতরনগরতো — কোন নগর হইতে?  ( ধর্মপদ অর্থকথা)  কতর নাম?  ( বিণ)  কতরং নাম —  কি নামের?  ( জাতক)  কতরস্মিং মগ্গে — কোন পথে?  কিরুপে?  কেমনে?  
কতরুপ ( বিণ)  স্বভাবকৃত, স্বাভাবিককৃত, স্বতঃস্ফূর্তভাবে করা হইয়াছে এরুপ, স্বভাবসিদ্ধভাবে করা হইয়াছে
এরুপ। ( জাতক বর্ণনায়)  কতজানিয, কত সভাব। 
কতলুদ্দ ( বিণ)  নিষ্ঠুর, হিংস্র, ভীষণ,  ভয়ঙ্কর, নির্দয়, ভীতিজনক, ভয়ানক। 
কতলুদ্দ ( বিণ)  লোভী, লোভপরায়ণ, আসক্ত, অনুরক্ত।
কতবিনাসক ( বিণ)  বিনষ্টকারী, অনুপমকারী, অনিষ্টকারী। 
কতবিস্সাস ( বিণ)  বিশ্বস্ত, আস্থাবান, বিশ্বাসকৃত,  বিশ্বাস্য, বিশ্বাসযোগ্য, নির্ভরযোগ্য। 
কতবেণী ( বিণ)  কৃতবেণী, বেণী করা হইয়াছে এমন, চুল আঁচড়াইয়া বেণী করা বা খোপা বাঁধা হইয়াছে এমন। 
কতবেদিতা ( স্ত্রী)  কৃতজ্ঞতা, উপকার স্মরণ বা স্বীকারকরণ, সাধ্যানুসারে উপকারী হিতচিন্তা বা অনিষ্ট  নিবারণের চেষ্টাকরণ, উপকারকের সহিত সদ্ব্যবহার - করণ। ⇒ কতঞ্ঞুতা। 
কতবেদিন ( বিণ)  কৃতজ্ঞ, মনোযোগী। = কতঞ্ঞু, কতবেদী। 
কতবেদী ( বিণ)  কৃতজ্ঞ, উপকারকের সহিত সদ্ব্যবহারকারী, কেহ উপকার করিলে যে ব্যক্তি তাহা চিরকাল মনে রাখে এবং উপকারীর অনিষ্ট নিবারণের এবং প্রত্যুপকারের চেষ্টা করে। = কতবেদিন। 
কতসংসগ্গ ( বিণ) কৃতসংসর্গ, সংসর্গ করা হইয়াছে এমন, একত্রে বাস করা হইয়াছে এমন, ঘরোয়া, বন্ধুত্বপূর্ণ, সংসর্গযুক্ত, সুপরিচিত, সচরাচর ঘটিত। ⇒ কত। 
কতসক্কার ( বিণ)  সৎকারপ্রাপ্ত, অতিথি সৎকার করা হইয়াছে এরুপ, সম্মানপ্রাপ্ত, সম্মান করা হইয়াছে এরুপ, সৎকারকৃত, সম্মানিত, পূজিত, পূজাপ্রাপ্ত, পূজা করা হইয়াছে এমন। 
কতসঙ্কেত ( বিণ)  পূর্বের সঙ্কেতমতো, সঙ্কেত করা হইয়াছে এরুপ, চুক্তিবদ্ধ, ঐক্যবদ্ধ, একমতের। ( ক্লী) পূর্বে যেরুপ সঙ্কেত করা হইয়াছে,  পূর্বপ্রদত্ত সঙ্কেত, পূর্বসূত্র,  পূর্বনিয়ম, পূর্বপ্রথা, পূর্বচিহ্ন, পূর্বলক্ষণ। 
কতসঙ্কত ( বিণ)  সংগ্রহকৃত, সংগ্রহ করিয়া একস্থানে সমাবেশ করা হইয়াছে এমন, সংগৃহীত। ( পুং)  যে ব্যক্তি ধর্মসম্বন্ধীয় পুস্তক লেখার অংশ গ্রহণ করিয়াছে। 
কতসভাব = কতরুপ। ⇒ কতরুপ। 
কতসল্লাহ ( বিণ)  বর্ম পরিহিত, বর্ম ধারণ করিয়াছে এমন,  যুদ্ধসজ্জা পরিহিত, যুদ্ধসজ্জা পরিধান করিয়াছে এমন।
কতসিক্খ ( বিণ)  শিক্ষিত, শিক্ষাপ্রাপ্ত। 
কতস্সম ( বিণ)  শ্রমকৃত, পরিশ্রম করিয়া করা হইয়াছে এমন, শ্রম করা হইয়াছে এরুপ। 
কতহত্থ ( বিণ)  কৃতহস্ত, সিদ্ধহস্ত, অভ্যস্ত হস্ত, ( তীরুন্দাজের)  অভ্যস্ত, হাতের অভ্যাস করিয়াছে এমন, চতুর, দক্ষ, নিপুণহস্ত। ( তুলনীয়)  কতূপাসন। ⇒ হত্থ (১)। 
কতহত্থিক — কৃত্রিম হস্তী,  খেলার পুতুলস্বরুপ হস্তী।। 
কতাকত ( বিণ)  [ কত + অকত ] কৃত এবং অকৃত, তৈরিকৃত এবং স্বাভাবিক, যাহা করা হইয়াছে আর যাহা করা হয়
নাই এমন। ( বিনয়)  কতাকত যূস; কতাকতং। 
কতাকতং [ কতং + অকতং ] যাহা করা হইয়াছে আর যাহা করা হয় নাই,  যাহা করা হইয়াছে আর যাহা ফেলিয়া রাখিয়াছে। ( ধর্মপদ)  কতানি অকতানি চ — যেই কার্য করা হইয়াছে আর যেই কার্য করা হয় নাই, কৃতকার্য আর অকৃতকার্য, সম্পন্ন এবং অসম্পন্ন কার্য। 
কতাগস ( বিণ)  পাপী, পাপকারী, পাপকর্মী। 
কতাধিকার ( বিণ)  [ কত + অধিকার ] উদ্যেগী,  যত্নশীল,  প্রেচেষ্টাকারী, আর্য অষ্টাঙ্গিক পথে গমনে উদ্যমশীল, সত্যপথে গমনে চেষ্টাকারী,  অধিকারপ্রাপ্ত, সঙ্কল্পিত, উচ্চাকাঙ্কিত। = কতাভিনীহার। 
কতানুগ্গহ ( বিণ)  [ কত + অনুগ্গহ ] অনুগ্রহকৃত, সাহায্যকৃত, অনুগ্রহ বা দয়া প্রদর্শন করা হইয়াছে এমন, সাহায্য প্রদত্ত, সহায়তা প্রদত্ত, অনুগৃহীত। 
কতাপরাধ ( বিণ)  দোষী, অপরাধকারী, অপরাধী, অপরাধ করিয়াছে এমন।
কতাভিনিবেস ( বিণ)  [ কত + অভিনিবেস ] অভিনিবেশ সহকারে যে ব্যক্তি শিক্ষা করে, একাগ্র মনে বা মনোযোগের সহিত যে ব্যক্তি শিক্ষা করে, যে ব্যক্তি দৃঢ়প্রতিজ্ঞ বা স্থিরসঙ্কল্প, মনোযোগী, মনোযোগ দেওয়া হইয়াছে এরুপ। = কতাভিনীহার। 
কতাভিনীহার ( বিণ)  [ কত + অভিনীহার ] ( বুদ্ধত্ব লাভের জন্য)  যিনি স্থিরসঙ্কল্প, দৃঢ়প্রতিজ্ঞ, ( কোন কাজের জন্য)  ঝুঁকি লইয়াছে এমন। = কতাধিকার,  কতাভিনিবেস। 
কতাভিসেক (বিণ)  [ কত + অভিষেক ] অভিষিক্ত, অভিষেক করা হইয়াছে এমন, উৎসর্গ বা পবিত্র করা হইয়াছে এরুপ,  উৎসর্গীকৃত।
কতাবকাস ( বিণ)  [ কত + অবকাস ] অবকাশ পাইয়া, অবকাশ করিয়া, সুযোগ করিয়া, ছুটিপ্রাপ্ত,  অভিভূত, সাধারণ সমীপে আত্নপ্রকাশিত। = কতোকাস, অবকাস। ⇒ অবকাস ( ১) । ( পুং)  আবির্ভাব হওন, সাধারণ সমীপে আত্নপ্রকাশকরণ। 
কতি ( বিণ)  কতিপয়, কয়টি, কতটি, কতেক, কতকগুলি। ( বিনয়)  কতিসিক্খাপদানি। ( সংযুক্ত নিকায়)  কতিসঙ্গাতিগ— কয়টি আসক্তিকে জয় করিয়া অথবা কয়টি আসক্তিকে জয় করা হইয়াছে?  
কতিকা ( স্ত্রী)  ১ চুক্তি, অন্বয়, ঐক্য, একমত, নিয়ম, নিয়মপত্র, একবার নামা, সন্ধি। ( সাহিত্য)  কাতিকা। ২ কথোপকথন, কথাবার্তা, গল্পকরণ, গল্প বলা, কথাবলা,  আলাপ, গল্প। অধম্মিকা কতিকা। ( তুলনীয়)  কথা, কথিকা। ( বিনয়)  কতিকাকতিকং করোতি -- শৃঙ্খলাবদ্ধ করা, সন্ধিবদ্ধ হওয়া, সন্ধি করা, চুক্তি করা। ( যৌগিকে)  কতিকাবত্ত। 
কতিকাবত্ত ( ক্লী)  সন্ধি, চুক্তি, অন্বয়, ঐক্য, একমত। ( ধর্মপদ)  কতিকবত্তং করোতি — সন্ধিসূত্রে বা চুক্তিতে বিশ্বাসী হওয়া। ( জাতক)  কতিকাবত্তং ভিন্দতি --
চুক্তিভঙ্গ করা। ( বিনয়)  কতিকাবত্তং সন্ঠাণ — সন্ধি বা চুক্তিতে প্রবেশকরণ। 
কতিত গোময— সিদ্ধ করা গোবর। = কঠগোময়, ছকণকা। ⇒ পক্কট্ঠী। 
কতিন্দিয — ইহা ভুল শব্দ এবং " কতিন্দ্রিয" শব্দই হইবে। ⇒ কতিন্দ্রিয। 
কতিন্দ্রিয ( বিণ)  ইন্দ্রিয় বিষয়ে দমিত বা শিক্ষিত, শান্ত - ইন্দ্রিয় সম্পন্ন, যাহার ইন্দ্রিয়সমূহ শান্ত বা দমিত হইয়াছে, শান্তিন্দ্রিয়। 
কতিপয ( বিণ, বহুবচন) কতিপয়, কিয়ৎ সংখ্যক, কতকগুলি, কতেক, কতক, কিছু,  মাঝারি। = অপ্পকে ( পেতবত্থু); ( একবচনে)  অপ্পক। 
কতিপযবারে — কয়েকবারে, কয়েক সময়ে, কতিপয় সময়ে। 
কতিপাহ ( ক্লী)  কয়েকদিন, সামান্য কয়েকদিন। 
কতিপাহং ( ক্রি - বিণ)  [ কতিপয + অহং ] কতিপয় দিবসের জন্য, কয়েক দিনের জন্য। ( করণ কারকে)  কতিপাহেন -- কয়েক দিনের মধ্যে। ( জাতক)  কতিপাহ'চ্চযেন। 
কতিপাহ'চ্চযেন — কয়েক দিন পর, কয়েক দিন থাকিয়া। = দ্বীহ তীহ অচ্চযেন। দ্বীহ তীহ। 
কতিপাহেন ( ক্রি - বিণ)  ⇒ কতিপাহং। 
কতিম ( পুং)  ( যৌগিকে)  কতিম পক্খস্স — পক্ষকালের মধ্যে যে কোন দিন, অর্ধমাসের মধ্যে কোন দিন, পক্ষকাল বা পনের দিনের মধ্যে কোন দিন ( বিনয়) । 
কতিমী ( স্ত্রী)  = কতিম। ⇒ কতিম।  কতিমী পক্খস্স। 
কতিবস্স ( বিণ)  ১ কয়েক বৎসর ধরিয়া,  কত বৎসর?  কত বৎসর বয়স? ২ ( ভিক্ষুদের উপসম্পাদা লাভের বয়স সম্বন্ধে)  কত বর্ষা, কত বর্ষাবাস গত হইয়াছে? কত বর্ষাবাসের ( ছোট বা বড়) ?  কত বর্ষার ( ছোট বা বড়) ? 
কতিবিধ ( বিণ)  [ কতি + বিধ ] কত প্রকারের?  কত রকমের?  কতবিধ? 
কতিসঙ্গাতিগ ⇒ কতি। 
কতিহং [ কতি + অহং ] কয়দিন, কতদিন ( অর্থাৎ আমি আর কতদিন? ) । 
কতুপাসন = কতূপাসন। ⇒ কতূপাসন।
কতুস্সাহ ( বিণ)  উদ্যোগী, উদ্যমশীল, উৎসাহী, বলবান, প্রাণবান। 
কতূপকার ( পুং)  কৃতজ্ঞতা, উপকারীর উপকার স্বীকার, কৃত উপকার, কৃত্যুপকার, কাহাকে সাহায্য করা, যাহাকে যে উপকার করা হইয়াছে। = কতকল্যাণ। ( বিণ)  উপকার করা হইয়াছে এমন, সাহায্য করা হইয়াছে এমন, সহায়তা প্রদত্ত, অনুকম্পাকৃত, মিত্রের ন্যায় সাহায্য করা হইয়াছে এমন। 
কতূপাসন ( বিণ)  নিপুণ, দক্ষ, ধনুর্বিদ্যায় নিপুণ, চতুর।

( মধ্যম নিকায় বর্ণনায়)  কতূপাসন ধনুবিজ্জায। 
কতে ( ক্রি - বিণ)  পক্ষে, খাতিরে, কাহারও পক্ষে, কাহারও বদলে, জন্য। ( কর্মকারকে)  মংকেত ( জাতক) । ( সম্বন্ধ পদে)  মংসস্স কতে ( জাতক) । 
কতোকাস ( বিণ)  [ কত + ওকাস ] সাধারণ সমীপে আত্নপ্রকাশ করিয়া, কাজে অবসর করিয়া বা পাইয়া, ছুটি লইয়া, অনুমতি গ্রহণ করিয়া বা পাইয়া, কিছু বলিবার সময় পাইয়া। ⇒ ওকাস। ( ৩) ।
কত্ত ( বিণ)  = কন্ত। ⇒ কন্ত ( ২) । "কত্ত" শব্দ কেবল "পরিকত্ত " শব্দের সহিত যৌগিক ব্যবহারে দৃষ্ট হয়। ⇒ কন্ততি। 
কত্তপাহ = কতিপাহ। ⇒ কতিপাহ, কতিপাহং। 
কত্তব্ব ( বিণ)  করিবার যোগ্য, করিতে হইবে এরুপ, করা উচিত, করিবার উপযুক্ত, সম্পন্ন করিতে হইবে এরুপ, করা কর্তব্য বা উচিত, করা যাইতে পারে এরুপ। = কাতব্ব। ( ক্লী)  কর্তব্যকার্য, কর্তব্য, বাধ্যতা, বশ্যতা, সামরিক কার্য, ধর্ম, বাধ্যবাধকতা, যাহা নিশ্চিত করিতে হইবে। 
কতব্বক ( ক্লী)  কর্তব্য, কর্তব্যকার্য, কার্যের ভার, অন্য কর্তৃক নির্দিষ্ট কার্য, কাজ, শ্রম, নির্দিষ্ট পাঠ। 
কত্তব্বকরণং ( ক্লী)  কর্তব্যকার্যকরণ, যাহা করিবার তাহা করণ। = কত্তব্বত্তং। 
কত্তব্বতা ( স্ত্রী)  যোগ্যতা, উপযুক্ততা, কর্তব্য, কর্তব্যকাজ, যাহা নিশ্চিত করিতে হইবে, করিবার উপযুক্ততা, করিবার যোগ্যতা । ইতি কত্তব্বতায — কারণ আমাকে ইহা এরুপে করিতে হইয়াছিল। 
কত্তব্বত্তং = কত্তব্বকরণং। ⇒ কত্তব্বকরণং। 
কত্তব্বযুত্তক ( ভ- বিণ)  করিবার উপযুক্ত, করিবার যোগ্য, কর্তব্যযুক্ত, উপযুক্ত কর্তব্য, কর্তব্যের উপযুক্ত। = কাতব্বযুত্তক। ⇒ যুত্তক। ( ক্লী)  কর্তব্য, কর্তব্যকার্য। = কত্তব্ব ( ক্লী) ।  ( তুলনীয়)  কাতব্ব। 
কত্তব্বাকত্তব্ব ( বিণ)  [ কত্তব্ব + অকত্তব্ব ] কর্তব্য এবং অকর্তব্য, যাহা করা উচিত এবং যাহা করা অনুচিত, যাহা করিবার আর যাহা না করিবার। ( ক্লী)  কর্তব্য এবং অকর্তব্য, কর্তব্যকার্য এবং অকর্তব্যকার্য, করিবার যোগ্য কার্য এবং করিবার অযোগ্য কার্য। 
কত্তর ( পুং)  প্রস্তুতকারক, যে করে সেই ব্যক্তি,  নির্মিতা, কারক।  ( বি - ভা)  কত্তর করোতি = পঞ্ঞহং করোতি — প্রশ্ন করা। ( সম্বোধন পদ)  কত্তে। ( কর্তকারক)  " কত্তরকত্তা " শব্দের পরিবর্তে "কত্তরকত্ত" ( সল্লকত্ত শব্দে) । কত্তা। ( বিণ) অতি ছোট, ক্ষুদ্র। 
কত্তরদগু ( পুং)  ১ ছড়ি, লাঠি, যন্ঠি, ( সন্ন্যাসীদের)  যন্ঠি। ২ গমন, পার হওন, পথ, রাস্তা, প্রবেশ পথ,  গমনাধিকার, বিধিবদ্ধকরণ, ( গাড়ির)  ভাড়া, পর্যটন, ঘটনা, পুস্তকাদির ক্ষুদ্র অংশ। 
কত্তরযট্ঠি ( স্ত্রী)  ভিক্ষুদের লাঠি, দরিদ্র বা ভিক্ষাজীবীর যন্ঠি। = কত্তরদগু। ⇒ যট্ঠি ( ২) ।
কত্তররথ ( পুং)  পুরাণ রথ, পুরাতন শকট, জীর্ণ গাড়ি। 
কত্তরসুপ্প ( পুং)  কুলা, তুষ ঝাড়িবার ঝুড়ি। ( ধর্মপদ অর্থকথা) কত্তরসুপ্পে পক্খিপিত্বা সঙ্কার - কূটে ছড্ডেহি। 
কত্তরি ( স্ত্রী)  কাঁচি, বড় কাঁচির মতো যন্ত্র, সাঁড়াশি, ( কাঁকড়ার পায়ের কাঁচির মতো)  সাঁড়াশি। 
কত্তরিকা ( স্ত্রী)  ছুরি, কাঁচি। = কত্তরি। 
কত্তরী ( স্ত্রী)  = কত্তরি। ⇒ কত্তরি। 
কত্তা ( পুং)  ১ ( ব্যাকরণে)  কর্তৃকারক, ক্রিয়া সম্পাদক বা প্রযোজক পদ। ২ কর্তা,কার্যকারক, স্বামী, প্রণেতা,  ক্রিয়ানুকূল,  নির্মাতা, কৃতিমান ব্যক্তি, প্রভু, অধ্যক্ষ। পঞ্হকত্তা — প্রশ্নকর্তা, প্রশ্নকারক, প্রশ্নকারী। মন্তানং কত্তারো — মন্ত্রকর্তা, মন্ত্রপ্রণেতা। ( যৌগিকে)  রাজ- কত্তারো ( কত্তারো)  যাহারা রাজপদে অভিষিক্ত করে বা রাজা করে তাহারা। কত্তা হোতি নো ভাসিতা — সে কাজের লোক কথার নহে। ৩ ( কর্মকারকে)  যে ব্যক্তি ( ভালোমন্দ)  কিছু করে। কত্তা ইট্ঠং করোতি = কতঞ্ঞু৷ কত্তা অনিট্ঠং করোতি = অকতঞ্ঞু৷ ( অর্থকথা বর্ণনায়) কত।  ৪ রাজ কর্মচারী, রাজকর্মী, রাজার কার্যকারক। = কত্তর ।  খত্তর। 
কত্তিক ( পুং)  মাসের নাম, কার্তিক মাস, আশ্বিনী পূর্ণিমা হইতে কার্তিক পূর্ণিমা পর্যন্ত একমাস। ⇒ মাস ( ১) । 
কত্তিক চাতুমাসিকা ( স্ত্রী)  শ্রাবণ ভাদ্র আশ্বিন কার্তিক অর্থাৎ আষাঢ়ী পূর্ণিমা হইতে কার্তিকা পূর্ণিমা পর্যন্ত — এই চারিমাসকে " কত্তিক চাতুমাসিকা" বলা হয়।
কত্তিক চাতুমাসিনী ( স্ত্রী)  = কত্তিক চাতুমাসিকা। ⇒ কত্তিক চাতুমাসিকা। 
কত্তিক তেমাসিকা ( স্ত্রী)  শ্রাবণ ভাদ্র আশ্বিন অর্থাৎ আষাঢ়ী পূণিমা হইতে আশ্বিনী পুর্ণিমা পর্যন্ত — এই তিনমাসকে "কত্তিক তেমাসিকা" বলা হয়। ইহা ভিক্ষুদের বর্ষাবাস ব্রত পালনের সময়। 
কত্তিক তেমাসিনী ( স্ত্রী)  = কত্তিক তেমাসিকা। ⇒ কত্তিক তেমাসিকা। 
কত্তিক দ্বেমাসিক — আশ্বিন মাস এবং কার্তিক মাস অর্থাৎ ভাদ্র পূর্ণিমা হইতে কার্তিক পূর্ণিমা পর্যন্ত — এই দুই মাস সময়কে "কত্তিক দ্বেমাসিক" বলা হয়। " পুব্বকত্তিকা" এবং " পচ্ছিম কত্তিকা"— এই দুই মাসকেও " কত্তিক দ্বেমাসিক" বলা হয়। 
কত্তিকমাস ( পুং)  কার্তিক মাস। = কত্তিক । ( ১) । 
কত্তিকা ( স্ত্রী)  ১ কার্তিক মাস। = কত্তিক, কত্তিকমাস। ২ এক নক্ষত্রের নাম অর্থাৎ কৃত্তিকা নক্ষত্র। 
কত্তিকা চাতুমাসিনী ( স্ত্রী)  = কত্তিক চাতুমাসিনী। ⇒ কত্তিক চাতুমাসিনী। 
কত্তিকা চোরক — কার্তিক মাসের চোর, ভিক্ষুদের চীবর বিতরণের পর অর্থাৎ কঠিন চীবর দান হওয়ার পর যে চোর চুরি করে, যে চোর চীবর দানের পর ভিক্ষুদিগকে আক্রমণ করে।
কত্তিকাছণ — আশ্বিনী পূর্ণিমা উৎসব, প্রবারণা উৎসব, ভিক্ষুদের বর্ষাবাস ত্যাগ করিবার উৎসব।
কত্তিকা তেমাসি ( বিণ)  বর্ষা ঋতুর ত্রৈমাসিক পূর্ণিমা অর্থাৎ আশ্বিনী পূর্ণিমা বা প্রবারণা পূর্ণিমা। ( বিনয়, মহাবর্গ)  কত্তিকা তেমাসি পুণ্ণমা। ( মহাবর্গ)  কত্তিকা তেমাসি পুণ্ণমাসী। 
কত্তিকামাস ( পুং)  = কত্তিকামাস। ⇒ মত্তিকামাস ( মহাবর্গে)  কত্তিকে মাসে। 
কত্তিকা সুক্কপক্খ — কার্তিকী শুক্লপক্ষ, কার্তিক পূর্ণিমার পূর্ব জ্যোৎস্নাময় পনের দিন অর্থাৎ অমাবস্যার পর কার্তিক পূর্ণিমা পর্যন্ত পনের দিন। 
কত্তু ( পুং)  নির্মাতাকারক, প্রস্তুতকারক, প্রণেতা,  গ্রন্থকার, কর্তা, দগুের অপরাধী। = কত্তা, কত্তর। 
কত্তুং ( তুম)  করিতে। = কাতুং। ⇒ করোতি। 
কত্তুকম্যতা ( স্ত্রী)  কিছু করিবার ইচ্ছা, কাজের অভিলাষ। 
কত্তুকাম ( বিণ)  কিছু করিবার ইচ্ছা করে এমন, কাজের জন্য ইচ্ছুক, (( কর্মাকাঙ্ক্ষী))  
কত্তুকামতা ( স্ত্রী)  কিছু করিবার বা সম্পাদন করিবার ইচ্ছা, কাজের অভিলাষ। = কত্তুকম্যতা। 
কত্তুকামতায — কাজ করিবার ইচ্ছায। ⇒ ছন্দ। 
কত্থ ( বিণ)  কর্ষিত । = কসিত, কট্ঠ ( ১) ।  ⇒ কসতি। ( ক্রি - বিণ)  কোথায়?  কোন স্থানে?  কোন দিকে? কোন বিষয়ে?  কিসে? যাহাতে, যেখানে, যথায়?  আমি কিসে আশ্চর্য হইলাম?  = কহং। 
কত্থকত্থচি ( অনির্দিষ্ট)  যে কোন স্থানে, কোন স্থানে অথবা অন্য কোন স্থানে। 
কত্থ নু খো ⇒ কত্থ ( ক্রি - বিণ) । 
কত্থচি — কোন স্থানে, যে কোন স্থানে, কোনও স্থানে। ( জাতক)  কত্থচিদ এব। 
কত্থচি কত্থচি — কোন কোন স্থানে,  কোন কোন জায়গায়। 
কত্থচি ন কত্থচি — কোন না কোন স্থানে। ন কত্থচি — কোন স্থানে নহে, কুত্রাপি নহে। 
কত্থচি বাস — কোন বাসস্থানে?  কোন আবাসে বা ঘরে?  
কত্থচি বাসিক — কোথায় বাস করো?  কোন স্থানের অধিবাসী?  
কত্থট্ঠিত = কত্থ ঠিত।  ⇒ কত্থ ঠিত। 
কত্থ ঠিত ( চরিত্রে)  কোন অবস্থায়?  কোন ভাবে?  ( উত্তরে)  এত্থ ঠিত — এই অবস্থায়, এইভাবে। 
কত্থতি ( ক্রি)  [ √কথ + অ ] অহঙ্কার করা, আছে বলিয়া গর্বিত হওয়া। ( অতীত)  কত্থি। ( অ - বিণ)  কত্থিত; ( তুলনীয়)  পবিকত্থিত। 
কত্থনা ( স্ত্রী)  অহঙ্কার, আছে বলিয়া গর্ব। 
কত্থিতর ( বিণ)  দাম্ভিক,  অহঙ্কারী, যে ব্যক্তি আছে বলিয়া গর্ব করে।

কত্থিন ( বিণ)  = কত্থিতর। ⇒ কত্থিতর। ( অঙ্গুত্তর নিকায়)  কত্থিন বিকত্থিন। 
কত্থী ( বিণ)  = কত্থিতর। ⇒ কত্থিতর। 
কত্থু ( পুং)  ১ শৃগাল। কত্থু সোণা — শৃগাল এবং কুকুর। = কোত্থু। 
কত্থু সোণা ⇒ কত্থু। = কোত্থু সুণা। 
কত্থুক = কোত্থুক। ⇒ কোত্থুক। 
কত্বা ( ক্রি - বি)  করিয়া, প্রস্তুত করিয়া। ⇒ করোতি। 
কত্বান = কত্বা।  ⇒ কত্বা। 
কথং ( ক্রি - বিণ)  ১ কিরুপে, কেমন করিয়া, কি প্রকারে, কতদূর, কেমন অবস্থায়। ( বর্তমান কালে অনির্দিষ্টভাবে)  কত্থং পুরিসো পটিলভতি?  ( ভবিষ্যৎ কালে অবস্থা বুঝাইলে)  কত্থং তত্থ গমিস্সামি?  কত্থং ন দস্সামি?  ২ কেন,  কি কারণে, কি উদ্দেশ্য। ( যৌগিকে)  কত্থং চ? ( বিনয়) । ( দীর্ঘনিকায়)  কত্থং চরহি?  ( বিনয়)  কত্থং নু? কত্থং নু খো? ( ⇒ এবং ন খো) ।  ( দীর্ঘনিকায়)  কত্থং পন?  ( জাতক)  কত্থংসু?  কত্থং হি?  ( ধর্মপদ অর্থকথা)  কত্থং হি নাম?  উক্ত যৌগিক উদাহারণগুলি একই অর্থবাচক।
কথংকথা ( স্ত্রী)  " কিরুপে বলে, কিরুপে" অর্থাৎ সন্দেহ, সংশয়, অনিশ্চয়তা, অমীমাংসিত বা অস্থিরকৃত মন। ( তুলনীয়)  কঙ্খা। ( বর্ণনায়)  বিচিকিচ্ছা, দুক্খে কঙ্খা। ( বিণ)  বিগত কথ।  ( বি - ভা)  তিণ্ণ - বিচিকিচ্ছ বেসারাজ্জপ্পত্ত ( দীর্ঘনিকায়) ।  ( সূত্রনিপাত)  তিণ্ণ - বিচিকিচ্ছ ( বিগত বিসল্ল) । ( দীর্ঘনিকায়)  কথং কথা সল্ল — সন্দেহরুপ শল্য। 
কথং কথা সল্ল = কথঙ্কথা সল্ল। ⇒ কথঙ্ককথা সল্ল, কথং কথা।
কথং কথিন ( বিণ)  ১ অনিশ্চিত, সংশয়যুক্ত, সন্দিগ্ধ, অস্থিরকৃত। ২ ভাষণকারী, যে ব্যক্তি কথা বলে। 
কথং কথী ( বিণ)  সন্দেহকারী, যে ব্যক্তি নিশ্চিত বা স্থিরীকৃত নহে। 
কথং চ — কেন?  কি কারণে?  ⇒ কথং ( ২) । 
কথং চরতি — কেন?  কি কারণে?  ⇒ কথং ( ২) । 
কথং জীবিন ( বিণ)  কিরুপ জীবন যাপনের অর্থাৎ কিরুপভাবে জীবনযাপন করে?  
কথং দস্সিন ( বিণ)  কোন দৃষ্টির অর্থাৎ কিরুপ দৃষ্টি রাখে?  ⇒ কথংসীল। 
কথং নু — কেন?  কি কারণে?  ⇒ কথং ( ২) । 
কথং নু খো — ( তুমি)  নাকি?  তাহা হইলে কি?  তাহা হইলে ইহা কি?  ( সমার্থবাচক শব্দ)  কিং নু খো?  এবং নু খো?  ন নু খো?  ⇒ কথং ( ২) ,  কিং। 
কথং পকার ( বিণ)  কোন প্রকারের?  কি রকমের?  ( সূত্রনিপাত)  কথংপ্পকার। 
কথং পটিপন্ন ( বিণ)  কিরুপ প্রতিপন্ন?  কোন গতিতে পতিত?  কোন পথে চলিতেছে অর্থাৎ কি করিতেছে?
কোন অবস্থায় পতিত?  
কথংপন — কেন?  কি কারণে?  ⇒ কথং  ( ২) ।
কথংপ্পকার = কথং পকার।  ⇒ কথং পকার। 
কথং ভাবিত ( বিণ)  কিরুপ কর্ষিত?  কিরুপ মার্জিত?  কিরুপ সভ্য?  কিরুপ সুরুচিসম্পন্ন?  কি রকম অভ্যস্ত?  
কথং ভুত ( বিণ)  কি রকমের?  কোন শ্রেণীর? কোন জাতীয়?  কোন বিভাগের?  কোন পদের?  কিসের মতো?  কোন প্রকারের?  
কথং রুপ ( বিণ)  কি রুপের?  কোন আকারের?  কি রকমের?  
কথং বণ্ণ ( বিণ)  কোন আকারের?  কিরুপ চেহারাযুক্ত?  কিসের মতো?  কোন আকৃতির?  
কথং বিধ ( বিণ)  কি রকমের?  কোন প্রকারের?  কোন শ্রেণীর?  কোন জাতীয়?  = কথং পকার, কথংভুত। 
কথং সমেত ( বিণ)  কিরুপ গঠিত?  কিরুপ সংস্থাপিত?  কিরুপ শারীরিক বা মানসিক সহজাত অবস্থার?  কিরুপ মানসিক প্রকৃতির?  কিরুপ বিধিযুক্ত বা বিধিবদ্ধ?  কোন নিয়মের?  কোন শাসনতন্ত্রের?  
কথং সীল ( বিণ)  কোন চরিত্রের?  কিরুপ স্বভাবের?  কিরুপ নৈতিক চরিত্রের?  ( সূত্রনিপাত)  কথংদস্সী কথং সীলো উপসন্তো'তি বুচ্চতি?  
কথং সু — কেন?  কি কারণে?  ⇒ কথং ( ২) ,  সু ( ৩) । 
কথং হি — কেন?  কি কারণে?  ⇒ কথং ( ২) । 
কথং হি নাম — কেন?  কি কারণে?  ⇒ কথং ( ২) । 
কথঙ্কথা = কথং কথা। ⇒ কথং কথা। 
কথঙ্কথা সল্ল — " কিরুপে বল?  কেন?  অর্থাৎ সন্দেহযুক্ত বা অনিশ্চয়তার অস্থির, মনরুপ শল্য বা শেল। = কথং কথা সল্ল। ⇒ সল্ল,  কথঙ্কথা। 
কথঙ্কথিন = কথং কথিন। ⇒ কথং কথিন। 
কথঙ্কথী = কথং কথী। ⇒ কথং কথী। 
কথঙ্কর ( বিণ)  [ কথং + কর ] কিরুপ কাজের?  কি রকম কার্যের?  কি কাজের?  ( জাতক)  কথঙ্কর অহং নো নিরযং পতেয্যং?  = কিস্তিকরো। 
কথষ্ণি [ কথং + চি ] কদাচিৎ, প্রায় না, তমুহূর্তেই, কষ্টে, দুঃখের সহিত, কষ্টে সৃষ্টে, কখনও না। 
কথন ( ক্লী)  ১ কথা, আলাপ,  কথোপকথন, কথাবার্তা, গল্প বলা। ২ কথন অর্থাৎ উত্তর প্রদান,  ( প্রশ্নের)  মীমাংসা করণ। ( জাতক)  পঞ্হকথন। ৩ ধর্ম প্রচার, ধর্মোপদেশ প্রদান। ৪ আবৃত্তিকরণ, মুখস্থ বলা, বর্ণনাকরণ, আখ্যায়িকা কথন। ( তুলনীয়)  কথিত। কথনং করোতি — কথা বলিতেছে,  কথায় যোগ দিয়েছ। ৫ দস্তপূর্ণ উক্তি, গর্বের বিষয় বা বস্তু। 
কথনং যেব — কথা বলামাত্রই তৎক্ষণাৎ। = ঠানসো ঞাণ। 
কথন সমত্থ ( বিণ)  কথা বলিতে পারে এমন ( অর্থাৎ জিহবা
আড়ষ্ট নহে অথবা শিশু কথা বলিতে শিখিয়াছে) ।  তেন সদ্ধিং কথনসমত্থো — তাহার সহিত কথা বলিতে বা আলাপ করিতে সমর্থ। 
কথনসীল ( বিণ)  অতিরিক্ত কথা বলিবার স্বভাব আছে যাহার, যে ব্যক্তি অতিরিক্ত কথা বলে, বাচাল, অতিভাষী, বহুভাষী। 
কথনাকার — কথা বলিতেছে এরুপ ভাব, কথায় যোগ দিয়াছে এমন ভাব। = কথানং করোতি। ⇒ কথন। 
কথযতি ( ক্রি)  [ √কথ + অয ] কথা বলা, আলাপ করা, বলা, কহা, বর্ণনা করা। = কথেতি। ধাতুরুপ নির্ণয়ঃ— বর্তমান ( উত্তম পুরুষ একবচনে)  কথযামি; ( বহুবচনে)  কথযাম। ( মধ্যম পুরুষ একবচনে)  কথযসি; ( বহুবচনে)  কথযথ। ( প্রথম পুরুষ একবচনে)  কথযতি; ( বহুবচনে)  কথযন্তি। পঞ্চমী ( উত্তম পুরুষ একবচনে)  কথযামি; ( বহুবচনে)  কথযাম। ( মধ্যম পুরুষ একবচনে)  কথযাহি; ( বহুবচনে)  কথযাথ। ( প্রথম পুরুষ একবচনে)  কথযতু; ( বহুবচনে)  কথযন্তু। সপ্তমী ( উত্তম পুরুষ একবচনে)  কথযেয্যামি; ( বহুবচনে)  কথযেয্যাম। ( মধ্যম পুরুষ একবচনে)  কথযেয্যাসি; ( বহুবচনে)  কথযেয্যাথ। ( প্রথমে পুরুষ একবচনে)  কথযেয্য; ( বহুবচনে)  কথযেয্যুং।  অতীত ( উত্তম পুরুষ একবচনে)  কথযিং; ( বহুবচনে)  কথযিম্হা। ( মধ্যম পুরুষ একবচনে) কথযি; ( বহুবচনে)  কথযিত্থ। ( প্রথম পুরুষ একবচনে)  কথযি; ( বহুবচনে)  কথযিংসু। ভবিষ্যত ( উত্তম পুরুষ একবচনে)  কথযিস্সামি; ( বহুবচনে)  কথযিস্সাম।  ( মধ্যম পুরুষ একবচনে)  কথযিস্সসি; ( বহুবচনে)  কথযিস্সথ। ( প্রথম পুরুষ একবচনে)  কথযিস্সতি; ( বহুবচনে)  কথযিস্সন্তি। ( ক্রি - বি)  কথযিত্বা,  কথিত্বা। ( তুম)  কথযিতুং, কথিতুং। 
কথল — বিশুদ্ধিমার্গে ইহা "কঠল" শব্দের ভুল বানান। ⇒ কঠল। 
কথলি ( বিনয়ের "উদান" এতে ব্যবহৃত শব্দ। = কথলিক। ⇒ কথলিক। 
কথলিক ( ক্লী)  ( এই শব্দটি সর্বদা যৌগিক শব্দে ব্যবহৃত হয়, যথা —)  পাদোদক পাদপীঠ পাদকথলিক। অতএব "পাদ কথলিক" শব্দের অর্থঃ — পদধৌত করিবার পর পা মুছিবার কাপড় অথবা পা রাখিবার চৌকি বা পাদপীঠ ( বিনয়) ; ( বিমানবত্থু অর্থকথা বর্ণনায় ইহার অর্থ)  পা রাখিবার চৌকি ( পাদ - ঠপনযোগ্গং দারুখগুং আসনং)  ( বুদ্ধঘোষের বর্ণনায়)  পাদপীঠ — ধৌত পদ রাখিবার চৌকি; আর পাদকথলিক — ধৌত না করা পদ রাখিবার চৌকি অথবা পা মুছিবার কাপড় ( ঘংসন) । 
কথা ( স্ত্রী)  ১ কথা, আলাপ, কথাকথন, কথোপকথন, কথাবার্তা। ( দীর্ঘনিকায়)  অন্তর কথা। ( তুলনীয়)  সল্লাপ। ( দীর্ঘনিকায়)  তিরচ্ছান কথা। ( মধ্যম নিকায়)  কথাবত্থুলি। ২ ভাষণ, ভাষা, ধর্মোপদেশ, বক্তৃতা। ( বিনয়)  কথং করোতি — ভাষণ করা, কথা বলা; অনুপুব্বি কথং — আনুপূর্বিক ধর্মোপদেশ বা ধর্মকথা
অর্থাৎ চতুর্বিধ আনুপূর্বিক ধর্মকথা,  যথা — দানকথা, সীলকথা, সগ্গকথা, মগ্গকথা ( ⇒ অনুপুব্ব) । ৩ গল্প, গল্প কথন, বিস্তারিত কথা, বর্ণনা, বিস্তারিত বর্ণনা। ( পেতবত্থু অর্থকথা)  বাহির কথা — বিস্তারিত কথা। ( খুদ্দকপাঠ অর্থকথা)  বাহির কথা — ধর্ম বিষয়ের বহির্ভূত কথা। ৪ শব্দ, পদ, কথা, আদেশ, বিবাদ, মন্তব্য, প্রতিজ্ঞা, সংবাদ, উপদেশ। ( জাতক)  কথং গণ্হাতি — উপদেশ গ্রহণ করা। ৫ ব্যাখ্যা, অর্থ,  ব্যাখ্যা পুস্তক, অর্থবহি ( অত্থকথা বর্ণনায়) । ( তুলনীয়)  গতিকথা। ৬ বিচার, আলোচনা, তর্ক, বাদানুবাদ। ( মহাবংস)  কথাবত্থু। ৭ = খত্তর। ⇒ খত্তর। 
কথা - উদয — তর্ক, বাদানুবাদ, বিচার- বিতর্ক, বিবাদ, কলহ,  ঝগড়া। 
কথা - ওজ্জ = কথা - উদয। ⇒ কথা - উদয। 
কথা কথা —কথার কথা, অসামঞ্জস্য কথা। = ছিন্নকথা। 
কথাকুসল — বাকচাতুরি,  বক্তৃতাচাতুরি, ভাষণদক্ষ, কথায় চালাকি, জ্ঞানপূর্ণ কথায় চতুর। 
কথাধম্ম — কথার প্রসঙ্গ, কথাপ্রসঙ্গ,  ভাষণের বা বক্তৃতার বিষয়,  তর্কের বিষয়। 
কথা নিগঘোস — প্রশংসাবাণী, কীর্তিশব্দ, তোশামোদ,  অনুচিত স্তুতি। 
কথপবত্তি — কথার গতি, কথাবার্তার ধারা। 
কথাপাভত ( ক্লী)  কথার বিষয়, গল্পের বিষয়, আলাপের বিষয়বস্তু। 
কথাপভতি ( ক্লী)  "কাহিনী বা গল্প আনয়ন" কিছু বলার সুযোগ, সংবাদ, গল্প। 
কথাপেতি  (ক্রি)  কথা বলায়, সংবাদ দেওয়া বা প্রেরণ করা, কাহারও দ্বারা কথা দেওয়া বা কথা বলানো। ⇒ কথেতি। ( অতীত)  কথাপেসি, কথাপথি। 
কথাবাহুল্ল — কথাবাহুল্য, প্রয়োজনের অতিরিক্ত পরিমাণ কথা,  কথাপ্রাচুর্য, বাকপ্রিয়তা, বহুভাষিতা। 
কথাভিঞাণ — প্রত্যুপন্নমতিত্ব, কথা বলার উপযুক্ত স্মরণশক্তি, কথা বলিবার পূর্বনির্দিষ্ট স্মৃতি, কথা বলার পূর্বনির্দিষ্ট অনুস্মরণ জ্ঞান। 
কথামগ্গ ( পুং)  আখ্যান,  বর্ণিত বিষয়, ইতিহাস, ইতিবৃত্ত, বিবরণ, বৃত্তান্ত, সত্যঘটনা, প্রকৃত বিবরণ, কারণ, হেতু, উৎকর্ষ। 
কথারস — কথারস, কথার মাধুর্য, শ্রবণসুখজনক কথা, শ্রুতিমধুর কথা। 
কথাবত্থু ( ক্লী)  ১ বিচার্য বিষয়, আলোচনার বিষয়, তর্কের বিষয়, বাদানুবাদের বিষয়, কথাবার্তার বিষয়, ভাষার বিষয়, পুস্তকের বিষয়বস্তু, বক্তৃতার বিষয়বস্তু, ধর্মোপদেশের বিষয়বস্তু, যুক্তি, হেতুবাদ, বাদানুবাদ, তর্কপ্রণালী, নির্ঘন্ট, আলোচ্য বিষয়। = কল্যাণমিত্তস্স গুণা।⇒ কথা, কথাকুসল। ২ অভিধর্মপিটকের অন্তর্গত পঞ্চম গ্রন্থের নাম "কথাবত্থু"।
কথা সমুট্ঠান -- কথার উৎপত্তি, তর্কের উৎপত্তি, বাদানুবাদের উৎপত্তি, আলোচনার উৎপত্তি,  কথাবার্তার উৎপত্তি। 
কথাসমুট্ঠাপন — কথা উত্থাপন,  আলোচনা উত্থাপন, কথাবার্তার উত্থাপন। 
কথা সম্পযোগ — আলাপ- সম্বন্ধীয় সম্পর্ক বা পত্র লেখালেখি,  আলাপ - সম্বন্ধীয় সংস্রব বা সংসর্গ, কথা সম্প্রযোগ। 
কথা সল্লাপ ( পুং)  মিত্রভাবাপন্ন আলাপ,  বন্ধুত্বভাবে কথা, অনুকূল কথা, সদ্ভাবসূচক কথাবার্তা, বন্ধু সদৃশ কথা বা আলোচনা। ( দীর্ঘনিকায় অর্থকথা বর্ণনায়)  কথন পটিকথন। ( ধর্মপদ অর্থকথায়)  কথং করোতি। 
কথিক ( বিণ)  কথিক, বর্ণনীয়, বলিবার যোগ্য, আলোচিত, ব্যাখ্যাকারী,  আলোচনাকারী, ব্যাখ্যা করার উপযুক্ত, যে ব্যাক্তি বলে, যে ব্যাক্তি আলোচনা বা ব্যাখ্যা করে,  যে ব্যাক্তি বর্ণনা করে। ( পুং)  বক্তা। ( যৌগিকে)  চিত্তকথিক ( থেরীগাথা) ; ( তুলনীয়)  চিত্তকথিন। ( অঙ্গুত্তর নিকায়)  তিরচ্ছান কথিক; ধম্মকথিক — যে ব্যক্তি ধর্মকথা বলেন; কথিকত্থের — যে থের বা স্থবির ভিক্ষু ধর্মবিষয় বলেন বা উপদেশ প্রদান করেন; মহাকথিক — বর্ণনাকারী, ব্যাখ্যাকারী। 
কথিকত্থের ( পুং)  ⇒ কথিক। 
কথিকা ( স্ত্রী)  ১ যে স্ক্রীলোক ধর্মকথা বলেন অথবা ধর্ম- সম্বন্ধে উপদেশ প্রদান করেন, ধর্মোপদেশকারিণী ভিক্ষুণী। ২ চুক্তি,  অন্বয়, ঐক্য, একমত। ⇒ কতিকা। 
কথিত ( বিণ)  কথিত, উক্ত, বর্ণিত,  ব্যাখ্যাত, উচ্চারিত। ⇒ কথেতি। সুকথিত — সুন্দররুপে কথিত বা ভাষিত অথবা বর্ণিত। ( করণ কারক ক্লীবলিঙ্গে)  তেন কথিতং — তাহার দ্বারা ভাষিত বা কথিত ( জাতক) । 
কথিন ( বিণ)  কথা বলে এমন, ভাষণকারী, উপদেশ প্রদানকারী। ( পুং)  বক্তা,  উপদেষ্টা,  ধর্মোপদেষ্টা, ধর্মপ্রচারক। ( জাতকে)  ধম্ম - কথিকেসু চিত্রকথী — ধর্মকথকদের মধ্যে বিচিত্র ভাষণকারী। ( মিলিন্দ প্রশ্নে)  কথিন সেট্ঠ — ধর্মকথিকদের মধ্যে শ্রেষ্ট ভাষণকারী। ⇒ কথং কথিন। 
কথিনসেট্ঠ ⇒ কথিন। 
কথীযতি ( ক্রি)  কহাইয়া, বলাইয়া। ( ব- বিণ)  কথীযমান, কচ্ছমান। ⇒ কথেতি।

থনপ ( পুং, ক্লী)  দুগ্ধপোষ্য শিশু, দুধের ছেলে বা মেয়ে, অতি শিশু, স্তনপায়ী শিশু, স্তনপ। 
থনমুখ ( ক্লী)  স্তনমুখ, স্তনাগ্র, স্তনশিখা। = থনগ্গ। 
থনযতি ( ক্রি)  [ √থন + অয ] গর্জন করা, গোঁ গোঁ শব্দ করা, চিৎকার করা, রব করা, উচ্চৈঃস্বরে প্রচার করা, বজ্রধ্বনি করা, বজ্রের মতো শব্দ করা, উচ্চধ্বনিতে প্রকাশ করা, তীব্র নিন্দা করা। ( অতীত)  থনযি।  ( ব- বিণ)  থনযন্ত।= 
থনযুগল ( ক্লী, বহুবচন)  স্তনযুগল, স্তনদ্বয়। ⇒ যুগল। 
থন-সিত- দারক ( পুং)  = থনপ। ⇒ থনপ, সিত। 
থনিত ( ক্লী)  মেঘগর্জন, বজ্রধ্বনি, ভীমনাদ, উচ্চশব্দ, ভীতি প্রদর্শন বা নিন্দাসূচক বাক্য। ( বিণ)  স্তনিত, শব্দিত। ⇒ থনেতি। 
থনিন ( বিণ)  যে স্ত্রীলোকের স্তন আছে, স্তন আছে এমন। ( সূত্রনিপাত)  তিম্বরুথনিন।  ( জাতক)  পুচিমন্দ থনিন। 
খনেতি ( ক্রি) [ √থন + এ ] বজ্রধ্বনি করা, বজ্রের মত শব্দ করা, উচ্চধ্বনিতে প্রকাশ করা, তীব্র নিন্দা করা, গর্জন করা, গোঁ গোঁ শব্দ করা, চিৎকার করা, রব করা, উচ্চৈঃস্বরে প্রচার করা। ( সংযুক্ত নিকায়)  মেঘো থনযং থনেতি; থনেতি দেবো।  মেঘো থনযিত্বা। ( অতীত)  থনেসি। ( অ- বিণ)  থনিত। ( ব- বিণ)  থনেন্ত। ( ক্রি - বি)  থনযিত্বা, থনেত্বা। ⇒ গজ্জতি, থুনতি।
